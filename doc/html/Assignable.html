Concept Assignable 可赋值概念

Assignable 可赋值

Description 说明

Assignable types must have copy constructors, operator= for assignment, and the swap() function defined.
可赋值类型必须具有复制构造函数、用于赋值的 operator=，并定义了 swap() 函数。
