Reference 参考手册

Elements:
元素：

  • Element access - Declares an access specification for class members 为类成员声明一个访问规格
  • Element boostbook - Defines a BoostBook book 定义一个 BoostBook 书
  • Element class - Declares a class or class template 声明一个类或类模板
  • Element class-specialization - A specialization (partial or full) of a class template 类模板的特化(偏特化或全特化)
  • Element code - Mimics the code tag in HTML 模仿 HTML 中的 code 标签
  • Element compile-fail-test - A testcase that should fail to compile 一个会编译失败的测试案例
  • Element compile-test - A testcase that should compile correctly 一个可以正确编译的测试案例
  • Element complexity - The time/space/etc. complexity of a function 一个函数的时间/空间/等的复杂度
  • Element constructor - Declares a constructor of the enclosing class 声明所属类的构造函数
  • Element copy-assignment - Declares a copy-assignment operator 声明一个复制赋值操作符
  • Element data-member - Declares a data member of a class 声明一个类的数据成员
  • Element default - The default value of a function or template parameter 一个函数或模板的参数的缺省值
  • Element description - Detailed description of a construct 一个结构的详细说明
  • Element destructor - Declares a destructor for the enclosing class 声明所属类的析构函数
  • Element effects - Declares the side effects of a function 声明一个函数的副作用
  • Element enum - Declares an enumeration type 声明一个枚举类型
  • Element enumname - References an enumeration type with the given name 引用一个给定名字的枚举类型
  • Element enumvalue - A single value of an enumeration 枚举类型的单个值
  • Element free-function-group - A set of functions that are grouped together under one name 在同一个名字之下的一组函数
  • Element function - Declares a function 声明一个函数
  • Element functionname - References a function with the given name 引用带给定名字的函数
  • Element globalname - References a global with the given name 引用一个给定名字的全局变量 
  • Element header - Declares a C++ header with the given name 声明给定名字的一个C++头文件
  • Element headername - References a C++ header with the given name 引用一个给定名字的C++头文件
  • Element if-fails - What it means when a testcase fails 当一个测试案例失败时意味着什么
  • Element inherit - Declares a base class of the enclosing class or struct 声明所属类或结构的基类
  • Element lib - A library dependency 一个库的依赖关系
  • Element library - Top-level element for a library 一个库的顶层元素
  • Element library-reference - Declares the reference material for a library 声明一个库的参考材料
  • Element librarycategory - Declares that the enclosing library is in this category 声明所属类属于这一类别
  • Element librarycategorydef - Defines a new library category 定义一个新的库类别
  • Element librarycategorylist - Categorized listing of libraries 库的分类列表
  • Element libraryinfo - Provides information about a library 提供有关一个库的信息
  • Element librarylist - Placeholder for an alphabetical list of libraries 用于库的字母序列表的占位符
  • Element libraryname - References a library of the given name 引用给定名字的库
  • Element librarypurpose - Describes in one short sentence or phrase the purpose of a library 用一句短句或短语描述库的用途
  • Element link-fail-test - Declares a test that should compile but fail to link 声明一个可以编译但不能链接的测试
  • Element link-test - Declares a test that should compile and link 声明一个可以编译和链接的测试
  • Element macroname - References a macro with the given name 引用一个给定名字的宏
  • Element method - Declares a method, i.e., a member function 声明一个方法，即一个成员函数
  • Element method-group - A set of methods that are grouped together under one name 在同一个名字之下的一组方法
  • Element namespace - Declares a namespace 声明一个名字空间
  • Element notes - Non-normative notes about a function's semantics 关于一个函数的语义的非标准化说明
  • Element overloaded-function - An overloaded function 一个重载的函数
  • Element overloaded-method - An overloaded method 一个重载的方法
  • Element parameter - A function parameter 一个函数参数
  • Element paramtype - The type of a function parameter 函数参数的类型
  • Element postconditions - Conditions that must hold after the function returns 在函数返回后必须满足的条件
  • Element precondition - Conditions that must be met prior to executing a function 在执行函数前必须被满足的条件
  • Element programlisting - A sample of program code 程序代码示例
  • Element purpose - A short description of an entity's use 一个实体用途的简短描述
  • Element rationale - Describes the rationale for a particular function's design 描述一个特定函数设计的原理
  • Element requirement - A requirement/property in the Jamfile for a testcase 一个测试案例的 Jamfile 中的要求/属性
  • Element requires - Declares the requirements of a function 声明一个函数的要求
  • Element returns - Description of the return value of a function 一个函数的返回值的说明
  • Element run-fail-test - A testcase that should compile and link, but fail on execution 一个可以编译和链接，但执行失败的测试案例
  • Element run-test - A testcase that should compile, link, and execute 一个可以编译、链接和执行成功的测试案例
  • Element signature - One signature of an overloaded function or method 一个重载函数或方法的签名
  • Element snippet - Pulls in a code snippet from a programlisting element 从一个 programlisting 元素取出的代码片断
  • Element source - Defines source code for a test 定义一个测试的源代码
  • Element specialization - Defines the specialization arguments for a class specialization 为一个特化类定义特化的参数
  • Element static-constant - Declares a static constant, e.g., const int foo = 5;. 定义一个静态常数，如 const int foo = 5;
  • Element struct - Declares a C++ struct 声明一个C++结构
  • Element struct-specialization - A specialization (full or partial) of a struct template 一个结构模板的特化(全特化或偏特化)
  • Element template - Declares the template parameters of a class or function 声明一个类或函数的模板参数
  • Element template-arg - A template argument in a specialization 在一个特化中的模板参数
  • Element template-nontype-parameter - A nontype template parameter 一个非类型模板参数
  • Element template-type-parameter - Declares a template type parameter 声明一个模板类型参数
  • Element template-varargs - Declares a variable-length list of template parameters 声明一个可变长度的模板参数列表
  • Element testsuite - Describes a library testsuite 描述一个库的测试套件
  • Element throws - Description of the exceptions thrown by a function 对由一个函数抛出的异常的说明
  • Element type - The type of an element or return type of a function 一个元素的类型或一个函数的返回类型
  • Element typedef - Declares a typedef 声明一个 typedef
  • Element union - Declares a C++ union or union template 声明一个C++联合或联合模板
  • Element union-specialization - A specialization (full or partial) of a union template 一个联合模板的特化(全特化或偏特化)
  • Element using-class - Injects the method and function names of a class into the local scope 将一个类的方法或函数名注入到本地作用域
  • Element using-namespace - Injects the declared names from a namespace into the local scope 将声明的名字从一个名字空间注入到本地作用域
