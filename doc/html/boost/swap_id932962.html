函数模板 swap

boost::swap

摘要

template<typename T1, typename T2, ..., typename TN> 
  void swap(variant<T1, T2, ..., TN> & lhs, variant<T1, T2, ..., TN> & rhs);

说明

作用： 通过调用 variant::swap 来交换 lhs 和 rhs.
抛出： 可能因 variant::swap 所引起的异常而失败。
