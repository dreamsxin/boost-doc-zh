函数 trim_left_copy_if

boost::algorithm::trim_left_copy_if — 左侧修剪 - 参数化

概述

template<typename OutputIteratorT, typename RangeT, typename PredicateT> 
  OutputIteratorT 
  trim_left_copy_if(OutputIteratorT Output, const RangeT & Input, 
                    PredicateT IsSpace);
template<typename SequenceT, typename PredicateT> 
  SequenceT trim_left_copy_if(const SequenceT & Input, PredicateT IsSpace);

说明

在输入中移除开头的所有空格。补充的判断式用于检测哪些 characters 可以被当做空格。结果为输入的一个被修剪的拷贝. 它以一个序列返回或者拷贝到输出迭代器中。

参数：

Input

一个输入 range

IsSpace

一个识别空格的一元判断式

Output

一个用来拷贝结果的输出迭代器

返回：

一个正好指向最后一个被插入 character 之后的输出迭代器或输入的一个拷贝

注意：

这个函数的第二个变体提供强异常安全保证
