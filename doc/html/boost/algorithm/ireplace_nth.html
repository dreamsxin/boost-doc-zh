函数模板 ireplace_nth

boost::algorithm::ireplace_nth — Replace nth 算法（大小写无关）

概述

template<typename SequenceT, typename Range1T, typename Range2T> 
  void ireplace_nth(SequenceT & Input, const Range1T & Search, int Nth, 
                    const Range2T & Format, 
                    const std::locale & Loc = std::locale());

说明

在输入中用格式 string 替换搜索 string 的第 N 次（从 0 开始索引）匹配。输入序列在原地被改变。搜索与大小写无关。

参数：

Format

一个用于替换的 string

Input

一个输入 string

Loc

一个用于大小写无关比较的地区设置

Nth

要被替换的匹配项的索引。这个索引以 0 为基数。对于负的 N，匹配项的计数从 string 的末尾开始。

Search

一个被搜索的 substring
