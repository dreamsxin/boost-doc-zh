函数模板 iter_split

boost::algorithm::iter_split — Split find 算法

概述

template<typename SequenceSequenceT, typename RangeT, typename FinderT> 
  SequenceSequenceT & 
  iter_split(SequenceSequenceT & Result, RangeT & Input, FinderT Finder);

说明

这个算法在输入上迭代执行一个给定的 finder，直到输入的末尾，或者没有找到匹配项。迭代是使用内建 find_iterator 进行的，所以只有在必要时才进行真正的搜索。每一个匹配项都被用作片段的分隔符。然后把这些片段返回到结果中。

参数：

Finder

供搜索使用的 Finder 对象

Input

一个将要被搜索的容器。

Result

用于容纳搜索结果的“容器的容器”。无论外层还是内层容器都必须有以一对迭代器为参数的构造函数。结果的典型类型是 std::vector<boost::iterator_range<iterator>>（这样一个 vector 的每一个元素都将包含一个限定匹配项的 range）。

返回：

一个结果的引用

注意：

结果中的先前内容将被覆盖。
