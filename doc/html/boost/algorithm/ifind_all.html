函数模板 ifind_all

boost::algorithm::ifind_all — Find all algorithm ( case insensitive ).

概述

template<typename SequenceSequenceT, typename Range1T, typename Range2T> 
  SequenceSequenceT & 
  ifind_all(SequenceSequenceT & Result, Range1T & Input, 
            const Range2T & Search, const std::locale & Loc = std::locale());

说明

这个算法在输入中查找搜索 string 的所有出现。 每一个部分被拷贝并作为一个新元素加入到输出容器中。因此结果容器必须能够持有匹配项（类似于 std::string 这样的兼容结构）的拷贝或引用（例如，使用迭代器范围类）。这样的容器的例子包括 std::vector<std::string> 或 std::list<boost::iterator_range<std::string::iterator>>。

Searching is case insensitive.

参数：

Input

一个将要被搜索的容器。

Loc

A locale used for case insensitive comparison

Result

一个可以持有 substrings 的引用的拷贝的容器

Search

一个被搜索的 substring。

返回：

一个结果的引用

注意：

结果中的先前内容将被覆盖。
