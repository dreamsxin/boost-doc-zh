说明

分割迭代器封装一个 Finder 并允许在一个 string 中进行递增搜索。以查找迭代器不同，风格迭代器迭代遍历匹配项之间的间隔。

查找迭代器是一个可读的前向遍历迭代器。

解引用这个迭代器生成一个限定当前匹配项的 iterator_range。

split_iterator 公有 构造/拷贝/析构

  1. split_iterator();

    构造空迭代器。所有的空迭代器都相等。

    后置条件：

    eof()==true

    		 
  2. split_iterator(const split_iterator & Other);

    构造一个 split_iterator 的拷贝

  3. template<typename FinderT> 
  split_iterator(IteratorT Begin, IteratorT End, FinderT Finder);

    为一个给定的 finder 和 range 构造新的 split_iterator。

  4. template<typename FinderT, typename RangeT> 
  split_iterator(RangeT & Col, FinderT Finder);

    为一个给定的 finder 和集合构造新的 split_iterator。

split_iterator 公有成员函数

  1. bool eof() const;

    检查 eof 条件。eof 条件就是没有更多的东西被搜索到，也就是说，split_iterator 在最后一个匹配项之后。
