函数模板 iends_with

boost::algorithm::iends_with — 'Ends with' 判断式（大小写无关）

概述

template<typename Range1T, typename Range2T> 
  bool iends_with(const Range1T & Input, const Range2T & Test, 
                  const std::locale & Loc = std::locale());

说明

当检测容器是 Input 的后缀时，换句话说，如果 input 以 test 结尾，这个判断式成立。元素比较与大小写无关。

参数：

Input

一个输入序列

Loc

一个用于大小写无关比较的地区设置

Test

一个检测序列

返回：

检测的结果

注意：

这个函数提供强异常安全保证
