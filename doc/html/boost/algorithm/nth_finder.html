函数 nth_finder

boost::algorithm::nth_finder — "Nth" finder

概述

template<typename RangeT> 
  unspecified nth_finder(const RangeT & Search, int Nth);
template<typename RangeT, typename PredicateT> 
  unspecified nth_finder(const RangeT & Search, int Nth, PredicateT Comp);

说明

Construct the nth_finder. The finder searches for the n-th (zero-indexed) occurrence of the string in a given input. 结果以限定这个匹配项的一个 iterator_range 的形式给出。

参数：

Nth

被查找的匹配项的索引

Search

一个被搜索的 substring。
