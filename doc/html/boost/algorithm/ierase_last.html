函数模板 ierase_last

boost::algorithm::ierase_last — Erase last 算法（大小写无关）

概述

template<typename SequenceT, typename RangeT> 
  void ierase_last(SequenceT & Input, const RangeT & Search, 
                   const std::locale & Loc = std::locale());

说明

从输入中移除 substring 的最后一次出现输入序列在原地被改变。搜索与大小写无关。

参数：

Input

一个输入 string

Loc

一个用于大小写无关比较的地区设置

Search

一个被搜索的 substring
