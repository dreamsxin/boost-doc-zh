类模板 typed_value

boost::program_options::typed_value

摘要

template<typename T, typename charT = char> 
class typed_value : public boost::program_options::value_semantic_codecvt_helper< charT >,
                    public boost::program_options::typed_value_base
{
public:
  // 构造/复制/析构
  typed_value(T *);

  // 公有成员函数
  typed_value * default_value(const T &) ;
  typed_value * default_value(const T &, const std::string &) ;
  typed_value * implicit_value(const T &) ;
  typed_value * implicit_value(const T &, const std::string &) ;
  typed_value * notifier(function1< void, const T & >) ;
  typed_value * composing() ;
  typed_value * multitoken() ;
  typed_value * zero_tokens() ;
  std::string name() const;
  bool is_composing() const;
  unsigned min_tokens() const;
  unsigned max_tokens() const;
  void xparse(boost::any &, const std::vector< std::basic_string< charT > > &) const;
  bool apply_default(boost::any &) const;
  void notify(const boost::any &) const;
  const std::type_info & value_type() const;
};

说明

用于处理指定类型的值的类。

typed_value 公有构造/复制/析构

  1. typed_value(T * store_to);

    构造函数。'store_to' 参数给出得知值后保存的地方。该参数不能为 NULL.

typed_value 公有成员函数

  1. typed_value * default_value(const T & v) ;

    指定缺省值，在没有显式指定值时使用。类型 'T' 应提供用于 ostream 的 operator<<。 

  2. typed_value * default_value(const T & v, const std::string & textual) ;

    指定缺省值，在没有显式指定值时使用。与上一个重载不同，类型 'T' 无需提供用于 ostream 的 operator<<，但用户必须提供缺省值的文字表示方式。 

  3. typed_value * implicit_value(const T & v) ;

    指定一个隐式的值，当给定一个选项而又没有给出值时使用。使用该值表示显式值是可选的，但如果给出显式值则必须严格紧随在选项后，如：'-ovalue' 或 '--option=value'。给定 '-o' 或 '--option' 将导致使用隐式值。 

  4. typed_value * implicit_value(const T & v, const std::string & textual) ;

    指定一个隐式的值，当给定一个选项而又没有给出值时使用。使用该值表示显式值是可选的，但如果给出显式值则必须严格紧随在选项后，如：'-ovalue' 或 '--option=value'。给定 '-o' 或 '--option' 将导致使用隐式值。与上一个重载不同，类型 'T' 无需提供用于 ostream 的 operator<<，但用户必须提供缺省值的文字表示方式。 

  5. typed_value * notifier(function1< void, const T & > f) ;

    指定一个函数，当最终值被确定时调用它。 

  6. typed_value * composing() ;

    指定该值是组合的。相关解释请见 'is_composing' 方法。 

  7. typed_value * multitoken() ;

    指定该值可以跨越多个 tokens。 

  8. typed_value * zero_tokens() ;
  9. std::string name() const;
  10. bool is_composing() const;
  11. unsigned min_tokens() const;
  12. unsigned max_tokens() const;
  13. void xparse(boost::any & value_store, 
                const std::vector< std::basic_string< charT > > & new_tokens) const;

    创建 'validator' 类的一个实例并调用其 operator() 执行真正的转换。 

  14. bool apply_default(boost::any & value_store) const;

    如果之前通过调用 'default_value' 指定了缺省值，则将缺省值保存在 'value_store'。如果缺省值被保存则返回 true。 

  15. void notify(const boost::any & value_store) const;

    如果在创建 *this 时指定了一个变量地址用于保存值，则将值保存在那里。否则，无动作。 

  16. const std::type_info & value_type() const;
