boost::program_options::value_semantic

class value_semantic {

  // 构造/复制/析构

说明

指定如何分析选项值并转换为C++类型的类。

value_semantic 公有构造/复制/析构

  1. ~value_semantic();

value_semantic 公有成员函数

  1. std::string name() const;

    返回选项的名字。名字仅对于自动帮助信息有意义。 

  2. unsigned min_tokens() const;

    该选项在命令行中出现的最少 tokens 数。 

  3. unsigned max_tokens() const;

    该选项在命令行中出现的最多 tokens 数。 

  4. bool is_composing() const;

    如果来自于不同源的选项值可以组合，则返回 true。否则使用来自第一个源的选项值而忽略其它来源。 

  5. void parse(boost::any & value_store, 
               const std::vector< std::string > & new_tokens, bool utf8) const;

    分析指定一个选项值的一组 tokens。保存结果到 'value_store'，使用希望的任意表示法。如果同一选项的值被多次指定，则可能多次调用。 

  6. bool apply_default(boost::any & value_store) const;

    被调用以将缺省值赋给 'value_store'。如果缺省值被赋值，则返回 true，如果缺省值不存在，则返回 false。 

  7. void notify(const boost::any & value_store) const;

    当一个选项的最终值被确定时调用。
