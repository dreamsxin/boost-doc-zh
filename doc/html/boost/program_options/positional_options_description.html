说明

描述位置选项。

该类允许猜测位置选项的选项名，位置选项是在命令行中以位置进行标识的。该类使用由用户提供的信息来为每一个位置选项关联一个名字，或称为名字未知。

主要的假设是，只有位置选项本身的相对位置是有意义的，其中插入的任何普通选项都不影响位置选项的解析。

用户在初始化时，指定头 N 个位置选项名字为 X1，接着的 M 个选项名字为 X2，等等。

positional_options_description 公有成员函数

  1. positional_options_description & add(const char * name, int max_count) ;

    指定接着最多 'max_count' 个位置选项命名为 'name'。值 '-1' 表示 'unlimited'。在 'max_value' 等于 '-1' 的调用之后不可以再调用 'add'。 

  2. unsigned max_total_count() const;

    返回可以出现的位置选项的最大数量。返回 numeric_limits<unsigned>::max() 表示无穷。 

  3. const std::string & name_for_position(unsigned position) const;

    返回与位置为 'position' 的位置选项相关联的名字。前置条件：position < max_total_count()
