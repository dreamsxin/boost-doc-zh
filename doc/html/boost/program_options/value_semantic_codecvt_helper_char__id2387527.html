类 value_semantic_codecvt_helper<char>

boost::program_options::value_semantic_codecvt_helper<char>

摘要

class value_semantic_codecvt_helper<char> :
  public boost::program_options::value_semantic
{
public:

  // 私有成员函数
  void parse(boost::any &, const std::vector< std::string > &, bool) const;

  // 保护成员函数
  void xparse(boost::any &, const std::vector< std::string > &) const;
};

说明

辅助转换类，接受 ascii 字符串作为输入。覆盖 'parse' 方法并定义新的 'xparse' 方法接受 std::string。根据分析的输入是 ascii 还是 UTF8，决定照原样传递给 xparse ，或是执行 UTF8->ascii 的转换。

value_semantic_codecvt_helper 私有成员函数

  1. void parse(boost::any & value_store, 
               const std::vector< std::string > & new_tokens, bool utf8) const;

    分析指定一个选项值的一组 tokens。保存结果到 'value_store'，使用所希望的任何表示法。如果同一个选项被指定多次，则可能多次调用。
