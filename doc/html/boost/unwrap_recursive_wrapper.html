类模板 unwrap_recursive_wrapper

boost::unwrap_recursive_wrapper —

解开给定参数的包装，如果它是 recursive_wrapper 的特化。

摘要

template<typename T> 
class unwrap_recursive_wrapper {
public:
  // types
  typedef unspecified type;
};

说明

type 等同于 T::type 如果 T 是 recursive_wrapper 的一个特化。否则，type 等同于 T.
