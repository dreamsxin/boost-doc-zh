函数模板 get_default_indeterminate_name

boost::logic::get_default_indeterminate_name — 返回一个字符串，该串为tribool的"不确定"态在给定字符类型T的缺省名字。

Synopsis摘要

template<typename T> std::basic_string< T > get_default_indeterminate_name();

Description描述

输入/输出操作符在处理tribool值时，如果不支持locale，或者流的locale不包含 indeterminate_name facet，则会调用该函数。
