函数模板 operator>>

boost::logic::operator>> — 从流中读入一个tribool值。

Synopsis摘要

template<typename CharT, typename Traits> 
  std::basic_istream< CharT, Traits > & 
  operator>>(std::basic_istream< CharT, Traits > & in, tribool & x);

Description描述

当 (out.flags() & std::ios_base::boolalpha) == 0 时，本函数从输入流in中读入一个 long 值，并将该值转换为 tribool。如果值为0，则x 为 false ; 如果值为 1, 则x 为 true ; 如果值为 2, 则x 为 indetermine ; 其它情况下操作失败 (将输入流in 的 fail bit 置位)。

当 (out.flags() & std::ios_base::boolalpha) != 0 时，本函数首先确定 false, true, 和 indeterminate 值的名字。false 和 true 的名字来自于输入流的locale中的 std::numpunct facet (如果C++标准库的实现支持locales)，或者来自于 default_false_name 和 default_true_name 函数 (如果没有locale支持)。indeterminate 的名字则来自于相应的 indeterminate_name facet (如果在输入流的locale中可用)，或者来自于 get_default_indeterminate_name 函数 (如果C++标准库的实现不支持locales，或是该locale对象中没有指定 indeterminate_name facet)。然后将输入与这些名字进行匹配，并将 tribool x 赋值为匹配到的最长名字。如果没有一个名字可匹配，或者所有名字为空，则操作失败 (将输入流in 的 fail bit 置位)。

Returns:

in
