函数模板 from_ftime

boost::posix_time::from_ftime — 从一个已初始化的 FILETIME 结构创建一个 time 对象的函数。

概要

template<typename time_type> time_type from_ftime(const FILETIME & ft);

说明

这是一个从一个已初始化的 FILETIME 结构创建一个 time 对象的函数。FILETIME 结构以100纳秒为单位(0.0000001)保存时间。以微秒解析度进行构建时，将截断 FILETIME 的多余小数位。对于纳秒解析度则没有截断。

注意，ftime 是 Win32 API 的一部分，所以它不可移植到非windows平台。
