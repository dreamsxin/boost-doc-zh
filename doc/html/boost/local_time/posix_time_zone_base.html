Description

POSIX 时区字符串的形式为："std offset dst [offset],start[/time],end[/time]" (带或不带空格均可) 'std' 表示时区的缩写。'offset' 为距UTC的偏移量。'dst' 表示时区夏时制的缩写。第二个 offset 表示在DST时位移多少个小时。其缺省值=1 。'start' 和 'end' 是 DST 进入(和退出)的日期。'offset' 的形式为：[+|-]hh[:mm[:ss]] {h=0-23, m/s=0-59} 'time' 和 'offset' 的形式相同。Time 的缺省值=02:00:00 'start' 和 'end' 的形式有三种：Mm.w.d {month=1-12, week=1-5 (5 总表示最后一周), day=0-6} Jn {n=1-365 Feb29 不计在内} n {n=0-365 闰年时 Feb29 计算在内} 例如："PST-5PDT01:00:00,M4.1.0/02:00:00,M10.1.0/02:00:00" 在以下条件时将抛出异常：日期表示无效(请见 date 类)，以下情况下抛出一个 boost::local_time::bad_offset 异常：DST 开始或结束偏移量为负或大小24小时、UTC 时区大于 +12 或小于 -12 小时。以下情况下抛出一个 boost::local_time::bad_adjustment 异常：DST 调整量为24小时或更多(正或负)
