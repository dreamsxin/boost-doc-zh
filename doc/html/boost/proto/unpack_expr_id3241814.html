Function unpack_expr

boost::proto::unpack_expr — Construct an expression of the requested tag type with a domain and with children from the specified Fusion Random Access Sequence.
boost::proto::unpack_expr — 按领域及来自于指定的Fusion随机访问序列的子节点，构造一个请求标签类型的表达式。

Synopsis

template<typename Tag, typename Sequence> 
  typename proto::result_of::unpack_expr<Tag, Sequence const>::type const 
  unpack_expr(Sequence const & sequence);
template<typename Tag, typename Domain, typename Sequence> 
  typename proto::result_of::unpack_expr<Tag, Domain, Sequence const>::type const 
  unpack_expr(Sequence const & sequence);

Description

This function template may be invoked either with or without specifying a Domain argument. If no domain is specified, the domain is deduced by examining in order the domains of the elements of sequence and taking the first that is not proto::default_domain, if any such domain exists, or proto::default_domain otherwise.
这个函数模板可以带或不带一个 Domain 模板参数来调用。如果未指定领域，则按顺序由给定参数的领域来推断，如果存在，则使用第一个非 proto::default_domain 的领域作为其领域，否则使用 proto::default_domain 。

Let WRAP<N>(s), where s has type S, be defined such that:
令 WRAP<N>(s) 的定义如下，其中 s 的类型为 S：

  • If fusion::value_at<S,N>::type is a reference, WRAP<N>(s) is equivalent to proto::as_child<Domain>(fusion::at_c<N>(s)).
    如果 fusion::value_at<S,N>::type 为引用，则 WRAP<N>(s) 等价于 proto::as_child<Domain>(fusion::at_c<N>(s)).

  • Otherwise, WRAP<N>(s) is equivalent to proto::as_expr<Domain>(fusion::at_c<N>(s)).
    否则，WRAP<N>(s) 等价于 proto::as_expr<Domain>(fusion::at_c<N>(s)).

Let MAKE<Tag>(a...) be defined as proto::expr<Tag, proto::listN<A...> >::make(a...) where Ax is the type of ax.
令 MAKE<Tag>(a...) 定义为 proto::expr<Tag, proto::listN<A...> >::make(a...) 其中 Ax 为 ax 的类型。

Parameters:

sequence

A Fusion Random Access Sequence.
一个Fusion随机访问序列。

Returns:

Domain()(MAKE<Tag>(WRAP<0>(s),... WRAP<N-1>(s))), where N is the size of Sequence.
Domain()(MAKE<Tag>(WRAP<0>(s),... WRAP<N-1>(s))), 其中 N 为 Sequence 的大小。
