类模板 is_recursive_wrapper

boost::is_recursive_wrapper —

判断指定类型是否 recursive_wrapper 的特化。

摘要

template<typename T> 
class is_recursive_wrapper {
public:
  // types
  typedef unspecified type;

  // static constants
  static const bool value = unspecified;
};

说明

Value 为 true 当且仅当 T 是 recursive_wrapper 的一个特化。

注：is_recursive_wrapper 符合 MPL 的 IntegralConstant 概念。
