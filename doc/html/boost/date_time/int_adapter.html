说明

该类用于内部的日期/时间表示法。它增加了类似于浮点数的无限值及非数值的特性。它按照以下规则提供特殊值的数学运算操作： 

  +infinity  -  infinity  == Not A Number (NAN)
   infinity  *  non-zero  == infinity
   infinity  *  zero      == NAN
  +infinity  * -integer   == -infinity
   infinity  /  infinity  == NAN
   infinity  *  infinity  == infinity
