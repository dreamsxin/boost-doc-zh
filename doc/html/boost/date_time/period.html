类模板 period

boost::date_time::period — 提供用于日期-时间系统的泛型的时间段类型。

概要

template<typename point_rep, typename duration_rep> 
class period {
public:
  // types
  typedef point_rep    point_type;   
  typedef duration_rep duration_type;

  // construct/copy/destruct
  period(point_rep, point_rep);
  period(point_rep, duration_rep);

};

说明

该模板用一个类来表示时间段的时间点，用另一个类来表示时间段的长度。结果是，该类不适用于数值和长度表示相同的情况(如：普通的数值领域)。

时间段可以通过开始点加时长来指定，也可以通过开始点和结束点来指定(结束点不属于时间段的一部分，而是时间段之后的下一个时间单位)。一个时间段 是“无效的”，如果 end_point <= begin_point 或者给定的时长 <= 0. 任何有效的时间段的 is_null() 都返回 false.

零长度的时间段也就视为无效。零长度时间段是指开始点与结束点相同，或给定时长为零的时间段。对于零长度时间段，last 时间点比开始时间点前一个时间单位。

对于 begin 和 last 相同的情形，时间段长度为一个单位。

处理时间段的最好方法通常是提供开始点和时长。所以，day1 + 7 天就是一个一周的时间段，包含第一天和后六天(如：星期天到星期六)。

period 构造/复制/析构

  1. period(point_rep first_point, point_rep end_point);

    如果 end <= begin 则时间段无效。 

  2. period(point_rep first_point, duration_rep len);

    如果 len is <= 0 则时间段无效。
