类模板 wrapping_int

boost::date_time::wrapping_int — 一个可回绕的整数，用于支持时间长度(警告：只能以有符号类型实例化)。

概要

template<typename int_type_, int_type_ wrap_val> 
class wrapping_int {
public:
  // types
  typedef int_type_ int_type;

  // construct/copy/destruct
  wrapping_int(int_type);

  // public member functions
  int_type as_int() const;
  operator int_type() const;
  int_type add(int_type) ;
  int_type subtract(int_type) ;

  // public static functions
  static int_type wrap_value() ;

  // private member functions
  int_type calculate_wrap(int_type) ;
};

说明

在合成的日期和时间类型中，这个类型被用于回绕日期边界。例如：wrapping_int<short, 10> 将在9之后回绕，也会在小于0时回绕。它给出一个区间 [0,9]

注意：强烈建议用 wrapping_int2 来替代 wrapping_int，因为 wrapping_int 将在未来某个时间被反对使用。

还要注意，如果以无符号类型来实例化，将产生警告。只能使用有符号类型！

  4. int_type subtract(int_type v) ;

    返回值的符号表示回绕的方向(正号表示下回绕，负号表示上回绕)。例如：减一个负数且发生上回绕时，将返回一个负数来表示。如果发生下回绕，则返回正数。
