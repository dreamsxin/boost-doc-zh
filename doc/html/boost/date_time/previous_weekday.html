函数模板 previous_weekday

boost::date_time::previous_weekday — 生成一个日期对象，表示给定日期前一个周日的日期。

概要

template<typename date_type, typename weekday_type> 
  date_type previous_weekday(const date_type & d, const weekday_type & wd);

Description

生成一个日期对象，表示给定日期前一个周日的日期。如果给定的日期是 2004-May-9 (星期天)而给定的周日为星期二，则结果是 2004-May-4.
