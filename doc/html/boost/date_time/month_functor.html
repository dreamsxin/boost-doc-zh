Class template month_functor

boost::date_time::month_functor — 提供计算给定日期后第n个月的方法。

// In header: <boost/date_time/adjust_functors.hpp>

template<typename date_type> 
class month_functor {
public:
  // types
  typedef date_type::duration_type duration_type;

  month_functor(int);

  duration_type get_neg_offset(const date_type &) const;
};

Description

本函数提供了在‘年月日’历法上的‘基于月份’的递增逻辑。它处理不存在的月末日期的策略是，往回调整至月末。同时，如果开始的日期是月末，则该函数对象也会尝试将结果调整到月末 。
