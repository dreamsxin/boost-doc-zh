Struct c_time

boost::date_time::c_time — 为一些 'ctime' 函数提供统一接口。 

Synopsis

// In header: <boost/date_time/c_time.hpp>


struct c_time {

  // public static functions
  static std::tm * localtime(const std::time_t *, std::tm *) ;
  static std::tm * gmtime(const std::time_t *, std::tm *) ;
  static std::tm * localtime(const std::time_t *, std::tm *) ;
  static std::tm * gmtime(const std::time_t *, std::tm *) ;
};

Description

为一些 ctime 函数及其 '_r' 变体提供统一接口。其中 '_r' 函数需要一个指向用户创建的 std::tm 结构的指针，而普通的函数则使用一个静态创建的结构并返回指向该结构的指针。这些包装函数要求用户创建一个 std::tm 结构并传送一个指向它的指针。这个指向用户创建结构的指针也将被返回。所有函数都会适当地检查C函数的结果，并在错误时抛出异常。因此该函数永远不会返回NULL。
