函数模板 convert_string_type

boost::date_time::convert_string_type — 将一种 value_type 的字符串转换为另一种。

概要

template<typename InputT, typename OutputT> 
  std::basic_string< OutputT > 
  convert_string_type(const std::basic_string< InputT > & inp_str);

说明

将 wstring 转换为 string (或是将 string 转换为 wstring)。如果两个模板参数为同一类型，则返回输入字符串的拷贝。
