Function template parse_date

boost::date_time::parse_date — 分析一个带分隔符的日期(如：2002-02-10)的通用函数。

Synopsis

// In header: <boost/date_time/date_parsing.hpp>


template<typename date_type> 
  date_type parse_date(const std::string & s, int order_spec = ymd_order_iso);

Description

接受形如："2003-02-10" 或 " 2003-Feb-10" 或 "2003-Feburary-10" 的格式。参数字符串中的月、日、年顺序可以通过适当的 ymd_order_spec 来适应。
