Function template gather_month_strings

boost::date_time::gather_month_strings — 该函数从 std::locale 中收集所有月份的字符串。 

Synopsis

// In header: <boost/date_time/strings_from_facet.hpp>


template<typename charT> 
  std::vector< std::basic_string< charT > > 
  gather_month_strings(const std::locale & locale, bool short_strings = true);

Description

使用 time_put facet, 本函数从 locale 创建一组月份名字符串。在创建需要本地化的定制日期分析器或格式化器时非常方便。

Parameters:

locale

收集字符串时使用的 locale

short_strings

True(缺省) 表示收集短字符串，false 表示长字符串。

Returns:

一个字符串 vector，按顺序包含一组字符串。如：Jan, Feb, Mar, 等等。
