Function template parse_delimited_time_duration

boost::date_time::parse_delimited_time_duration — 从一个带分隔符的字符串创建一个 time_duration 对象。

Synopsis

// In header: <boost/date_time/time_parsing.hpp>


template<typename time_duration> 
  time_duration parse_delimited_time_duration(const std::string & s);

Description

期望的字符串格式是 "[-]h[h][:mm][:ss][.fff]". 如果提供的小数的位数大于时间长度类型的精度，则多余的位数将被截掉。

如果字符串的第一个字符是 '-'，则创建一个负的时长，其它 ‘-’ 则被视为分隔符。可接受的分隔符有 "-:,.".
