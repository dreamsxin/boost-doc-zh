说明

用于时间类型基于格式的输出的 facet. 这个类提供给输出时间的格式化字符串使用。除了用于日期元素格式化的标志外，还可以使用以下格式化标志：

  • x X => 缺省格式 - 可以和更多的缺省标志叠加(如："%x %X %z")

  • f => 小数秒 ".123456"

  • F => 小数秒或没有：和小数秒相似但如果小数秒 == 0 则为空

  • s => 带小数的秒 "02.123" (与 "%S%f” 相同)

  • S => 秒数 "02"

  • z => 时区缩写 "EDT"

  • Z => 时区全称 "Eastern Daylight Time"
