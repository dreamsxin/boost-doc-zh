类 any

boost::any — 一个类，其实例可以持有任意满足ValueType要求的类型的实例。

Synopsis摘要

class any {
public:
  // construct/copy/destruct
  any();
  any(const any &);
  template<typename ValueType> any(const ValueType &);
  any & operator=(const any &);
  template<typename ValueType> any & operator=(const ValueType &);
  ~any();

  // modifiers
  any & swap(any &);

  // queries

  const std::type_info & type() const;
};

Description描述

any 构造/复制/析构

  1. any();
    Postconditions: this->empty() 
  2. any(const any & other);
    Effects: 复制构造函数复制 other 的内容到一个新的实例中，因此 any 所含的类型和值都与 other的一样，如果 other 为空则any也为空。
    Throws: 可能会抛出一个 std::bad_alloc 异常，或者从所含类型的复制构造函数抛出的任意异常。 
  3. template<typename ValueType> any(const ValueType & value);
    Effects: 生成一个 value的拷贝，因此新实例的初始内容的类型和值与 value一致。
    Throws: std::bad_alloc 或者其它从所含类型的复制构造函数抛出的异常。 
  4. any & operator=(const any & rhs);
    Effects: 将 rhs 的内容复制到当前实例，放弃之前的内容，因此新内容的类型和值与 rhs一致，如果 rhs.empty()则也为空。
    Throws: std::bad_alloc 或者其它从所含类型的复制构造函数抛出的异常。赋值操作应满足强异常安全的保证。 
  5. template<typename ValueType> any & operator=(const ValueType & rhs);
    Effects: 生成一个 rhs的拷贝，放弃之前的内容，因此新内容的类型和值与 rhs一致。
    Throws: std::bad_alloc 或者其它从所含类型的复制构造函数抛出的异常。赋值操作应满足强异常安全的保证。 
  6. ~any();
    Effects: 释放 any 及所有用于实例管理的资源。
    Throws: 无

any 修改函数

  1. any & swap(any & rhs);
    Effects: 交换 *this 和 rhs的内容
    Returns: *this
    Throws: 无

any 查询函数

  1. bool empty() const;
    Returns: true 如果实例为空，否则返回 false.
    Throws: 不会抛出异常。 
  2. const std::type_info & type() const;
    Returns: 如果实例非空，则返回所含值的 typeid，否则返回 typeid(void).
    Notes: 用于在编译期或运行期查询相应类型。
