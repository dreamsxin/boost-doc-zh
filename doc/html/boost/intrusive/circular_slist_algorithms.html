Class template circular_slist_algorithms

boost::intrusive::circular_slist_algorithms

Synopsis

template<typename NodeTraits> 
class circular_slist_algorithms {
public:
  // types
  typedef NodeTraits::node           node;          
  typedef NodeTraits::node_ptr       node_ptr;      
  typedef NodeTraits::const_node_ptr const_node_ptr;
  typedef NodeTraits                 node_traits;   

  // public static functions
  static void init(node_ptr) ;
  static bool unique(const_node_ptr) ;
  static bool inited(const_node_ptr) ;
  static void unlink_after(node_ptr) ;
  static void unlink_after(node_ptr, node_ptr) ;
  static void link_after(node_ptr, node_ptr) ;
  static void transfer_after(node_ptr, node_ptr, node_ptr) ;
  static void init_header(node_ptr) ;
  static node_ptr get_previous_node(node_ptr, node_ptr) ;
  static node_ptr get_previous_node(node_ptr) ;
  static node_ptr get_previous_previous_node(node_ptr) ;
  static node_ptr get_previous_previous_node(node_ptr, node_ptr) ;
  static std::size_t count(const_node_ptr) ;
  static void unlink(node_ptr) ;
  static void link_before(node_ptr, node_ptr) ;
  static void swap_nodes(node_ptr, node_ptr) ;
  static void reverse(node_ptr) ;
  static node_ptr move_backwards(node_ptr, std::size_t) ;
  static node_ptr move_forward(node_ptr, std::size_t) ;
};

Description

circular_slist_algorithms provides basic algorithms to manipulate nodes forming a circular singly linked list. An empty circular list is formed by a node whose pointer to the next node points to itself.
circular_slist_algorithms 提供了操纵形成一个循环单链表的节点的基本算法。一个空的循环链表由一个 next 指针指向它自己的节点组成。

circular_slist_algorithms is configured with a NodeTraits class, which encapsulates the information about the node to be manipulated. NodeTraits must support the following interface:
circular_slist_algorithms 用一个 NodeTraits 类来配置，它封装了被操纵的节点的信息。NodeTraits 必须支持以下接口：

Typedefs:

node: The type of the node that forms the circular list 形成循环链表的节点的类型

node_ptr: A pointer to a node 节点指针

const_node_ptr: A pointer to a const node 常量节点指针

Static functions: 静态函数:

static node_ptr get_next(const_node_ptr n);

static void set_next(node_ptr n, node_ptr next);

circular_slist_algorithms public static functions

  1. static void init(node_ptr this_node) ;

    Effects: Constructs an non-used list element, putting the next pointer to null: NodeTraits::get_next(this_node) == 0
    作用:     构造一个未用的链表元素，将 next 指针置为空：NodeTraits::get_next(this_node) == 0。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。

  2. static bool unique(const_node_ptr this_node) ;

    Requires: this_node must be in a circular list or be an empty circular list.
    作用:     this_node 必须在一个循环链表或空循环链表中。

    Effects: Returns true is "this_node" is the only node of a circular list: or it's a not inserted node: return !NodeTraitsget_next(this_node) || NodeTraits::get_next(this_node) == this_node
    作用:     如果 "this_node" 是循环链表中的唯一节点，则返回 true：return !NodeTraitsget_next(this_node) || NodeTraits::get_next(this_node) == this_node。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。     

  3. static bool inited(const_node_ptr this_node) ;

    Effects: Returns true is "this_node" has the same state as if it was inited using "init(node_ptr)"
    作用:     如果 "this_node" 为由 "init(node_ptr)" 函数初始化的未用状态，则返回 true。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。     

  4. static void unlink_after(node_ptr prev_node) ;

    Requires: prev_node must be in a circular list or be an empty circular list.
    要求:     prev_node 必须是一个循环链表中的节点或者是一个空循环链表。

    Effects: Unlinks the next node of prev_node from the circular list.
    作用:     在循环链表中断开 prev_node 的下一个节点的链接。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。     

  5. static void unlink_after(node_ptr prev_node, node_ptr last_node) ;

    Requires: prev_node and last_node must be in a circular list or be an empty circular list.
    要求:     prev_node 和 last_node 必须是一个循环链表中的节点或者是一个空循环链表。

    Effects: Unlinks the range (prev_node, last_node) from the circular list.
    作用:     在循环链表中断开区间 (prev_node, last_node) 的链接。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。     

  6. static void link_after(node_ptr prev_node, node_ptr this_node) ;

    Requires: prev_node must be a node of a circular list.
    要求:     prev_node 必须是一个循环链表中的节点。

    Effects: Links this_node after prev_node in the circular list.
    作用:     在循环链表中将 this_node 链接到 prev_node 之后。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。     

  7. static void transfer_after(node_ptr p, node_ptr b, node_ptr e) ;

    Requires: b and e must be nodes of the same circular list or an empty range. and p must be a node of a different circular list.
    要求:     b 和 e 必须是同一个循环链表或空循环链表中的节点，且 p 必须是另一个循环链表的节点。

    Effects: Removes the nodes from (b, e] range from their circular list and inserts them after p in p's circular list.
    作用: 从循环链表断开    节点 [b, e) 的链接并将它们插入到 p 的循环链表中，位于 p 之后。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。     

  8. static void init_header(node_ptr this_node) ;

    Effects: Constructs an empty list, making this_node the only node of the circular list: NodeTraits::get_next(this_node) == this_node.
    作用: 构造一个空链表，    this_node 为循环链表中的唯一节点：NodeTraits::get_next(this_node) == this_node。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。     

  9. static node_ptr 
    get_previous_node(node_ptr prev_init_node, node_ptr this_node) ;

    Requires: this_node and prev_init_node must be in the same circular list.
    要求:     this_node 和 prev_init_node 必须是同一个循环链表的节点。

    Effects: Returns the previous node of this_node in the circular list starting. the search from prev_init_node. The first node checked for equality is NodeTraits::get_next(prev_init_node).
    作用: 返回     this_node 在循环链表中的前一个节点，从 prev_init_node 开始搜索。对第一个节点检查等式 NodeTraits::get_next(prev_init_node).

    Complexity: Linear to the number of elements between prev_init_node and this_node.
    复杂度: 与     prev_init_node 和 this_node 之间的元素数量成线性。

    Throws: Nothing.
    抛出: 无。     

  10. static node_ptr get_previous_node(node_ptr this_node) ;

    Requires: this_node must be in a circular list or be an empty circular list.
    要求:     this_node 必须是一个循环链表的节点或是一个空循环链表。

    Effects: Returns the previous node of this_node in the circular list.
    作用: 返回     this_node 在循环链表中的前一个节点。

    Complexity: Linear to the number of elements in the circular list.
    复杂度: 与循环链表中    的元素数量成线性。

    Throws: Nothing.
    抛出: 无。     

  11. static node_ptr get_previous_previous_node(node_ptr this_node) ;

    Requires: this_node must be in a circular list or be an empty circular list.
    要求:     this_node 必须是一个循环链表的节点或是一个空循环链表。

    Effects: Returns the previous node of the previous node of this_node in the circular list.
    作用: 返回     this_node 在循环链表中的前一个节点的前一个节点。

    Complexity: Linear to the number of elements in the circular list.
    复杂度: 与循环链表中    的元素数量成线性。

    Throws: Nothing.
    抛出: 无。     

  12. static node_ptr 
    get_previous_previous_node(node_ptr prev_prev_init_node, node_ptr this_node) ;

    Requires: this_node and prev_prev_init_node must be in the same circular list.
    要求:     this_node 和 prev_prev_init_node 必须是同一个循环链表的节点。

    Effects: Returns the previous node of the previous node of this_node in the circular list starting. the search from prev_init_node. The first node checked for equality is NodeTraits::get_next((NodeTraits::get_next(prev_prev_init_node)).
    作用: 返回     this_node 在循环链表中的前一个节点的前一个节点，从 prev_prev_init_node 开始搜索。对第一个节点检查等式 NodeTraits::get_next((NodeTraits::get_next(prev_prev_init_node)).

    Complexity: Linear to the number of elements in the circular list.
    复杂度: 与循环链表中    的元素数量成线性。

    Throws: Nothing.
    抛出: 无。     

  13. static std::size_t count(const_node_ptr this_node) ;

    Requires: this_node must be in a circular list or be an empty circular list.
    要求:     this_node 必须在一个循环链表中或是一个空循环链表。

    Effects: Returns the number of nodes in a circular list. If the circular list is empty, returns 1.
    作用:     返回循环链表中的节点数量。如果循环链表为空，返回 1。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。     

  14. static void unlink(node_ptr this_node) ;

    Requires: this_node must be in a circular list, be an empty circular list or be inited.
    要求:     this_node 必须在一个循环链表中或是一个空循环链表或是已初始化。

    Effects: Unlinks the node from the circular list.
    作用: 从循环链表断开该节点    的链接。

    Complexity: Linear to the number of elements in the circular list
    复杂度: 与循环链表中    的元素数量成线性。

    Throws: Nothing.
    抛出: 无。     

  15. static void link_before(node_ptr nxt_node, node_ptr this_node) ;

    Requires: nxt_node must be a node of a circular list.
    要求:     nxt_node 必须是一个循环链表中的节点。

    Effects: Links this_node before nxt_node in the circular list.
    作用:     在循环链表中将 this_node 链接到 nxt_node 之前。

    Complexity: Linear to the number of elements in the circular list.
    复杂度: 与循环链表中    的元素数量成线性。

    Throws: Nothing.
    抛出: 无。     

  16. static void swap_nodes(node_ptr this_node, node_ptr other_node) ;

    Requires: this_node and other_node must be nodes inserted in circular lists or be empty circular lists.
    要求:     this_node 和 other_node 必须是已插入一循环链表中的节点或者是空循环链表。

    Effects: Swaps the position of the nodes: this_node is inserted in other_nodes position in the second circular list and the other_node is inserted in this_node's position in the first circular list.
    作用:     交换两个节点的位置：this_node 插入到第二个循环链表中的 other_nodes 位置，而 other_node 插入到第一个循环链表中的 this_node 位置。

    Complexity: Linear to number of elements of both lists
    复杂度: 与两个链表中    的元素数量成线性。

    Throws: Nothing.
    抛出: 无。     

  17. static void reverse(node_ptr p) ;

    Effects: Reverses the order of elements in the list.
    作用: 倒转将链表中元素的顺序。

    Throws: Nothing.
    抛出: 无。

    Complexity: This function is linear to the contained elements.
    复杂度: 该函数与所含    元素数量成线性。 

  18. static node_ptr move_backwards(node_ptr p, std::size_t n) ;

    Effects: Moves the node p n positions towards the end of the list.
    作用: 将节点     p 向链表尾部移动 n 个位置。

    Returns: The previous node of p after the function if there has been any movement, Null if n leads to no movement.
    返回:     如果有移动，则返回在函数调用后 p 的前一个节点，如果 n 引致没有移动则返回 null。

    Throws: Nothing.
    抛出: 无。

    Complexity: Linear to the number of elements plus the number moved positions.
    复杂度: 与元素数量加上移动的位置数成线性。     

  19. static node_ptr move_forward(node_ptr p, std::size_t n) ;

    Effects: Moves the node p n positions towards the beginning of the list.
    作用: 将节点     p 向链表头部移动 n 个位置。

    Returns: The previous node of p after the function if there has been any movement, Null if n leads equals to no movement.
    返回:     如果有移动，则返回在函数调用后 p 的前一个节点，如果 n 引致没有移动则返回 null。

    Throws: Nothing.
    抛出: 无。

    Complexity: Linear to the number of elements plus the number moved positions.
    复杂度: 与元素数量加上移动的位置数成线性。
