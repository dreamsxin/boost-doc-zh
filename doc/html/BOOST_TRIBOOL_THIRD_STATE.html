宏 BOOST_TRIBOOL_THIRD_STATE

BOOST_TRIBOOL_THIRD_STATE — 为tribool的第三态声明一个新的名字。

Synopsis摘要

BOOST_TRIBOOL_THIRD_STATE(Name)

Description描述

使用这个宏可以为tribool的第三态声明一个新的名字。"不确定"态可以有任意多的名字(除 indeterminate 之外)，这些名字都是等价的。新的名字将被置于扩展该宏时所处的命名空间。

例子：BOOST_TRIBOOL_THIRD_STATE(true_or_false)

tribool x(true_or_false); // 潜在地设置 x 

if (true_or_false(x)) { // 不知道 x 是什么 }
