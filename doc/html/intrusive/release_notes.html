Boost 1.36 Release

Boost 1.36 Release

  • Added linear<> and cache_last<> options to singly linked lists.
    单链表增加了 linear<> 和 cache_last<> 选项。
  • Added optimize_multikey<> option to unordered container hooks.
    无序容器钩子增加了 optimize_multikey<> 选项。
  • Optimized unordered containers when store_hash option is used in the hook.
    当在钩子中使用 store_hash 选项时，对无序容器进行优化。
  • Implementation changed to be exception agnostic so that it can be used in environments without exceptions.
    实现改为异常不可知的，以便可在无异常的环境中使用。
  • Added container_from_iterator function to tree-based containers.
    基于树的容器增加了 container_from_iterator 函数。
