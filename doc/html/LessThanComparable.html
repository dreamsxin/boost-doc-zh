Concept LessThanComparable 可比较大小概念

LessThanComparable 可比较大小

Description 说明

LessThanComparable types must have <, >, <=, and >= operators.
可比较大小的类型必须有 <, >, <=, 和 >= 操作符。

Notation 符号

X
A type playing the role of comparable-type in the LessThanComparable concept.
在 可比较大小 概念中担任可比较类型角色的类型。
x, y
Objects of type X
类型 X 的对象
