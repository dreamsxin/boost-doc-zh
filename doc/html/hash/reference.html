支持函数（Boost 扩展）

  1. template<typename T> void hash_combine(size_t & seed, T const& v);

    重复地调用，从几个变量递增地创建一个散列值。

    效果：

    		 
    seed ^= hash_value(v) + 0x9e3779b9 + (seed << 6) + (seed >> 2);

    注意：

    hash_value 被调用是不带限定的，以便可以通过 ADL 找到重载。

    这是一个对 TR1 的扩展

    抛出：

    		 只有 hash_value(T) 抛出时，才会抛出。只要 hash_value(T) 具有强异常安全，它就具有强异常安全。 

  2. template<typename It> std::size_t hash_range(It first, It last);
    template<typename It> void hash_range(std::size_t& seed, It first, It last);

    计算某个迭代器区间的各元素的复合散列值。

    效果：

    对于双参数重载版本：

    size_t seed = 0;

    for(; first != last; ++first)
    {
     hash_combine(seed, *first);
    }

    return seed;

    对于三参数重载版本：

    for(; first != last; ++first)
    {
     hash_combine(seed, *first);
    }

    注意：

    hash_range 是受元素的顺序影响的，所以将它用于无序容器是不合适的。

    这是一个对 TR1 的扩展

    抛出：

    只有 hash_value(std::iterator_traits<It>::value_type) 抛出时，才会抛出。只要 hash_value(std::iterator_traits<It>::value_type) 具有基本异常安全，hash_range(std::size_t&, It, It) 就具有基本异常安全。

    散列函数的实现。

    通常不应该由用户直接调用，他们应该使用 boost::hash, boost::hash_range 或 boost::hash_combine， 它们以不带名字空间限定的方式调用 hash_value， 以便通过 ADL 找到对定制类型的重载。

    注意：

    这是一个对 TR1 的扩展

    抛出：

    		 只有用户使用的 hash_value 版本对于一个容器的元素或储存在一个 pair 中的类型会抛出时，才会抛出。

    返回：
    类型 返回
    bool, char, signed char, unsigned char, wchar_t, short, unsigned short, int, unsigned int, long, unsigned long val
    long long, unsigned long long val when abs(val) <= std::numeric_limits<std::size_t>::max().
    float, double, long double 一个不定的值，但是对于相等的参数应该导致同样的结果。
    T* 一个不定的值，但是对于相等的参数应该导致同样的结果。
    T val[N], const T val[N] hash_range(val, val+N)
    std:basic_string<Ch, std::char_traits<Ch>, A>, std::vector<T, A>, std::list<T, A>, std::deque<T, A>, std::set<K, C, A>, std::multiset<K, C, A>, std::map<K, T, C, A>, std::multimap<K, T, C, A> hash_range(val.begin(), val.end())
    std::pair<A, B> 
    size_t seed = 0;
    hash_combine(seed, val.first);
    hash_combine(seed, val.second);
    return seed;
    std::complex<T> 当 T 是一个内建类型，且 val.imag() == 0， 结果与 hash_value(val.real()) 相等。否则是一个不定的值，但是对于相等的参数应该导致同样的结果。
