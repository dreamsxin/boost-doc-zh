BoostBook element boostbook

boostbook — Defines a BoostBook book
boostbook — 定义一本 BoostBook 书

Synopsis 概要

boostbook ::= (title, (chapter| library)*)

Description 说明

This element is the topmost level defined by boost.xml for all Boost documentation. It will not generally be used by developers.
对于所有 Boost 文档，这一元素是由 boost.xml 定义的最顶层元素。它通常不被开发者使用。

Attributes 属性

Name 名字 Type 类型 Value 值 Purpose 用途
last-revision #IMPLIED CDATA Set to $Date: 2008-06-15 15:48:40 -0400 (Sun, 15 Jun 2008) $ to keep "last revised" information in sync with CVS changes
设置为 $Date: 2008-06-15 15:48:40 -0400 (Sun, 15 Jun 2008) $，以保持 "最后修订" 信息与CVS的变化同步
id #IMPLIED CDATA A global identifier for this element
该元素的全局标识符
xml:base #IMPLIED CDATA Implementation detail used by XIncludes
用于 XIncludes 的实现细节
