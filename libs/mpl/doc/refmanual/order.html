order

摘要

template<
      typename Sequence
    , typename Key
    >
struct order
{
    typedef unspecified type;
};

描述

返回一个唯一的无符号 整型常量，它与 Sequence 中的键值 Key 相关联。

头文件

#include <boost/mpl/order.hpp>

Model of

Tag 分派元函数

参数

参数 要求 描述
Sequence 关联序列 被查询的序列
Key 任意类型 查询的键值

表达式语义

对于任意 关联序列 s, 以及任意类型 key:

typedef order<s,key>::type n;
返回类型: 无符号 整型常量
语义: 如果 has_key<s,key>::value == true, 则 n 为一个唯一的无符号 整型常量，与 s 中的 key 相关联; 否则，n 等价于 void_.

复杂度

分期常量时间。

例子

typedef map< pair<int,unsigned>, pair<char,long> > m;

BOOST_MPL_ASSERT_NOT(( is_same< order<m,int>::type, void_ > ));
BOOST_MPL_ASSERT(( is_same< order<m,long>::type,void_ > ));

参见

关联序列, has_key, count, map
