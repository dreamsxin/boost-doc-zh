min_element

摘要

template<
      typename Sequence
    , typename Pred = less<_1,_2>
    >
struct min_element
{
    typedef unspecified type;
};

描述

返回指向 Sequence 中最小元素的迭代器。

头文件

#include <boost/mpl/min_element.hpp>

参数

参数 要求 描述
Sequence 前向序列 查找的序列
Pred 二元 Lambda 表达式 比较标准

表达式语义

对于任意 前向序列 s 以及二元 Lambda 表达式 pred:

typedef min_element<s,pred>::type i;
返回类型: 前向迭代器
语义:

i 为 [begin<s>::type, end<s>::type) 中第一个满足以下条件的迭代器，对于  [begin<s>::type, end<s>::type) 中的每一个迭代器 j,

apply< pred, deref<j>::type, deref<i>::type >::type::value == false

复杂度

线性。零次比较如果 s 为空，否则刚好 size<s>::value - 1 次比较。

例子

typedef vector<bool,char[50],long,double> types;
typedef min_element<
      transform_view< types,sizeof_<_1> >
    >::type iter;

BOOST_MPL_ASSERT(( is_same< deref<iter::base>::type, bool> ));

参见

查询算法, max_element, find_if, upper_bound, find
