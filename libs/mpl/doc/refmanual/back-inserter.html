back_inserter

摘要

template<
      typename Seq
    >
struct back_inserter
{
    // unspecified
    // ...
};

描述

在序列的后端插入元素。

头文件

#include <boost/mpl/back_inserter.hpp>

Model of

插入器

参数

参数 要求 描述
Seq 后端可扩展序列 绑定到插入器的序列

表达式语义

以下仅列出在 插入器 中没有定义或定义不同的表达式语义。.

对于任意 后端可扩展序列 s:

表达式 语义
back_inserter<s>

一个 插入器 in, 等价于

struct in : inserter<s,push_back<_1,_2> > {};

复杂度

分期常量时间。

例子

typedef copy<
      range_c<int,5,10>
    , back_inserter< vector_c<int,0,1,2,3,4> >
    >::type range;
   
BOOST_MPL_ASSERT(( equal< range, range_c<int,0,10> > ));
