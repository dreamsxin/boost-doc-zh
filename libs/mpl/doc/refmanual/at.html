at

摘要

template<
      typename Sequence
    , typename N
    >
struct at
{
    typedef unspecified type;
};

template<
      typename AssocSeq
    , typename Key
    , typename Default = unspecified
    >
struct at
{
    typedef unspecified type;
};

描述

at 是一个 重载的名字：

  • at<Sequence,N> 返回 前向序列 Sequence 中从开头计第 N 个元素。
  • at<AssocSeq,Key,Default> 返回 关联序列 AssocSeq 中与 Key 关联的第一个元素，如果没有这样的元素则返回 Default.

头文件

#include <boost/mpl/at.hpp>

Model of

Tag 分派元函数

参数

参数 要求 描述
Sequence 前向序列 被检查的序列
AssocSeq 关联序列 被检查的序列
N 整型常量 从序列的开头计，要取出的元素的位移量
Key 任意类型 要取出的元素的键
Default 任意类型 当满足条件的元素不存在时返回的缺省值

表达式语义

对于任意的 前向序列 s, 和 整型常量 n:

typedef at<s,n>::type t;
返回类型: 一个类型
后置条件: 0 <= n::value < size<s>::value.
语义:

等价于

typedef deref< advance< begin<s>::type,n >::type >::type t;

对于任意的 关联序列 s, 和任意类型 key 和 x:

typedef at<s,key,x>::type t;
返回类型: 一个类型
语义: 如果 has_key<s,key>::value == true, 则 t 为与 key 关联的值类型; 否则 t 等同于 x. 
typedef at<s,key>::type t;
返回类型: 一个类型
语义:

等同于

typedef at<s,key,void_>::type t;

复杂度

序列原型 复杂度
前向序列 线性
随机访问序列 分期常量时间
关联序列 分期常量时间

例子

typedef range_c<long,10,50> range;
BOOST_MPL_ASSERT_RELATION( (at< range, int_<0> >::value), ==, 10 );
BOOST_MPL_ASSERT_RELATION( (at< range, int_<10> >::value), ==, 20 );
BOOST_MPL_ASSERT_RELATION( (at< range, int_<40> >::value), ==, 50 );
typedef set< int const,long*,double > s;

BOOST_MPL_ASSERT(( is_same< at<s,char>::type, void_ > ));
BOOST_MPL_ASSERT(( is_same< at<s,int>::type, int > ));
