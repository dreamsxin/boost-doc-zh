BOOST_MPL_ASSERT_MSG

摘要

#define BOOST_MPL_ASSERT_MSG( condition, message, types ) \
    unspecified token sequence \
/**/

描述

当条件不满足时生成一个包含客户化信息的编译错误。

头文件

#include <boost/mpl/assert.hpp>

参数

参数 要求 描述
condition 整型常量表达式 进行断言的条件
message 合法的标识符 以一个合法的C++标识符格式给出的客户化信息
types 合法的函数参数列表 用括号括起来的类型列表，将被显示在错误信息中

表达式语义

对于任意整型常量表达式 expr, 合法 C++ 标识符 message, 以及任意类型 t1, t2,... tn:

BOOST_MPL_ASSERT_MSG( expr, message, (t1, t2,... tn) );
返回类型: 无
前置条件: t1, t2,... tn 为非-void.
语义:

生成一个编译错误，如果 expr::value != true, 否则没有作用。

如果编译器的诊断能力可以支持，错误信息将包含 message 标识符以及 t1, t2,... tn 类型的列表，并具有通用的格式：

... ************( ...::message )************)(t1, t2,... tn) ...
      
BOOST_MPL_ASSERT_MSG( expr, message, (types<t1, t2,... tn>) );
返回类型: 无
前置条件: 无
语义:

生成一个编译错误，如果 expr::value != true, 否则没有作用。

如果编译器的诊断能力可以支持，错误信息将包含 message 标识符以及 t1, t2,... tn 类型的列表，并具有通用的格式：

... ************( ...::message )************)(types<t1, t2,... tn>) ...

例子

template< typename T > struct my
{
    // ...
    BOOST_MPL_ASSERT_MSG( 
          is_integral<T>::value
        , NON_INTEGRAL_TYPES_ARE_NOT_ALLOWED
        , (T)
        );
};

my<void*> test;

// In instantiation of `my<void*>':
//   instantiated from here
// conversion from `
//   mpl_::failed************(my<void*>::
//   NON_INTEGRAL_TYPES_ARE_NOT_ALLOWED::************)(void*)
//   ' to non-scalar type `mpl_::assert<false>' requested
