宏

作为一个模板元编程的框架，MPL专注于把一件事做好，而将与预处理相关的工作留给相关的特定库 [PRE], [Ve03] 来完成。但是不论我们是否喜欢，在今天的C++元编程中，宏还是扮演着一个重要的角色，而且如果不将预处理相关的实现作为本库的公有接口开放，那么某些有用的MPL级功能将无法实现。

[PRE] Vesa Karvonen, Paul Mensonides, The Boost Preprocessor Metaprogramming library
[Ve03] Vesa Karvonen, The Order Programming Language, 2003.
