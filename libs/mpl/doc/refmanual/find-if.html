find_if

摘要

template<
      typename Sequence
    , typename Pred
    >
struct find_if
{
    typedef unspecified type;
};

摘要

返回一个迭代器，指向 Sequence 中第一个满足谓词 Pred 的元素。

头文件

#include <boost/mpl/find_if.hpp>

参数

参数 要求 描述
Sequence 前向序列 要查找的序列
Pred 单参 Lambda 表达式 查询条件

表达式语义

对于任意 前向序列 s 以及单参 Lambda 表达式 pred:

typedef find_if<s,pred>::type i;
返回类型: 前向迭代器
语义:

i 是区间 [begin<s>::type, end<s>::type) 中第一个满足

apply< pred,deref<i>::type >::type::value == true

的迭代器。如果这样的迭代器不存在，i 等同于 end<s>::type.

复杂度

线性。最多 size<s>::value 次对 pred 的调用。

例子

typedef vector<char,int,unsigned,long,unsigned long> types;
typedef find_if<types, is_same<_1,unsigned> >::type iter;

BOOST_MPL_ASSERT(( is_same< deref<iter>::type, unsigned > ));
BOOST_MPL_ASSERT_RELATION( iter::pos::value, ==, 2 );
