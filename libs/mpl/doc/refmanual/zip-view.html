zip_view

摘要

template<
      typename Sequences
    >
struct zip_view
{
    // unspecified
    // ...
};

描述

提供多个序列的 "压缩" 视图；即将多个序列表示为一个序列，其中每个元素为由各个 Sequences 的对应元素组成的序列。

头文件

#include <boost/mpl/zip_view.hpp>

Model of

  • 前向序列

参数

参数 要求 描述
Sequences 由多个 前向序列 组成的 前向序列 被"压缩"的序列

表达式语义

以下只列出 前向序列 中没有定义或定义不同的表达式语义。

在下表中，v 是一个 zip_view 实例，seq 是一个由 n 个 前向序列 组成的 前向序列。

表达式 语义 
zip_view<seq>
zip_view<seq>::type
 一个惰性 前向序列 v，对于区间 [begin<v>::type, end<v>::type) 中的每个 i 和区间 [begin<seq>::type, end<seq>::type) 中的每个 j，deref<i>::type 等同于 transform< deref<j>::type, deref<_1> >::type.
size<v>::type

v 的大小; size<v>::value 等于

deref< min_element<
      transform_view< seq, size<_1> >
    >::type >::type::value;

线性复杂度；请见 前向序列。

例子

对三个向量中的对应元素求和。

typedef vector_c<int,1,2,3,4,5> v1;
typedef vector_c<int,5,4,3,2,1> v2;
typedef vector_c<int,1,1,1,1,1> v3;

typedef transform_view<
      zip_view< vector<v1,v2,v3> >
    , unpack_args< plus<_1,_2,_3> >
    > sum;

BOOST_MPL_ASSERT(( equal< sum, vector_c<int,7,7,7,7,7> > ));
