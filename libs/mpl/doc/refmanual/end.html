end

摘要

template<
      typename X
    >
struct end
{
    typedef unspecified type;
};

描述

返回序列的 past-the-end 迭代器。如果其参数不是一个 前向序列, 返回 void_.

头文件

#include <boost/mpl/begin_end.hpp>

Model of

Tag 分派元函数

参数

参数 要求 描述
X 任意类型 要返回其 end 迭代器的类型

表达式语义

对于任意类型 x:

typedef end<x>::type last;
返回类型: 前向迭代器 或 void_.
语义: 如果 x 为 前向序列, last 是一个指向 s 最后一个元素之后的迭代器; 否则 last 为 void_.
后置条件: 如果 last 是一个迭代器，则它是 past-the-end.

复杂度

分期常量时间。

例子

typedef vector<long> v;
typedef begin<v>::type first;
typedef end<v>::type last;

BOOST_MPL_ASSERT(( is_same< next<first>::type, last > ));

参见

迭代器, 前向序列, begin, end, next
