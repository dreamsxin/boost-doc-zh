for_each

摘要

template<
      typename Sequence
    , typename F
    >
void for_each( F f );

template<
      typename Sequence
    , typename TransformOp
    , typename F
    >
void for_each( F f );

描述

for_each 是一族重载函数模板：

  • for_each<Sequence>( f ) 将运行期函数对象 f 应用于区间 [begin<Sequence>::type, end<Sequence>::type) 中的每个元素。
  • for_each<Sequence,TransformOp>( f ) 将运行期函数对象 f 应用于对区间 [begin<Sequence>::type, end<Sequence>::type) 中每个元素执行 TransformOp 转化后所得到的结果区间。

头文件

#include <boost/mpl/for_each.hpp>

参数

参数 要求 描述
Sequence 前向序列 迭代的序列
TransformOp Lambda 表达式 转化操作
f 单参函数对象 调用的运行期操作

表达式语义

对于任意 前向序列 s, Lambda 表达式 op , 以及 单参函数对象 f:

for_each<s>( f );
返回类型: void
后置条件:

等价于

typedef begin<Sequence>::type i1;
value_initialized< deref<i1>::type > x1;
f(boost::get(x1));

typedef next<i1>::type i2;
value_initialized< deref<i2>::type > x2;
f(boost::get(x2));
...
value_initialized< deref<in>::type > xn;
f(boost::get(xn));
typedef next<in>::type last;

其中 n == size<s>::value 且 last 等同于 end<s>::type; 如果 empty<s>::value == true 则没有作用

 
for_each<s,op>( f );
返回类型: void
后置条件:

等价于

for_each< tranform_view<s,op> >( f );

复杂度

线性。刚好 size<s>::value 次对 op 和 f 的调用。

例子

struct value_printer
{
    template< typename U > void operator()(U x)
    {
        std::cout << x << '\n';
    }
};

int main()
{
    for_each< range_c<int,0,10> >( value_printer() );
}
