empty

摘要

template<
      typename Sequence
    >
struct empty
{
    typedef unspecified type;
};

描述

返回一个 整型常量 c 满足 c::value == true 当且仅当序列为空。

头文件

#include <boost/mpl/empty.hpp>

Model of

Tag 分派元函数

参数

参数 要求 描述
Sequence 前向序列 被测试的序列

表达式语义

对于任意 前向序列 s:

typedef empty<s>::type c;
返回类型: 布尔 整型常量
语义: 等价于 typedef is_same< begin<s>::type,end<s>::type >::type c;.
后置条件: empty<s>::value == ( size<s>::value == 0 ).

复杂度

分期常量时间。

例子

typedef range_c<int,0,0> empty_range;
typedef vector<long,float,double> types;

BOOST_MPL_ASSERT( empty<empty_range> );
BOOST_MPL_ASSERT_NOT( empty<types> );

参见

前向序列, 整型常量, size, begin / end
