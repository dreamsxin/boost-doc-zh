count

摘要

template<
      typename Sequence
    , typename T
    >
struct count
{
    typedef unspecified type;
};

描述

返回 Sequence 中等同于 T 的元素数量。

头文件

#include <boost/mpl/count.hpp>

参数

参数 要求 描述
Sequence 前向序列 被检查的序列
T 任意类型 查找的类型

表达式语义

对于任意 前向序列 s 和任意类型 t:

typedef count<s,t>::type n;
返回类型: 整型常量
语义:

等价于

typedef count_if< s,is_same<_,T> >::type n;

复杂度

线性。刚好 size<s>::value 次比较。

例子

typedef vector<int,char,long,short,char,short,double,long> types;
typedef count<types, short>::type n;

BOOST_MPL_ASSERT_RELATION( n::value, ==, 2 );

参见

查询算法, count_if, find, find_if, contains, lower_bound
