BOOST_MPL_ASSERT

摘要

#define BOOST_MPL_ASSERT( pred ) \
    unspecified token sequence \
/**/

描述

在谓词 pred 为假时生成一个编译错误。

头文件

#include <boost/mpl/assert.hpp>

参数

参数 要求 描述
pred 布尔无参 元函数 被用于断言的谓词

表达式语义

对于任意布尔无参 元函数 pred:

BOOST_MPL_ASSERT(( pred ));
返回类型: 无
语义:

生成一个编译错误，如果 pred::type::value != true, 否则没有作用。注意，即使条件中没有逗号也要求使用两重括号。

如果编译器的诊断能力可以支持，错误信息将包含谓词的完整类型名，并具有通用的格式：

... ************ pred::************ ...

例子

template< typename T, typename U > struct my
{
    // ...
    BOOST_MPL_ASSERT(( is_same< T,U > ));
};

my<void*,char*> test;

// In instantiation of `my<void, char*>':
//   instantiated from here
// conversion from `
//   mpl_::failed************boost::is_same<void, char*>::************' to 
//   non-scalar type `mpl_::assert<false>' requested

参见

断言, BOOST_MPL_ASSERT_NOT, BOOST_MPL_ASSERT_MSG, BOOST_MPL_ASSERT_RELATION
