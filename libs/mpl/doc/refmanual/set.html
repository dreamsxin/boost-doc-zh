set

描述

set 是一个 variadic, 关联, 可扩展 的类型序列，它支持常量时间的元素插入、删除和成员资格测试。set 中的每个键最多有一个元素。

头文件

序列形式 头文件
Variadic #include <boost/mpl/set.hpp>
Numbered #include <boost/mpl/set/setn.hpp>

Model of

  • Variadic 序列
  • 关联序列
  • 可扩展关联序列

表达式语义

在下表中，s 是一个 set 实例，pos 是 s 中的一个迭代器，而 x, k, 和 t1,t2,... tn 为任意类型。

表达式 语义 
set<t1,t2,... tn>
setn<t1,t2,... tn>
 set 元素为 t1,t2,... tn；请见 Variadic 序列 
set<t1,t2,... tn>::type
setn<t1,t2,... tn>::type
 等同于 setn<t1,t2,... tn>；请见 Variadic 序列
begin<s>::type 指向 s 头部的迭代器；请见 关联序列
end<s>::type 指向 s 尾部的迭代器；请见 关联序列
size<s>::type s 的大小；请见 关联序列
empty<s>::type 布尔 整型常量 c 满足 c::value == true 当且仅当 s 为空；请见 关联序列
front<s>::type s 中的第一个元素；请见 关联序列
has_key<s,k>::type 布尔 整型常量 c 满足 c::value == true 当且仅当 s 中有一个或多个元素的键为 k；请见 关联序列
count<s,k>::type 在 s 中键值为 k 的元素数量；请见 关联序列
order<s,k>::type 唯一的无符号 整型常量，与 s 中的键值 k 关联；请见 关联序列 
at<s,k>::type
at<s,k,def>::type
 在 s 中与键值 k 相关联的元素；请见 关联序列
key_type<s,x>::type 等同于 x；请见 关联序列
value_type<s,x>::type 等同于 x；请见 关联序列
insert<s,x>::type

一个新的 set 等价于 s 除了

at< t, key_type<s,x>::type >::type

等同于 value_type<s,x>::type.
insert<s,pos,x>::type 等同于 insert<s,x>::type; pos 被忽略
erase_key<s,k>::type 一个新的 set 等价于 s 除了 has_key<t, k>::value == false.
erase<s,pos>::type 等价于 erase<s, deref<pos>::type >::type.
clear<s>::type 一个空的 set；请见 clear.

例子

typedef set< int,long,double,int_<5> > s;

BOOST_MPL_ASSERT_RELATION( size<s>::value, ==, 4 );
BOOST_MPL_ASSERT_NOT(( empty<s> ));

BOOST_MPL_ASSERT(( is_same< at<s,int>::type, int > ));
BOOST_MPL_ASSERT(( is_same< at<s,long>::type, long > ));
BOOST_MPL_ASSERT(( is_same< at<s,int_<5> >::type, int_<5> > ));
BOOST_MPL_ASSERT(( is_same< at<s,char>::type, void_ > ));
