erase_key

摘要

template<
      typename AssocSeq
    , typename Key
    >
struct erase_key
{
    typedef unspecified type;
};

描述

删除 可扩展关联序列 AssocSeq 中与键 Key 相关联的元素。

头文件

#include <boost/mpl/erase_key.hpp>

Model of

Tag 分派元函数

参数

参数 要求 描述
AssocSeq 可扩展关联序列 要删除元素的序列
Key 任意类型 要删除的元素的键

表达式语义

对于任意 可扩展关联序列 s, 以及任意类型 key:

typedef erase_key<s,key>::type r;
返回类型: 可扩展关联序列
语义: r 为 概念上等同 且等价于 s 除了 has_key<r,k>::value == false.
后置条件: size<r>::value == size<s>::value - 1.

复杂度

分期常量时间。

例子

typedef map< pair<int,unsigned>, pair<char,long> > m;
typedef erase_key<m,char>::type m1;

BOOST_MPL_ASSERT_RELATION( size<m1>::type::value, ==, 1 );
BOOST_MPL_ASSERT(( is_same< at<m1,char>::type,void_ > ));
BOOST_MPL_ASSERT(( is_same< at<m1,int>::type,unsigned > ));

参见

可扩展关联序列, erase, has_key, insert
