关联序列

描述

关联序列 是一种 前向序列，它可以基于键值高效地取出元素。与C++标准库中的关联容器不同，MPL关联序列没有关联的排序关系。取而代之的是，使用 类型标识 来实现键值上的等价关系，且在迭代中对序列元素的遍因历顺序是未指定的。

定义

  • 键 是元素类型的一部分，用于标识和取出序列中的元素。
  • 值 是元素类型的一部分，它通过键从序列中取出。

表达式要求

在下表以及后面的说明中，s 是一个 关联序列，x 是序列中的一个元素，而 k 和 def 是任意的类型。

除了 前向序列 中所定义的表达式以外，必须满足以下：

表达式 类型 复杂度
has_key<s,k>::type 布尔 整型常量 分期常量时间
count<s,k>::type 整型常量 分期常量时间
order<s,k>::type 整型常量 或 void_ 分期常量时间
at<s,k>::type 任意类型 分期常量时间
at<s,k,def>::type 任意类型 分期常量时间
key_type<s,x>::type 任意类型 分期常量时间
value_type<s,x>::type 任意类型 分期常量时间

表达式语义

以下只列出在 前向序列 中没有定义或定义不一样的表达式语义。

表达式 语义
has_key<s,k>::type 布尔 整型常量 c 满足 c::value == true 当且仅当 s 中有一个或多个元素的键为 k；请见 has_key.
count<s,k>::type s 中键为 k 的元素数量；请见 count.
order<s,k>::type 唯一的无符号 整型常量，与序列 s 中的键 k 关联；请见 order. 
at<s,k>::type
at<s,k,def>::type
 序列 s 中的第一个键为 k 的元素；请见 at.
key_type<s,x>::type 元素 x 中用于在 s 中标识 x 的部分；请见 key_type.
value_type<s,x>::type 元素 x 中用于在 s 中的 x 的部分；请见 value_type.

Models

  • set
  • map

参见

序列，可扩展关联序列，has_key, count, order, at, key_type, value_type
