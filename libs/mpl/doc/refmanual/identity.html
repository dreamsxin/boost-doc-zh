identity

摘要

template< 
      typename X
    >
struct identity
{
    typedef X type;
};

描述

identity 元函数返回 X 本身。

头文件

#include <boost/mpl/identity.hpp>

Model of

元函数

参数

参数 要求 描述
X 任意类型 被返回的参数

表达式语义

对于任意类型 x:

typedef identity<x>::type r;
返回类型: 一个类型
语义:

等价于

typedef x r;
后置条件: is_same<r,x>::value == true.

例子

typedef apply< identity<_1>, char >::type t1;
typedef apply< identity<_2>, char,int >::type t2;

BOOST_MPL_ASSERT(( is_same< t1, char > ));
BOOST_MPL_ASSERT(( is_same< t2, int > ));

参见

元函数, 占位符, 平凡元函数, always, apply
