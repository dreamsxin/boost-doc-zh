前向迭代器

描述

前向迭代器 i 是一个类型，它表示了一个 前向序列 中的某个元素的位置引用。可以通过提领操作一访问该元素，并提供了获得序列中下一个元素的迭代器的方法。

定义

  • 一个迭代器可以被提领，意即 deref<i>::type 是一个有良好定义的表达式。
  • 一个迭代器是 past-the-end 的如果它指向某个序列最后一个元素之生；past-the-end 迭代器是不可提领的。
  • 一个迭代器 i 是 可递增的，如果存在一个 "next" 迭代器，即如果 next<i>::type 表达式是有良好定义的；past-the-end 迭代器是不可递增的。
  • 同一序列中的两个迭代器是 等价的，如果它们具有相同的类型。
  • 一个迭代器 j 可以从另一个迭代器 i 到达，如果在对 i 调用 next 元函数有限次后，i 等价于 j.
  • 记号 [i,j) 表示一个迭代器 区间，从 i 开始，直至但不包含 j.
  • 区间 [i,j) 是一个 有效区间，如果 j 可从 i 到达。

表达式要求

表达式 类型 复杂度
deref<i>::type 任意类型 分期常量时间
next<i>::type 前向迭代器 分期常量时间
i::category 整型常量, 可转换为 forward_iterator_tag 常量时间

表达式语义

typedef deref<i>::type j;
前置条件: i 是可提领的
语义: j 是与被指元素相同的类型 
typedef next<i>::type j;
前置条件: i 是可递增的
语义: j 是序列中的下一个迭代器
后置条件: j 是可提领的或 past-the-end 
typedef i::category c;
语义: c 是与迭代器的类别 tag 相同的类型

不变式

对于任意前向迭代器 i 和 j，有以下不变式：

  • i 和 j 相等当且仅当它们指向同一个元素。
  • 如果 i 是可提领的，且 j 等于 i, 则 j 也是可提领的。
  • 如果 i 和 j 相等且可提领，则 deref<i>::type 和 deref<j>::type 相同。
  • 如果 i 是可递增的，且 j 等于 i, 则 j 也是可递增的。
  • 如果 i 和 j 相等且可递增，则 next<i>::type 和 next<j>::type 相同。
