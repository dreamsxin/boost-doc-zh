equal

摘要

template<
      typename Seq1
    , typename Seq2
    , typename Pred = is_same<_1,_2>
    >
struct equal
{
    typedef unspecified type;
};

描述

返回一个真值 整型常量 如果两个序列 Seq1 和 Seq2 逐个元素比较时是相同的。

头文件

#include <boost/mpl/equal.hpp>

参数

参数 要求 描述
Seq1, Seq2 前向序列 比较的序列
Pred 二元 Lambda 表达式 比较标准

表达式语义

对于任意 前向序列 s1 和 s2 以及二元 Lambda 表达式 pred:

typedef equal<s1,s2,pred>::type c;
返回类型: 整型常量
语义:

c::value == true 当且仅当 size<s1>::value == size<s2>::value 且对于 [begin<s1>::type, end<s1>::type) 中的每一个迭代器 i，deref<i>::type 等同于

advance< begin<s2>::type, distance< begin<s1>::type,i >::type >::type

复杂度

线性。最多 size<s1>::value 次比较。

例子

typedef vector<char,int,unsigned,long,unsigned long> s1;
typedef list<char,int,unsigned,long,unsigned long> s2;

BOOST_MPL_ASSERT(( equal<s1,s2> ));

参见

查询算法, find, find_if
