Variadic 序列

描述

Variadic 序列 由 variadic 和 numbered 形式组成的序列类族中的某个成员。如果 seq 是某个 Variadic 序列 的通用名字，那么它的 variadic 形式允许我们指定由 n 个元素 t1,t2,... tn 所组成的序列，n 可以为从 0 到 可预处理器配置的界限 BOOST_MPL_LIMIT_seq_SIZE 间的任意数，使用如下表示方法：

seq<t1,t2,... tn>

作为比对，每个 numbered 序列形式接受确定数量的元素，它的名字为如下类模板：

seqn<t1,t2,... tn>

对于 numbered 形式，没有对 n 的上限进行预定义，除了编译器对模板参数的限制以外。

表达式要求

在下表和后面的说明中，seq 是一个占位符符号，表示实际的 Variadic 序列 名字。

表达式 类型 复杂度
seq<t1,t2,... tn> 前向序列 分期常量时间
seq<t1,t2,... tn>::type 前向序列 分期常量时间
seqn<t1,t2,... tn> 前向序列 分期常量时间
seqn<t1,t2,... tn>::type 前向序列 分期常量时间

表达式语义

typedef seq<t1,t2,... tn> s;
typedef seqn<t1,t2,... tn> s;
语义： s 是一个序列，元素为 t1,t2,... tn.
后置条件：

size<s>::value == n.

 
typedef seq<t1,t2,... tn>::type s;
typedef seqn<t1,t2,... tn>::type s;
语义： s 等同于 seqn<t1,t2,... tn >.
后置条件： size<s>::value == n.

Models

  • vector
  • list
  • map
