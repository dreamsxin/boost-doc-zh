advance

摘要

template<
      typename Iterator
    , typename N
    >
struct advance
{
    typedef unspecified type;
};

描述

将 Iterator 移动 N 个位置。对于 双向 和 随机访问 迭代器，距离可以为负。

头文件

#include <boost/mpl/advance.hpp>

参数

参数 要求 描述
Iterator 前向迭代器 移动的迭代器
N 整型常量 距离

Model Of

Tag 分派元函数

表达式语义

对于一个 前向迭代器 iter 和任意 整型常量 n:

typedef advance<iter,n>::type j;
返回类型: 前向迭代器
前置条件: 如果 Iterator 为 前向迭代器, n::value 必须为非负
语义:

等价于：

typedef iter i0; 
typedef next<i0>::type i1; 
...
typedef next<in-1>::type j;

如果 n::value > 0, 否则

typedef iter i0; 
typedef prior<i0>::type i1; 
...
typedef prior<in-1>::type j;
后置条件: j 是可提领的或 past-the-end; distance<iter,j>::value == n::value 如果 n::value > 0, 否则 distance<j,iter>::value == n::value

复杂度

分期常量时间如果 iter 符合 随机访问迭代器，否则为线性时间。

例子

typedef range_c<int,0,10> numbers;
typedef begin<numbers>::type first;
typedef end<numbers>::type last;

typedef advance<first,int_<10> >::type i1;
typedef advance<last,int_<-10> >::type i2;

BOOST_MPL_ASSERT(( boost::is_same<i1,last> ));
BOOST_MPL_ASSERT(( boost::is_same<i2,first> ));

参见

迭代器, Tag 分派元函数, distance, next
