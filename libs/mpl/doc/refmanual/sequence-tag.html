sequence_tag

摘要

template<
      typename X
    >
struct sequence_tag
{
    typedef unspecified type;
};

描述

sequence_tag 是一个 tag 元函数，用于所有 tag 分派 固有序列操作。

头文件

#include <boost/mpl/sequence_tag.hpp>

参数

参数 要求 描述
X 任意类型 要获取序列 tag 的类型

表达式语义

对于任意类型 x:

typedef sequence_tag<x>::type tag;
返回类型: 一个类型
语义: tag 是 x 的序列 tag，类型未指定

复杂度

分期常量时间。

参见

固有元函数, Tag 分派元函数
