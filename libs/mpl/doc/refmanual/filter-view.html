filter_view

摘要

template<
      typename Sequence
    , typename Pred
    >
struct filter_view
{
    // unspecified
    // ...
};

描述

一个视图，由 Sequence 中满足谓词 Pred 的元素组成。

头文件

#include <boost/mpl/filter_view.hpp>

Model of

  • 前向序列

参数

表达式 要求 描述
Sequence 前向序列 包装的序列
Pred 单参 Lambda 表达式 过滤用的谓词

表达式语义

以下只列出 前向序列 中没有定义或定义不同的表达式语义。

在下表中，v 是一个 filter_view 实例，s 是任意一个 前向序列，pred 为一个单参 Lambda 表达式。

表达式 语义 
filter_view<s,pred>
filter_view<s,pred>::type
 一个惰性的 前向序列，包含区间 [begin<s>::type, end<s>::type) 中满足谓词 pred 的所有元素。
size<v>::type v 的大小；size<v>::value == count_if<s,pred>::value；线性复杂度；请见 前向序列。

例子

在一个序列中查找最大的浮点类型。

typedef vector<int,float,long,float,char[50],long double,char> types;
typedef max_element<
      transform_view< filter_view< types,boost::is_float<_> >, size_of<_> >
    >::type iter;

BOOST_MPL_ASSERT(( is_same< deref<iter::base>::type, long double > ));
