contains

摘要

template<
      typename Sequence
    , typename T
    >
struct contains
{
    typedef unspecified type;
};

描述

返回一个真值 整型常量，如果 Sequence 中有一个或以上元素等同于 T.

Header

#include <boost/mpl/contains.hpp>

参数

参数 要求 描述
Sequence 前向序列 被检查的序列
T 任意类型 查找的类型

表达式语义

对于任意 前向序列 s 和任意类型 t:

typedef contains<s,t>::type r;
返回类型: 整型常量
语义:

等价于

typedef not_< is_same< 
      find<s,t>::type
    , end<s>::type
    > >::type r;

复杂度

线性。最多 size<s>::value 次比较。

例子

typedef vector<char,int,unsigned,long,unsigned long> types;
BOOST_MPL_ASSERT_NOT(( contains<types,bool> ));

参见

查询算法, find, find_if, count, lower_bound
