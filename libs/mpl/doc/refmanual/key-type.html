key_type

摘要

template<
      typename Sequence
    , typename X
    >
struct key_type
{
    typedef unspecified type;
};

描述

返回用于在 Sequence 中标识 X 的 键。

头文件

#include <boost/mpl/key_type.hpp>

Model of

Tag 分派元函数

参数

参数 要求 描述
Sequence 关联序列 被查询的序列
X 任意类型 用于取出 键 的类型

表达式语义

对于任意 关联序列 s, s 中的迭代器 pos1 和 pos2, 以及任意类型 x:

typedef key_type<s,x>::type k;
返回类型: 一个类型
前置条件: x 可以被放入 s.
语义: k 是用于在 s 中标识 x 的 键。
后置条件: 如果 key_type< s,deref<pos1>::type >::type 等同于 key_type< s,deref<pos2>::type >::type 则 pos1 等同于 pos2.

复杂度

分期常量时间。

例子

typedef key_type< map<>,pair<int,unsigned> >::type k1;
typedef key_type< set<>,pair<int,unsigned> >::type k2;

BOOST_MPL_ASSERT(( is_same< k1,int > ));
BOOST_MPL_ASSERT(( is_same< k2,pair<int,unsigned> > ));

参见

关联序列, value_type, has_key, set, map
