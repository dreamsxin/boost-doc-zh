iterator_range

摘要

template<
      typename First
    , typename Last
    >
struct iterator_range
{
    // unspecified
    // ...
};

描述

一个视图，由一对迭代器指定的序列子集中的元素组成。

头文件

#include <boost/mpl/fold.hpp>

Model of

  • 前向, 双向, 或 随机访问序列, 取决于底层迭代器的类别。

参数

参数 要求 描述
First, Last 前向迭代器 标识视图边界的迭代器

表达式语义

以下只列出 前向序列 中没有定义或定义不同的表达式语义。

在下表中，v 是一个 iterator_range 实例, first 和 last 是某个 前向序列 的迭代器, 且 [first, last) 表示一个有效区间。

表达式 语义 
iterator_range<first,last>
iterator_range<first,last>::type
 一个惰性序列，所有元素位于区间 [first, last).

例子

typedef range_c<int,0,100> r;
typedef advance_c< begin<r>::type,10 >::type first;
typedef advance_c< end<r>::type,-10 >::type last;

BOOST_MPL_ASSERT(( equal< 
      iterator_range<first,last>
    , range_c<int,10,90>
    > ));

参见
