negate

摘要

template<
      typename T
    >
struct negate
{
    typedef unspecified type;
};

描述

返回其参数的负值(加性逆元)。

头文件

#include <boost/mpl/negate.hpp>
#include <boost/mpl/arithmetic.hpp>

Model of

数值元函数

参数

参数 要求 描述
T 整型常量 操作的参数

[注: 这里所列出的要求是缺省实现所要求的。关于如何为一个不满足 整型常量 要求的用户定义数值类型提供相关实现，请见 数值元函数 概念 — 注完]

表达式语义

对于任意 整型常量 c:

typedef negate<c>::type r;
返回类型: 整型常量
语义:

等价于

typedef integral_c< c::value_type, ( -c::value ) > r;
 
typedef negate<c> r;
返回类型: 整型常量
语义:

等价于

struct r : negate<c>::type {};

复杂度

分期常量时间。

例子

typedef negate< int_<-10> >::type r;
BOOST_MPL_ASSERT_RELATION( r::value, ==, 10 );
BOOST_MPL_ASSERT(( is_same< r::value_type, int > ));

参见

算术操作, 数值元函数, numeric_cast, plus, minus, times
