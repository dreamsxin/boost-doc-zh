distance

摘要

template<
      typename First
    , typename Last
    >
struct distance
{
    typedef unspecified type;
};

描述

返回 First 和 Last 迭代器间的距离，即一个 整型常量 n 满足 advance<First,n>::type 等同于 Last.

头文件

#include <boost/mpl/distance.hpp>

参数

参数 要求 描述
First, Last 前向迭代器 计算距离的迭代器

Model Of

Tag 分派元函数

表达式语义

对于任意 前向迭代器 first 和 last:

typedef distance<first,last>::type n;
返回类型: 整型常量
前置条件: [first, last) 是有效区间
语义:

等价于

typedef iter_fold<
      iterator_range<first,last>
    , long_<0>
    , next<_1>
    >::type n;
后置条件: is_same< advance<first,n>::type, last >::value == true.

复杂度

分期常量时间如果 first 和 last 为 随机访问迭代器，否则为线性时间。

例子

typedef range_c<int,0,10>::type range;
typedef begin<range>::type first;
typedef end<range>::type last;

BOOST_MPL_ASSERT_RELATION( (distance<first,last>::value), ==, 10);

参见

迭代器, Tag 分派元函数, advance, next, prior
