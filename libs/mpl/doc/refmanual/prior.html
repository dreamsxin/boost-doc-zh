prior

摘要

template<
      typename Iterator
    >
struct prior
{
    typedef unspecified type;
};

描述

返回序列中的前一个迭代器。[注意: prior 有多个重载的含义，取决于其参数的类型。例如，如果 X 是一个 整型常量，prior<X> 返回相同类型的一个递减后的 整型常量。以下说明是针对迭代器的。其它语义的详细说明请见相关概念的文档 — 注完].

头文件

#include <boost/mpl/next_prior.hpp>

参数

参数 要求 描述
Iterator 双向迭代器 递减的迭代器

表达式语义

对于任意 双向迭代器 iter:

typedef prior<iter>::type j;
返回类型: 双向迭代器
前置条件: iter 是可递减的
语义:

j 是一个指向序列中前一个元素的迭代器。如果 iter 是一个用户定义的迭代器，库提供的缺省实现等同于

typedef iter::prior j;

复杂度

分期常量时间。

例子

typedef vector_c<int,1> v;
typedef begin<v>::type first;
typedef end<v>::type last;

BOOST_MPL_ASSERT(( is_same< prior<last>::type, first > ));

参见

迭代器, begin / end, next, deref
