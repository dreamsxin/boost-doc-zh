list_c

描述

list_c 是 整型序列包装器 的一个 list。它具有 list 的所有特性和要求，不同的只是指定原始序列内容的方法。

头文件

序列形式 头文件
Variadic #include <boost/mpl/list_c.hpp>
Numbered #include <boost/mpl/list/listn_c.hpp>

Model of

  • 整型序列包装器
  • Variadic 序列
  • 前向序列
  • 可扩展序列
  • 前端可扩展序列

表达式语义

以下只列出 list 中没有定义或定义不同的表达式语义。

表达式 语义 
list_c<T,c1,c2,... cn>
listn_c<T,c1,c2,... cn>
 一个 list，其元素为整型常量包装器 integral_c<T,c1>, integral_c<T,c2>, ... integral_c<T,cn>；请见 整型常量包装器 
list_c<T,c1,c2,... cn>::type
listn_c<T,c1,c2,... cn>::type
 等同于 listn< integral_c<T,c1>, integral_c<T,c2>, ... integral_c<T,cn> >；请见 整型常量包装器 
list_c<T,c1,c2,... cn>::value_type
listn_c<T,c1,c2,... cn>::value_type
等同于 T；请见 整型常量包装器

例子

typedef list_c<int,1,2,3,5,7,12,19,31> fibonacci;
typedef push_front<fibonacci,int_<1> >::type fibonacci2;

BOOST_MPL_ASSERT_RELATION( front<fibonacci2>::type::value, ==, 1 );

参见

序列，整型序列包装器，list, integral_c, vector_c, set_c, range_c
