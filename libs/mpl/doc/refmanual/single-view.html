single_view

摘要

template<
      typename T
    >
struct single_view
{
    // unspecified
    // ...
};

描述

一个视图，含任意类型 T 的单元素序列。

头文件

#include <boost/mpl/single_view.hpp>

Model of

  • 随机访问序列

参数

参数 要求 描述
T 任意类型 被包装在序列中的类型

表达式语义

以下只列出 随机访问序列 中没有定义或定义不同的表达式语义。

在下表中，v 是一个 single_view 实例, x 为任意类型。

表达式 语义 
single_view<x>
single_view<x>::type
 一个单元素 随机访问序列 v 满足 front<v>::type 等同于 x.
size<v>::type v 的大小；size<v>::value == 1; 请见 随机访问序列

例子

typedef single_view<int> view;
typedef begin<view>::type first;
typedef end<view>::type last;

BOOST_MPL_ASSERT(( is_same< deref<first>::type,int > ));
BOOST_MPL_ASSERT(( is_same< next<first>::type,last > ));
BOOST_MPL_ASSERT(( is_same< prior<last>::type,first > ));

BOOST_MPL_ASSERT_RELATION( size<view>::value, ==, 1 );
