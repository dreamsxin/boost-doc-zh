char_

摘要

template<
      char N
    >
struct char_
{
    // unspecified
    // ...
};

描述

用于 char 的 整型常量 包装器。

头文件

#include <boost/mpl/char.hpp>

Model of

整型常量

参数

参数 要求 描述
N 字符常量 被包装的值。

表达式语义

只定义在 整型常量 中没有定义或定义不同的表达式语义。

对于任意的字符常量 c:

表达式 语义
char_<c> 一个 整型常量 x 满足 x::value == c 且 x::value_type 等同于 char.

例子

typedef char_<'c'> c;

BOOST_MPL_ASSERT(( is_same< c::value_type, char > ));
BOOST_MPL_ASSERT(( is_same< c::type, c > ));
BOOST_MPL_ASSERT(( is_same< next< c >::type, char_<'d'> > ));
BOOST_MPL_ASSERT(( is_same< prior< c >::type, char_<'b'> > ));
BOOST_MPL_ASSERT_RELATION( (c::value), ==, 'c' );
assert( c() == 'c' );
