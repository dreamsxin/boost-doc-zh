前向序列

描述

前向序列 是一个MPL概念，表示一个编译期的元素序列。序列的元素是类型，并可通过 迭代器 来访问。begin 和 end 元函数提供了界定该序列范围的迭代器。序列保证它的元素以一个明确的但可能不指明的顺序排列。每个MPL序列都是 前向序列。

定义

  • 序列的大小就是它所包含的元素的数量。序列的大小是一个非负整数。
  • 如果序列的大小为零，则序列为空。

表达式要求

对于任意的 前向序列 s，以下表达式必须有效：

表达式 类型 复杂度
begin<s>::type 前向迭代器 分期常量时间
end<s>::type 前向迭代器 分期常量时间
size<s>::type 整型常量 不指定
empty<s>::type 布尔整型常量 常量时间
front<s>::type 任何类型 分期常量时间

表达式语义

表达式 语义
begin<s>::type 指向序列中第一个元素的迭代器；请见 begin.
end<s>::type 指向序列末尾后的迭代器；请见 end.
size<s>::type 序列的大小；请见 size.
empty<s>::type 一个布尔整型常量 c，满足 c::value == true 当且仅当序列为空，请见 empty.
front<s>::type 序列中的第一个元素；请见 front.

不变式

对于任意 前向序列 s，总有以下不变式：

  • [begin<s>::type, end<s>::type) 总是一个有效区间。
  • 一个对区间 [begin<s>::type, end<s>::type) 进行迭代的算法将遍历 s 中的每一个元素恰好一次。
  • begin<s>::type 等同于 end<s>::type 当且仅当 s 为空。
  • 对 s 的两次不同遍历将以相同序列访问其元素。

Models

  • vector
  • map
  • range_c
  • iterator_range
  • filter_view

参见

序列，双向序列，前向迭代器，begin / end, size, empty, front
