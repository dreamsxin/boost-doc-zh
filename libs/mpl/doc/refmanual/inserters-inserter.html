inserter

摘要

template<
      typename State
    , typename Operation
    >
struct inserter
{
    typedef State state;
    typedef Operation operation;
};

描述

插入器 概念的一个通用 model.

头文件

#include <boost/mpl/inserter.hpp>

Model of

插入器

参数

参数 要求 描述
State 任意类型 初始状态
Operation 二元 Lambda 表达式 输出操作

表达式语义

以下仅列出在 插入器 中没有定义或定义不同的表达式语义。

对于任意二元 Lambda 表达式 op 以及任意类型 state:

表达式 语义
inserter<op,state> 一个 插入器

复杂度

分期常量时间。

例子

template< typename N > struct is_odd : bool_< ( N::value % 2 ) > {};

typedef copy<
      range_c<int,0,10>
    , inserter< // a filtering 'push_back' inserter
          vector<>
        , if_< is_odd<_2>, push_back<_1,_2>, _1 >
        >
    >::type odds;
   
BOOST_MPL_ASSERT(( equal< odds, vector_c<int,1,3,5,7,9>, equal_to<_,_> > ));
