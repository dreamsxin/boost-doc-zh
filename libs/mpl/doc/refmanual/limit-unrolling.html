BOOST_MPL_LIMIT_UNROLLING

摘要

#if !defined(BOOST_MPL_LIMIT_UNROLLING)
#   define BOOST_MPL_LIMIT_UNROLLING \
        implementation-defined integral constant \
/**/
#endif

描述

BOOST_MPL_LIMIT_UNROLLING 是一个可覆盖的配置宏，用于调节本库的迭代算法的解开深度。在本库的实现中， BOOST_MPL_LIMIT_UNROLLING 的缺省值为 4. 要覆盖这个缺省的上限，请在包含任何本库的头文件之前将 BOOST_MPL_LIMIT_UNROLLING 定义为所需的值。[注: 覆盖只在本库被配置为不使用 预先预处理的头文件 时才有效。详情请见 BOOST_MPL_CFG_NO_PREPROCESSED_HEADERS — 注完]

例子

覆盖 BOOST_MPL_LIMIT_UNROLLING 的缺省值只对库的性能有影响，没有其它用户可感知的影响。

参见

配置, BOOST_MPL_CFG_NO_PREPROCESSED_HEADERS
