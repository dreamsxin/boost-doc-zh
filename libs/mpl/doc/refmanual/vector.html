vector

描述

vector 是一个 variadic, 随机访问, 可扩展 的类型序列，它支持在两端的常量时间的元素插入和删除操作，以及在中间的线性时间的元素插入和删除操作。在支持 typeof 扩展的编译器上，vector 非常简单，在多数情况下是最高效的序列。

头文件

序列形式 头文件
Variadic #include <boost/mpl/vector.hpp>
Numbered #include <boost/mpl/vector/vectorn.hpp>

Model of

  • Variadic 序列
  • 随机访问序列
  • 可扩展序列
  • 后端可扩展序列
  • 前端可扩展序列

表达式语义

在下表中，v 是 vector 的一个实例，pos 和 last 是 v 中的迭代器，r 是一个 前向序列，n 是一个 整型常量，而 x 和 t1,t2,... tn 则是任意类型。

表达式 语义 
vector<t1,t2,... tn>
vectorn<t1,t2,... tn>
 vector，元素为 t1,t2,... tn; 请见 Variadic 序列 
vector<t1,t2,... tn>::type
vectorn<t1,t2,... tn>::type
 等同于 vectorn<t1,t2,... tn>; 请见 Variadic 序列
begin<v>::type 指向 v 的开头的迭代器；请见 随机访问序列
end<v>::type 指向 v 的结尾的迭代器；请见 随机访问序列
size<v>::type v 的大小；请见 随机访问序列
empty<v>::type 布尔 整型常量 c 满足 c::value == true 当且仅当该序列为空；请见 随机访问序列
front<v>::type v 中的第一个元素；请见 随机访问序列
back<v>::type v 中的最后一个元素；请见 随机访问序列
at<v,n>::type 从 v 的开始计第 n 个元素；请见 随机访问序列
insert<v,pos,x>::type 一个新的 vector，具有以下元素：[begin<v>::type, pos), x, [pos, end<v>::type)；请见 可扩展序列
insert_range<v,pos,r>::type 一个新的 vector，具有以下元素：[begin<v>::type, pos), [begin<r>::type, end<r>::type) [pos, end<v>::type)；请见 可扩展序列
erase<v,pos>::type 一个新的 vector，具有以下元素：[begin<v>::type, pos), [next<pos>::type, end<v>::type)；请见 可扩展序列
erase<v,pos,last>::type 一个新的 vector，具有以下元素：[begin<v>::type, pos), [last, end<v>::type)；请见 可扩展序列
clear<v>::type 一个空的 vector；请见 可扩展序列
push_back<v,x>::type 一个新的 vector，具有以下元素：[begin<v>::type, end<v>::type), x；请见 后端可扩展序列
pop_back<v>::type 一个新的 vector，具有以下元素：[begin<v>::type, prior< end<v>::type >::type)；请见 后端可扩展序列
push_front<v,x>::type 一个新的 vector，具有以下元素：[begin<v>::type, end<v>::type), x；请见 前端可扩展序列
pop_front<v>::type 一个新的 vector，具有以下元素：[next< begin<v>::type >::type, end<v>::type)；请见 前端可扩展序列

例子

typedef vector<float,double,long double> floats;
typedef push_back<floats,int>::type types;

BOOST_MPL_ASSERT(( is_same< at_c<types,3>::type, int > ));
