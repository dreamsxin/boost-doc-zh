apply_wrap

摘要

template< 
      typename F
    >
struct apply_wrap0
{
    typedef unspecified type;
};

template< 
      typename F, typename A1
    >
struct apply_wrap1
{
    typedef unspecified type;
};

...

template< 
      typename F, typename A1,... typename An
    >
struct apply_wrapn
{
    typedef unspecified type;
};

描述

以参数 A1,... An 调用 元函数类 F.

本质上，apply_wrap 形式只是 F::apply<A1,... An>::type / F::apply::type 表达式的一个语法包装(因此而得名)而已。它们提供了更简明的表示法以及比底层结构更高的可移植性，而付出的代价则是一次额外的模板实例化。

头文件

#include <boost/mpl/apply_wrap.hpp>

参数

参数 要求 描述
F 元函数类 被调用的元函数类
A1,... An 任意类型 调用所使用的参数

表达式语义

对于任意 元函数类 f 和任意类型 a1,... an:

typedef apply_wrapn<f,a1,...an>::type t;
返回类型: 任意类型
语义: 如果 n > 0，则等价于 typedef f::apply<a1,... an>::type t;，否则等价于 typedef f::apply::type t; 或 typedef f::apply<>::type t;，取决于 f::apply 是一个类还是一个类模板。

例子

struct f0
{
    template< typename T = int > struct apply
    {
        typedef char type;
    };
};
    
struct g0
{
    struct apply { typedef char type; };
};

struct f2
{
    template< typename T1, typename T2 > struct apply
    {
        typedef T2 type;
    };
};


typedef apply_wrap0< f0 >::type r1;
typedef apply_wrap0< g0 >::type r2;
typedef apply_wrap2< f2,int,char >::type r3;

BOOST_MPL_ASSERT(( is_same<r1,char> ));
BOOST_MPL_ASSERT(( is_same<r2,char> ));
BOOST_MPL_ASSERT(( is_same<r3,char> ));
