元函数类

概要

元函数类 元函数的一种特定表示方式，以支持高阶元编程。具体地说，它是一个带有名为 apply 的公有访问的内嵌 元函数 的类。对应地，元函数类的调用被定义为对内嵌的 apply 元函数的调用。

表达式要求

在下表以及后续说明中，f 是一个 元函数。

表达式 类型 复杂度
f::apply::type 任意类型 未指定
f::apply<>::type 任意类型 未指定
f::apply<a1,...an>::type 任意类型 未指定

表达式语义

typedef f::apply::type x;
前置条件: f 是一个无参 元函数类; f::apply::type 是一个 类型名
语义: x 为元函数调用的结果 
typedef f::apply<>::type x;
前置条件: f 是一个无参 元函数类; f::apply<>::type 是一个 类型名
语义: x 为元函数调用的结果 
typedef f::apply<a1,...an>::type x;
前置条件: f 是一个 n-元 元函数类; apply 是一个 元函数
语义: x 为以实参 a1,... an 调用元函数所得的结果
