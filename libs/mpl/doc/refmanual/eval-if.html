eval_if

摘要

template< 
      typename C
    , typename F1
    , typename F2
    >
struct eval_if
{
    typedef unspecified type;
};

描述

根据 C 的值对两个 无参元函数 参数 F1 或 F2 中的某一个进行求值。

头文件

#include <boost/mpl/eval_if.hpp>

参数

参数 要求 描述
C 整型常量 求值的条件
F1, F2 无参 元函数 选择进行求值的元函数

表达式语义

对于任意 整型常量 c 和无参 元函数 f1, f2:

typedef eval_if<c,f1,f2>::type t;
返回类型: 任意类型
语义: 如果 c::value == true, t 等同于 f1::type; 否则 t 等同于 f2::type.

例子

typedef eval_if< true_, identity<char>, identity<long> >::type t1;
typedef eval_if< false_, identity<char>, identity<long> >::type t2;

BOOST_MPL_ASSERT(( is_same<t1,char> ));
BOOST_MPL_ASSERT(( is_same<t2,long> ));

参见

元函数, 整型常量, eval_if_c, if_
