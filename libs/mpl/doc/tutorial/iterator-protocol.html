迭代器协议

本库不再保证或依赖于迭代器类型中的 advance / distance / next / prior / type 成员；下表列出了新的要求：

以前 现在
i::type deref<i>::type
i::next next<i>::type
i::prior prior<i>::type
i::advance<n>::type advance<i,n>::type
i::distance<j>::type distance<i,j>::type
i::category i::category
