Boost.MultiArray: 其它说明

编译器的支持

Boost.MultiArray 在设计时考虑了编译器的支持。虽然接口被保持为最小化，但我们针对各编译器与C++标准的符合度方面作出了努力，包括本库编写时各个主流的编译器。具体讲，Boost.MultiArray 已在以下编译器上测试：
  1. Gnu C++ Compiler v2.95.2 (编译开关 -ftemplate-depth-50)
  2. Gnu C++ Compiler v3.0.2
  3. Kuck and Associates C++ Compiler v4.0f
  4. Microsoft Visual C++ v6.0 sp5
  5. Comeau C++ Compiler v4.2.45 beta 2 (libcomo beta 14)
  6. Intel C++ Compiler v5.0 (带 MS lib v6.0 sp5)
  7. Metrowerks CodeWarrior C++ Compiler v7.1
Boost.MultiArray 已知不能工作于 Borland C++ v5.5.1. 对该编译器的支持正在进行中。
