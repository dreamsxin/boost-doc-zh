Boost.MultiArray: 测试说明

以下是关于 Boost.Multi_Array (B.M) 中所包含的测试案例的说明。

术语

以下列出在下面的说明中将用到的术语：
  • B.M - Boost.MultiArray 的缩写
  • 主要组件primary components - 指 boost::multi_array, boost::const_multi_array_ref, 和 boost::multi_array_ref.
  • 数组类型array types - 指所有主要组件，以及每个主要组件的"数组"，array::const_subarray, array::subarray, array::const_array_view, 和 array::array_view

验收测试

以下测试检验了 Boost.Multi_Array 各个特性，以确认运行期的正确行为。
程序 说明
libs/multi_array/test/constructors.cpp 检验 B.M 各主要组件的所有构造函数。
libs/multi_array/test/access.cpp 测试所有 B.M 数组类型的 operator[] 和 operator().
libs/multi_array/test/compare.cpp 测试 B.M 各主要组件的所有比较操作符。
libs/multi_array/test/iterators.cpp 测试所有 B.M 数组类型的所有迭代器遍历和访问功能。
libs/multi_array/test/slice.cpp 测试所有 B.M 数组类型的各种子视图生成。
libs/multi_array/test/assign.cpp 测试各种 B.M 数组类型的 operator=().
libs/multi_array/test/assign_to_array.cpp 确认一种 multi_array 可以从另一种数组类型构造。
libs/multi_array/test/index_bases.cpp 测试 B.M 各主要组件的重索引功能。
libs/multi_array/test/storage_order.cpp 测试 B.M 各主要组件的各种 storage_order.
libs/multi_array/test/reshape.cpp 测试 B.M 各主要组件的整形功能。
libs/multi_array/test/range1.cpp 测试使用 array::index_range 指定索引范围的各种语法。
libs/multi_array/test/idxgen1.cpp 测试 array::index_gen 对象。
libs/multi_array/test/stl_interaction.cpp 测试数组类型与STL容器的互操作性。
libs/multi_array/test/resize.cpp 测试 multi_array 类的大小调整功能。
libs/multi_array/test/concept_checks.cpp 确认所有数组类型符合所定义的概念。
libs/multi_array/test/generative_tests.hpp 用于简化对所有数组类型的测试的一个工具。用于 slice.cpp, iterators.cpp, 和 access.cpp.
