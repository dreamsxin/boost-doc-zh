dfs_visitor<EventVisitorList>

该类是一个适配器，将一个 事件遍历器 列表(用 std::pair 构造而成的)转换为一个 DFS遍历器。

Example 示例

参见 事件遍历器 的例子。

Model of 以...为模型

DFS遍历器

Template Parameters 模板参数

ParameterDescriptionDefault
EventVisitorList 一个用 std::pair 创建的 事件遍历器 列表。 null_visitor

Where Defined 定义于

boost/graph/depth_first_search.hpp

Member Functions 成员函数

该类实现了 DFS遍历器 要求的所有成员函数。在每个函数中，正确的事件被分派至事件遍历器列表中的 事件遍历器。
