一个无向图的 带宽bandwidth 是指，两个邻接顶点间的最大距离，这里的距离是通过将所有顶点放在一个单元间隔的直线上来进行测量的。换句话说，如果一个无向图 G=(V,E) 的每个顶点被赋予从 0 至 |V| - 1 的一个整数，由 index[v] 给出，则 G 的带宽为

B(G) = max { |index[u] - index[v]|  | (u,v) in E }

一个图的 第i带宽 是指，第i个顶点与它任一邻接点间的最大距离。

Bi(G) = max { |index[i] - index[j]|  | (i,j) in E }

所以，带宽 B(G) 可以表示为 第i带宽 Bi(G) 的最大值。

B(G) = max { Bi(G)   | i=0...|V|-1 }
