tsp_tour_len_visitor<Graph, WeightMap, OutputIterator, Length>

这个类型是一个TSP巡游遍历器。它提供了用于巡游顶点的输出迭代器，并记录巡游的总长度。

Example 示例

double d(0.0);
std::vector<Vertex> c;
boost::metric_tsp_approx
  (g, get(edge_weight, g),
  make_tsp_tour_len_visitor(g, std::back_inserter(c), d, get(edge_weight, g)));

Model of 以...为模型

TSP巡游遍历器

Template Parameters 模板参数

ParameterDescriptionDefault
Graph 图类型 无
WeightMap 图中各边的权重。类型 WeightMap 必须是一个 可读属性映射Readable Property Map。图的边描述符类型必须可用作这个权重映射的键类型。 无
OutputIterator 一个输出迭代器 无
Length 巡游长度的一个适合的容器。必须实现加法操作符。 无
