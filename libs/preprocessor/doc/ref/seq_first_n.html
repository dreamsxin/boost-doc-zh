BOOST_PP_SEQ_FIRST_N 宏展开为一个 seq，由另一个 seq 的头 n 个元素组成。

用法

BOOST_PP_SEQ_FIRST_N(n, list)

参数

n
要取出的元素数量。
list
要从中取出元素的 seq.

备注

该宏从 seq 的头部取出 n 个元素，并将它们作为一个新的 seq 返回。

参见

  • BOOST_PP_SEQ_REST_N

要求

头文件:  <boost/preprocessor/seq/first_n.hpp>

代码示例 

#include <boost/preprocessor/seq/first_n.hpp>

#define SEQ (a)(b)(c)(d)(e)

BOOST_PP_SEQ_FIRST_N(2, SEQ) // 展开为 (a)(b)
