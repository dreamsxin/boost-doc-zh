BOOST_PP_XOR 宏展开为其操作数的逻辑 XOR.

用法

BOOST_PP_XOR(p, q)

参数

p
左操作数。有效值范围为 0 到 BOOST_PP_LIMIT_MAG.
q
右操作数。有效值范围为 0 到 BOOST_PP_LIMIT_MAG. 

备注

如果 p 或 q 中有且只有一个为非零，则该宏展开为 1. 否则展开为 0.
该宏在执行逻辑异或操作之前会对每个操作数执行布尔类型转换。如果不需要转换，请改用 BOOST_PP_BITXOR.

参见

  • BOOST_PP_BITXOR
  • BOOST_PP_LIMIT_MAG

要求

头文件:  <boost/preprocessor/logical/xor.hpp>

代码示例

#include <boost/preprocessor/logical/xor.hpp>

BOOST_PP_XOR(4, 3) // 展开为 0
BOOST_PP_XOR(5, 0) // 展开为 1
