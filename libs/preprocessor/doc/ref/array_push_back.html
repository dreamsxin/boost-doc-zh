BOOST_PP_ARRAY_PUSH_BACK 宏向一个数组的尾部增加一个元素。

用法

BOOST_PP_ARRAY_PUSH_BACK(array, elem)

参数

array
要增加元素的数组。
elem
增加的元素。

要求

头文件:  <boost/preprocessor/array/push_back.hpp>

代码示例

#include <boost/preprocessor/array/push_back.hpp>

#define ARRAY (3, (a, b, c))

BOOST_PP_ARRAY_PUSH_BACK(ARRAY, d) // 展开为 (4, (a, b, c, d))
