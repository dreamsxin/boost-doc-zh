BOOST_PP_ITERATION_LIMITS 宏是一个用户定义的 命名外部参数，用于 BOOST_PP_ITERATE. 它表示了一个 文件迭代 的下界和上界。

用法

#define BOOST_PP_ITERATION_LIMITS (start, finish)

参数

start
一个 文件迭代 的下界(含该值)。有效值为 0 到 BOOST_PP_LIMIT_ITERATION.
finish
一个 文件迭代 的上界(含该值)。有效值为 0 到 BOOST_PP_LIMIT_ITERATION. 

备注

注意在宏标识符后要有一个空白字符。
该宏是传递参数给 BOOST_PP_ITERATE 次选方法的一部分。另一部分为 BOOST_PP_FILENAME_x. start 和 finish 均为 已求值参数。这意味着它们可以包含简单的算术运算。
该宏会被自动去定义，以重新调用 BOOST_PP_ITERATE.
