BOOST_PP_ENUM_PARAMS 宏生成一个以逗号分隔的参数列表。

用法

BOOST_PP_ENUM_PARAMS(count, param)

参数

count
要生成的参数数量。有效值为 0 到 BOOST_PP_LIMIT_REPEAT.
param
参数的文本。BOOST_PP_ENUM_PARAMS 会将范围 0 到 count - 1 间的数字连接到它以生成参数。

备注

该宏展开为一个以逗号分隔的序列: 
param ## 0, param ## 1, ... param ## count - 1
要使用从其它使用了 BOOST_PP_REPEAT 的宏所传入的 z 参数，请见 BOOST_PP_ENUM_PARAMS_Z.
以前，该宏不可以在 BOOST_PP_REPEAT 中递归使用。现在这个限制已不存在，因为本库可以自动检测下一个可用的迭代深度。

参见

  • BOOST_PP_ENUM_PARAMS_Z
  • BOOST_PP_LIMIT_REPEAT

要求

头文件:  <boost/preprocessor/repetition/enum_params.hpp>

代码示例

#include <boost/preprocessor/repetition/enum_params.hpp>

BOOST_PP_ENUM_PARAMS(3, class T) // 展开为 class T0, class T1, class T2
