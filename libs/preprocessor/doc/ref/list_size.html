BOOST_PP_LIST_SIZE 宏展开为一个链表的大小。

用法

BOOST_PP_LIST_SIZE(list)

参数

list
计算其大小的链表。

备注

以前，该宏不可用在 BOOST_PP_WHILE 中。现在已没有该限制。不过在此情况下使用 BOOST_PP_LIST_SIZE_D 更为高效。

参见

  • BOOST_PP_LIST_SIZE_D

要求

头文件:  <boost/preprocessor/list/size.hpp>

代码示例

#include <boost/preprocessor/list/size.hpp>

#define LIST (0, (1, (2, (3, BOOST_PP_NIL))))

BOOST_PP_LIST_SIZE(LIST) // 展开为 4
