BOOST_PP_ARRAY_ELEM 宏从一个数组中提取一个元素。

用法

BOOST_PP_ARRAY_ELEM(i, array)

参数

i
要从数组中提取的元素索引，从零起计。
array
要从中提取元素的数组。该数组必须包含至少 i + 1 个元素。

要求

头文件:  <boost/preprocessor/array/elem.hpp>

代码示例

#include <boost/preprocessor/array/elem.hpp>

#define ARRAY (4, (a, b, c, d))

BOOST_PP_ARRAY_ELEM(0, ARRAY) // 展开为 a
BOOST_PP_ARRAY_ELEM(3, ARRAY) // 展开为 d
