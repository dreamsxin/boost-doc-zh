BOOST_PP_TUPLE_TO_LIST 宏将一个 tuple 转换为链表。

用法

BOOST_PP_TUPLE_TO_LIST(size, tuple)

参数

size
要转换的 tuple 的大小。有效的 tuple 大小范围为 0 到 BOOST_PP_LIMIT_TUPLE. 
tuple
要转换的 tuple.

备注

size 参数必须为 tuple 的真实大小。

参见

  • BOOST_PP_LIMIT_TUPLE

要求

头文件:  <boost/preprocessor/tuple/to_list.hpp>

代码示例

#include <boost/preprocessor/tuple/to_list.hpp>

BOOST_PP_TUPLE_TO_LIST(3, (x, y, z))
   // 展开为 (x, (y, (z, BOOST_PP_NIL)))
