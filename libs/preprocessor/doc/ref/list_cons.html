BOOST_PP_LIST_CONS 宏是一个链表构造器。

用法

BOOST_PP_LIST_CONS(head, tail)

参数

head
链表中的一个元素。
tail
要么是一个链表，要么是 BOOST_PP_LIST_NIL 或 BOOST_PP_NIL.

备注

该宏将一个新的头元素添加到已有的链表中，或从 BOOST_PP_LIST_NIL 创建一个链表。
该宏不再需要。例如...
BOOST_PP_LIST_CONS(a, BOOST_PP_LIST_CONS(b, BOOST_PP_LIST_NIL)))
...可以更清楚地写为:
(a, (b, BOOST_PP_NIL))
因此，该宏已被反对使用。

参见

  • BOOST_PP_LIST_NIL
  • BOOST_PP_NIL

要求

头文件:  <boost/preprocessor/list/adt.hpp>

代码示例

#include <boost/preprocessor/list/adt.hpp>

#define OLD \
   BOOST_PP_LIST_CONS( \
      a, \
      BOOST_PP_LIST_CONS( \
         b, \
         BOOST_PP_LIST_CONS( \
            c, \
            BOOST_PP_LIST_NIL \
         ) \
      ) \
   ) \
   /**/

#define NEW (a, (b, (c, BOOST_PP_NIL)))

BOOST_PP_LIST_FIRST(OLD) == BOOST_PP_LIST_FIRST(NEW)
   // 展开为 a == a

BOOST_PP_LIST_REST(OLD) == BOOST_PP_LIST_REST(NEW)
   // 展开为 (b, (c, BOOST_PP_NIL)) == (b, (c, BOOST_PP_NIL))
