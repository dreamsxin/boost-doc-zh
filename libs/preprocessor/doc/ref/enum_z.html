BOOST_PP_ENUM_z 宏表示一个重入到 BOOST_PP_ENUM 的迭代结构。

用法

BOOST_PP_ENUM_ ## z(count, macro, data)

参数

z
下一个可用的 BOOST_PP_REPEAT 维度。
count
重复调用 macro 的次数。有效值为 0 到 BOOST_PP_LIMIT_REPEAT.
macro
一个三元操作，形如 macro(z, n, data). 该宏被 BOOST_PP_ENUM 展开，参数分别为下一个可用的迭代深度，当前迭代次数，以及辅助的 data 参数。
data
传给 macro 的辅助数据。

备注

该宏展开为一个以逗号分隔的序列:
macro(z, 0, data), macro(z, 1, data), ... macro(z, count - 1, data)
在某些特定时候，需要使用 BOOST_PP_CAT 而不是预处理器的 token-pasting 操作符来执行串联。这发生在 z 值是一个调用其自身的宏时。这就需要延迟对它的展开。这种情况下的语法将变成: 
BOOST_PP_CAT(BOOST_PP_ENUM_, z)(count, macro, data)

参见

  • BOOST_PP_CAT
  • BOOST_PP_ENUM
  • BOOST_PP_LIMIT_REPEAT

要求

头文件:  <boost/preprocessor/repetition/enum.hpp>

代码示例

#include <boost/preprocessor/arithmetic/inc.hpp>
#include <boost/preprocessor/repetition/enum.hpp>

#define TEXT(z, n, text) text

#define TTP(z, n, _) \
   template< \
      BOOST_PP_ENUM_ ## z(BOOST_PP_INC(n), TEXT, class) \
   > \
   class T ## n \
   /**/

BOOST_PP_ENUM(3, TTP, nil)
   /*
      展开为...
      template<class> class T0,
      template<class, class> class T1,
      template<class, class, class> class T2
   */
