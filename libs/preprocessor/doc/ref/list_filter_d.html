BOOST_PP_LIST_FILTER_D 宏根据给定的标准过滤一个链表。它以最高的效率重入 BOOST_PP_WHILE.

用法

BOOST_PP_LIST_FILTER_D(d, pred, data, list)

参数

d
下一个可用的 BOOST_PP_WHILE 迭代。
pred
形如 pred(d, data, elem) 三元谓词。该谓词被 BOOST_PP_LIST_FILTER 对链表中的每个元素进行展开，以下一个可用的 BOOST_PP_WHILE 迭代、一个辅助数据 data 以及链表中的当前元素 elem 为参数。该宏必须返回一个范围 0 到 BOOST_PP_LIMIT_MAG 间的整数值。如果该谓词对于某个元素展开为非零，则该元素被包含在结果链表中。
data
传递给 pred 的辅助数据。
list
被过滤的链表。

备注

该宏对 list 中每个元素展开 pred. 它以令 pred 返回非零值的各个元素创建一个新的链表。 

参见

  • BOOST_PP_LIMIT_MAG
  • BOOST_PP_LIST_FILTER

要求

头文件:  <boost/preprocessor/list/filter.hpp>

代码示例

#include <boost/preprocessor/comparison/less_equal.hpp>
#include <boost/preprocessor/list/filter.hpp>
#include <boost/preprocessor/list/fold_right.hpp>

#define A (1, (2, (3, (4, BOOST_PP_NIL))))
#define B (A, (A, (A, (A, BOOST_PP_NIL))))

#define PRED(d, data, x) BOOST_PP_LESS_EQUAL(x, data)
#define OP(d, state, x) (BOOST_PP_LIST_FILTER_D(d, PRED, 2, x), state)

BOOST_PP_LIST_FOLD_RIGHT(OP, BOOST_PP_NIL, B)
/*
   展开为:
   ((1, (2, BOOST_PP_NIL)), 
   ((1, (2, BOOST_PP_NIL)), 
   ((1, (2, BOOST_PP_NIL)), 
   ((1, (2, BOOST_PP_NIL)), 
      BOOST_PP_NIL))))
*/
