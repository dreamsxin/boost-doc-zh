BOOST_PP_ARRAY_REMOVE_D 宏从一个数组中删除一个元素。它以最高的效率重入 BOOST_PP_WHILE.

用法

BOOST_PP_ARRAY_REMOVE_D(d, array, i)

参数

d
下一个可用的 BOOST_PP_WHILE 迭代。
array
要删除元素的数组。
i
被删除的元素在数组中的位置，从零起计。有效值为 0 到 BOOST_PP_ARRAY_SIZE(array) - 1.

参见

  • BOOST_PP_ARRAY_REMOVE

要求
