BOOST_PP_MUL_D 宏展开为其参数的积。它以最高效率重入 BOOST_PP_WHILE.

用法

BOOST_PP_MUL_D(d, x, y)

参数

d
下一个可用的 BOOST_PP_WHILE 迭代。
x
操作的被乘数。有效值为 0 到 BOOST_PP_LIMIT_MAG.
y
操作的乘数。有效值为 0 到 BOOST_PP_LIMIT_MAG.

备注

如果 x 和 y 的积大于 BOOST_PP_LIMIT_MAG, 则结果被调整到 BOOST_PP_LIMIT_MAG.
当 x 小于等于 y 时，这个宏最为高效。不过，得到的效率提高并不值得在调用之前对两个参数进行比较。换言之，x 很可能是两个参数中较大的一个。

参见

  • BOOST_PP_LIMIT_MAG
  • BOOST_PP_MUL

要求

头文件:  <boost/preprocessor/arithmetic/mul.hpp>

代码示例

#include <boost/preprocessor/arithmetic/dec.hpp>
#include <boost/preprocessor/arithmetic/mul.hpp>
#include <boost/preprocessor/control/while.hpp>
#include <boost/preprocessor/tuple/elem.hpp>

#define PRED(d, data) BOOST_PP_TUPLE_ELEM(3, 0, data)

#define OP(d, data) \
   ( \
      BOOST_PP_DEC( \
         BOOST_PP_TUPLE_ELEM(3, 0, data) \
      ), \
      BOOST_PP_TUPLE_ELEM(3, 1, data), \
      BOOST_PP_MUL_D( \
         d, \
         BOOST_PP_TUPLE_ELEM(3, 2, data), \
         BOOST_PP_TUPLE_ELEM(3, 1, data) \
      ) \
   ) \
   /**/

// 'x' 的 'n' 次方
#define EXP(x, n) BOOST_PP_TUPLE_ELEM(3, 2, BOOST_PP_WHILE(PRED, OP, (n, x, 1)))

EXP(4, 2) // 展开为 16
EXP(2, 3) // 展开为 8
