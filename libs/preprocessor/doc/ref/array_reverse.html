BOOST_PP_ARRAY_REVERSE 宏将一个数组中的元素反序。

用法

BOOST_PP_ARRAY_REVERSE(array)

参数

array
进行元素反序的数组。

要求

头文件:  <boost/preprocessor/array/reverse.hpp>

代码示例

#include <boost/preprocessor/array/reverse.hpp>

#define ARRAY (3, (a, b, c))

BOOST_PP_ARRAY_REVERSE(ARRAY) // 展开为 (3, (c, b, a))
