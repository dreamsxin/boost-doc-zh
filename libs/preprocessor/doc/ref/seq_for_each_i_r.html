BOOST_PP_SEQ_FOR_EACH_I_R 宏对一个 seq 中的每个元素重复调用一个指定的宏。它以最高的效率重入 BOOST_PP_FOR.

用法

BOOST_PP_SEQ_FOR_EACH_I_R(r, macro, data, seq)

参数

r
下一个可用的 BOOST_PP_FOR 迭代值。
macro
形如 macro(r, data, i, elem) 的宏。该宏由 BOOST_PP_SEQ_FOR_EACH_I 对 seq 中的每个元素展开。展开的参数为下一个可用的 BOOST_PP_FOR 迭代值、辅助数据 data、当前元素的索引，以及当前元素。
data
传给 macro 的辅助数据。
seq
macro 将对其每个元素进行调用的 seq.

备注

该宏是一个迭代结构。如果 seq 为 (a)(b)(c), 则它展开为序列:
macro(r, data, 0, a) macro(r, data, 1, b) macro(r, data, 2, c)
