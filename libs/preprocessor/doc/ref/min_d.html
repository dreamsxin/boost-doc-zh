BOOST_PP_MIN_D 宏展开为两个参数中的较小者。它以最高的效率重入 BOOST_PP_WHILE.

用法

BOOST_PP_MIN_D(d, x, y)

参数

d
下一个可用的 BOOST_PP_WHILE 迭代值。
x
第一个操作数。有效值为 0 到 BOOST_PP_LIMIT_MAG. 
y
第二个操作数。有效值为 0 到 BOOST_PP_LIMIT_MAG. 

备注

该宏返回其两个参数中的较小者，或者如果两个参数相等，则返回该值。

参见

  • BOOST_PP_LIMIT_MAG
  • BOOST_PP_MIN

要求

头文件:  <boost/preprocessor/selection/min.hpp>

代码示例

#include <boost/preprocessor/list/fold_left.hpp>
#include <boost/preprocessor/selection/min.hpp>

#define LIST (1, (3, (5, (2, (4, BOOST_PP_NIL)))))

#define OP(d, state, x) BOOST_PP_MIN_D(d, state, x)

#define LIST_MAX(list) BOOST_PP_LIST_FOLD_LEFT(OP, 20, LIST)

LIST_MIN(LIST) // 展开为 1
