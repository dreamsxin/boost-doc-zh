BOOST_PP_CAT 宏对其参数的展开结果进行连接。

用法

BOOST_PP_CAT(a, b)

参数

a
连接的左操作数。
b
连接的右操作数。

备注

预处理器的 token-pasting 操作符(##) 会阻止参数进行展开。本宏则允许其参数在连接之前先进行展开。
连接的结果不可以引起对一个使用了 BOOST_PP_CAT 的宏的调用。如果发生，则 BOOST_PP_CAT 不会进行第二次展开。

Requirements

Header:  <boost/preprocessor/cat.hpp>

Sample Code

#include <boost/preprocessor/cat.hpp>

BOOST_PP_CAT(x, BOOST_PP_CAT(y, z)) // 展开为 xyz
