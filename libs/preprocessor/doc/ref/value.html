BOOST_PP_VALUE 宏是一个用户自定义的 命名外部参数，用于 BOOST_PP_ASSIGN_SLOT.

用法

#define BOOST_PP_VALUE value

参数

value
一个整型常量表达式，被 BOOST_PP_ASSIGN_SLOT 求值。
