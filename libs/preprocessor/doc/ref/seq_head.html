BOOST_PP_SEQ_HEAD 宏展开为一个 seq 的第一个元素。

用法

BOOST_PP_SEQ_HEAD(seq)

参数

seq
从中取出第一个元素的 seq.

参见

  • BOOST_PP_SEQ_TAIL

要求

头文件:  <boost/preprocessor/seq/seq.hpp>

代码示例 

#include <boost/preprocessor/seq/seq.hpp>

#define SEQ (a)(b)(c)

BOOST_PP_SEQ_HEAD(SEQ) // 展开为 a
BOOST_PP_SEQ_TAIL(SEQ) // 展开为 (b)(c)
