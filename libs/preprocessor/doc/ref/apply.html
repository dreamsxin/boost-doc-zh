BOOST_PP_APPLY 宏抽象了有参数与无参数之间的差异。

用法

BOOST_PP_APPLY(x)

参数

x
被抽象的参数。这个参数必须是 BOOST_PP_NIL 或带一个元素的 tuple -- 如 (arg) 或 ((a, b)).

备注

如果 x 为 BOOST_PP_NIL, 该宏展开为空。如果 x 为一个单元素的 tuple, 则展开为 tuple 中的内容。

参见

  • BOOST_PP_NIL

要求

头文件:  <boost/preprocessor/facilities/apply.hpp>

代码示例

#include <boost/preprocessor/facilities/apply.hpp>
#include <boost/preprocessor/tuple/elem.hpp>

#define CV(i) \
   BOOST_PP_APPLY( \
      BOOST_PP_TUPLE_ELEM( \
         4, i, \
         (BOOST_PP_NIL, (const), (volatile), (const volatile)) \
      ) \
   ) \
   /**/

CV(0) // 展开为空
CV(1) // 展开为 const
