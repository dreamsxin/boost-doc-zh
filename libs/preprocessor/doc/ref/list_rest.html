BOOST_PP_LIST_REST 宏展开为一个链表的尾部。

用法

BOOST_PP_LIST_REST(list)

参数

list
某个非空链表。

要求

头文件:  <boost/preprocessor/list/adt.hpp>

代码示例

#include <boost/preprocessor/list/adt.hpp>

#define OLD \
   BOOST_PP_LIST_CONS( \
      a, \
      BOOST_PP_LIST_CONS( \
         b, \
         BOOST_PP_LIST_CONS( \
            c, \
            BOOST_PP_LIST_NIL \
         ) \
      ) \
   ) \
   /**/

#define NEW (a, (b, (c, BOOST_PP_NIL)))

BOOST_PP_LIST_FIRST(OLD) == BOOST_PP_LIST_FIRST(NEW)
   // 展开为 a == a

BOOST_PP_LIST_REST(OLD) == BOOST_PP_LIST_REST(NEW)
   // 展开为 (b, (c, BOOST_PP_NIL)) == (b, (c, BOOST_PP_NIL))
