BOOST_PP_SEQ_SIZE 宏展开为一个 seq 的大小。

用法

BOOST_PP_SEQ_SIZE(seq)

参数

seq
计算大小的 seq.

要求

头文件:  <boost/preprocessor/seq/size.hpp>

代码示例 

#include <boost/preprocessor/seq/size.hpp>

#define SEQ (a)(b)(c)

BOOST_PP_SEQ_SIZE(SEQ) // 展开为 3
