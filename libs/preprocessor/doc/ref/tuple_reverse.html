BOOST_PP_TUPLE_REVERSE 宏将一个指定大小的 tuple 反序。

用法

BOOST_PP_TUPLE_REVERSE(size, tuple)

参数

size
要反序的 tuple 的大小。有效的 tuple 大小范围为 0 到 BOOST_PP_LIMIT_TUPLE. 
tuple
要反序的 tuple.

备注

size 参数必须为 tuple 的真实大小。

参见

  • BOOST_PP_LIMIT_TUPLE

要求

头文件:  <boost/preprocessor/tuple/reverse.hpp>

代码示例

#include <boost/preprocessor/tuple/reverse.hpp>

BOOST_PP_TUPLE_REVERSE(3, (x, y, z)) // 展开为 (z, y, x)
