BOOST_PP_REPEAT_FROM_TO_z 宏表示一个重入到 BOOST_PP_REPEAT_FROM_TO 的重复结构。

用法

BOOST_PP_REPEAT_FROM_TO_ ## z(first, last, macro, data)

参数

z
下一个可用的 BOOST_PP_REPEAT 维度。
first
迭代的下界。有效值为 0 到 BOOST_PP_LIMIT_MAG.
last
迭代的上界。有效值为 0 到 BOOST_PP_LIMIT_MAG.
macro
形如 macro(z, n, data) 的三元操作。该宏被 BOOST_PP_REPEAT_FROM_TO 展开，参数分别为下一个可用的迭代深度、当前迭代值，以及辅助数据 data 参数。
data
传递给 macro 的辅助数据。

备注

该宏展开为序列:
macro(z, first, data) macro(z, first + 1, data) ... macro(z, last - 1, data)
迭代次数(即 last - first)必须不超过 BOOST_PP_LIMIT_REPEAT. 
在特定的时候，可能需要以 BOOST_PP_CAT 而不是预处理器的 token-pasting 操作符来执行串联。当 z 值为一个调用其自身的宏时就会如此。这需要延迟其展开。在此情形下该语法变为:
BOOST_PP_CAT(BOOST_PP_REPEAT_FROM_TO_, z)(count, macro, data).

参见

  • BOOST_PP_CAT
  • BOOST_PP_LIMIT_REPEAT
  • BOOST_PP_REPEAT

要求

头文件:  <boost/preprocessor/repetition/repeat_from_to.hpp>

代码示例

#include <boost/preprocessor/repetition/repeat_from_to.hpp>

#define TEXT(z, n, data) n data

#define MACRO(z, n, data) \
  ( \
     BOOST_PP_REPEAT_FROM_TO_ ## z( \
        1, 4, \
        TEXT, xyz \
     ) \
  ) \
  /**/

BOOST_PP_REPEAT(3, MACRO, _)
   /*
      展开为:
      (1 xyz 2 xyz 3 xyz)
      (1 xyz 2 xyz 3 xyz)
      (1 xyz 2 xyz 3 xyz)
   */
