BOOST_PP_EMPTY 宏是一个无参工具宏，展开为空。

用法

BOOST_PP_EMPTY()

备注

该宏用于避免低效的宏展开。其主要用途是作为 BOOST_PP_IF 或 BOOST_PP_IIF 的参数。

要求

头文件:  <boost/preprocessor/facilities/empty.hpp>

代码示例

#include <boost/preprocessor/control/if.hpp>
#include <boost/preprocessor/facilities/empty.hpp>

#define X() result
#define MACRO(c) BOOST_PP_IF(c, X, BOOST_PP_EMPTY)()

MACRO(0) // 展开为空
MACRO(1) // 展开为 result
