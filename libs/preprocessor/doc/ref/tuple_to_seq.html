BOOST_PP_TUPLE_TO_SEQ 宏将一个 tuple 转换为 seq.

用法

BOOST_PP_TUPLE_TO_SEQ(size, tuple)

参数

size
tuple 的元素数量。
tuple
要转换的 tuple.

要求

头文件:  <boost/preprocessor/tuple/to_seq.hpp>

代码示例 

#include <boost/preprocessor/tuple/to_seq.hpp>

BOOST_PP_TUPLE_TO_SEQ(3, (a, b, c)) // 展开为 (a)(b)(c)
