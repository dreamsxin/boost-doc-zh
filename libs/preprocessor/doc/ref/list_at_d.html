BOOST_PP_LIST_AT_D 宏从一个链表中取出一个元素。它以最高的效率重入 BOOST_PP_WHILE. 

用法

BOOST_PP_LIST_AT_D(d, list, index)

参数

d
下一个可用的 BOOST_PP_WHILE 迭代。
list
取出元素的链表。该链表必须至少有 index + 1 个元素。
index
要取出的元素在链表中的位置，从零起计。

参见

  • BOOST_PP_LIST_AT

要求

头文件:  <boost/preprocessor/list/at.hpp>

代码示例

#include <boost/preprocessor/control/while.hpp>
#include <boost/preprocessor/list/at.hpp>

#define LIST (7, (2, (0, (1, BOOST_PP_NIL))))

#define PRED(d, state) BOOST_PP_LIST_AT_D(d, state, 0)
#define OP(d, state) BOOST_PP_LIST_REST(state)

BOOST_PP_WHILE(PRED, OP, LIST) // 展开为 (0, (1, BOOST_PP_NIL))
