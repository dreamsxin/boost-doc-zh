BOOST_PP_ENUM_TRAILING_BINARY_PARAMS 宏生成一个以逗号分隔的二元参数列表，它以逗号开头。

用法

BOOST_PP_ENUM_TRAILING_BINARY_PARAMS(count, p1, p2)

参数

count
要生成的参数数量。有效值为 0 到 BOOST_PP_LIMIT_REPEAT.
p1
参数的第一部分的文本。BOOST_PP_ENUM_TRAILING_BINARY_PARAMS 会将范围 0 到 count - 1 间的数字连接到它以生成参数。
p2
参数的第二部分的文本。BOOST_PP_ENUM_TRAILING_BINARY_PARAMS 会将范围 0 到 count - 1 间的数字连接到它以生成参数。

备注

该宏展开为一个以逗号分隔的序列:
, p1 ## 0 p2 ## 0, p1 ## 1 p2 ## 1, ... p1 ## count - 1 p2 ## count - 1
要使用从其它使用了 BOOST_PP_REPEAT 的宏所传入的 z 参数，请见 BOOST_PP_ENUM_TRAILING_BINARY_PARAMS_Z.

参见

  • BOOST_PP_ENUM_TRAILING_BINARY_PARAMS_Z
  • BOOST_PP_LIMIT_REPEAT

要求

头文件:  <boost/preprocessor/repetition/enum_trailing_binary_params.hpp>

代码示例

#include <boost/preprocessor/facilities/intercept.hpp>
#include <boost/preprocessor/repetition/enum_trailing_binary_params.hpp>

template<class X BOOST_PP_ENUM_TRAILING_BINARY_PARAMS(4, class A, = X BOOST_PP_INTERCEPT)>
struct sample {
   // ...
};

/* 展开为...

template<class X, class A0 = X, class A1 = X, class A2 = X, class A3 = X>
struct sample {
   // ...
}
*/
