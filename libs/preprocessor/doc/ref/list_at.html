BOOST_PP_LIST_AT 宏从一个链表中取出一个元素。

用法

BOOST_PP_LIST_AT(list, index)

参数

list
取出元素的链表。该链表必须至少有 index + 1 个元素。
index
要取出的元素在链表中的位置，从零起计。

备注

以前，该宏不能在 BOOST_PP_WHILE 中使用。现在已没有这个限制。不过在此情况下，使用 BOOST_PP_LIST_AT_D 会更高效。

参见

  • BOOST_PP_LIST_AT_D

要求

头文件:  <boost/preprocessor/list/at.hpp>

代码示例

#include <boost/preprocessor/list/at.hpp>

#define LIST (a, (b, (c, BOOST_PP_NIL)))

BOOST_PP_LIST_AT(LIST, 0) // 展开为 a
BOOST_PP_LIST_AT(LIST, 2) // 展开为 c
