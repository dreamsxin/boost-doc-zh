BOOST_PP_INTERCEPT 宏用于截断后接的数字并展开为空。

用法

BOOST_PP_INTERCEPT

备注

该宏用于截断由库中其它结构所连接的数字。它主要用在其它文本之后，以吃掉连接符并展开为空。该宏只能截断范围在 0 到 BOOST_PP_LIMIT_MAG 间的整型常数。

参见

  • BOOST_PP_LIMIT_MAG

要求

头文件:  <boost/preprocessor/facilities/intercept.hpp>

代码示例

#include <boost/preprocessor/facilities/intercept.hpp>
#include <boost/preprocessor/repetition/enum_binary_params.hpp>

BOOST_PP_ENUM_BINARY_PARAMS(3, class T, = U)
// 展开为 class T0 = U0, class T1 = U1, class T2 = U2

BOOST_PP_ENUM_BINARY_PARAMS(3, class T, = int BOOST_PP_INTERCEPT)
// 展开为 class T0 = int, class T1 = int, class T2 = int
