BOOST_PP_IDENTITY 宏展开为其调用时的参数。

用法

BOOST_PP_IDENTITY(item)()

参数

item
展开的结果。

备注

这个宏被设计用于只有一个子句需要被调用的 BOOST_PP_IF 和 BOOST_PP_IIF.

参见

  • BOOST_PP_IF
  • BOOST_PP_IIF

要求

头文件:  <boost/preprocessor/facilities/identity.hpp>

代码示例

#include <boost/preprocessor/control/if.hpp>
#include <boost/preprocessor/facilities/empty.hpp>
#include <boost/preprocessor/facilities/identity.hpp>

#define MACRO(n) BOOST_PP_IF(n, BOOST_PP_IDENTITY(x), BOOST_PP_EMPTY)()

MACRO(0) // 展开为空
MACRO(1) // 展开为 x
