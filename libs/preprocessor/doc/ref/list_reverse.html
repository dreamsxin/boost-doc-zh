BOOST_PP_LIST_REVERSE 宏展开为某个链表的反序。

用法

BOOST_PP_LIST_REVERSE(list)

参数

list
被反序的链表。

备注

以前，该宏不可用在 BOOST_PP_WHILE 中。现在已没有该限制。不过在此情况下使用 BOOST_PP_LIST_REVERSE_D 更为高效。

参见

  • BOOST_PP_LIST_REVERSE_D

要求

头文件:  <boost/preprocessor/list/reverse.hpp>

代码示例

#include <boost/preprocessor/list/reverse.hpp>

#define LIST (0, (1, (2, (3, BOOST_PP_NIL))))

BOOST_PP_LIST_REVERSE(LIST) // 展开为 (3, (2, (1, (0, BOOST_PP_NIL))))
