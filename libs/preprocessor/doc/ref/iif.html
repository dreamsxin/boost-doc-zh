BOOST_PP_IIF 宏根据一个bit的条件进行选择。

用法

BOOST_PP_IIF(bit, t, f)

参数

bit
确定选择结果的条件。该值必须展开为 0 或 1.
t
bit 为 1 时的展开结果。
f
bit 为 0 时的展开结果。

备注

该宏不对其第一个参数执行布尔类型转换。如果需要转换，请改用 BOOST_PP_IF. 

参见

  • BOOST_PP_IF
  • BOOST_PP_LIMIT_MAG

要求

头文件:  <boost/preprocessor/control/iif.hpp>

代码示例

#include <boost/preprocessor/control/iif.hpp>
#include <boost/preprocessor/logical/or.hpp

#define OR_IF(p, q, t, f) BOOST_PP_IIF(BOOST_PP_OR(p, q), t, f)

OR_IF(1, 0, abc, xyz) // 展开为 abc
OR_IF(0, 0, abc, xyz) // 展开为 xyz
