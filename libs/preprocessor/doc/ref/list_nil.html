BOOST_PP_LIST_NIL 宏是一个空链表构造器。

用法

BOOST_PP_LIST_NIL

备注

该宏表示一个链表的结尾。
该宏不再需要。它只是展开为 BOOST_PP_NIL. 因此，该宏被反对继续使用。

参见

  • BOOST_PP_NIL

要求

头文件:  <boost/preprocessor/list/adt.hpp>

代码示例

#include <boost/preprocessor/list/adt.hpp>

#define OLD \
   BOOST_PP_LIST_CONS( \
      a, \
      BOOST_PP_LIST_CONS( \
         b, \
         BOOST_PP_LIST_CONS( \
            c, \
            BOOST_PP_LIST_NIL \
         ) \
      ) \
   ) \
   /**/

#define NEW (a, (b, (c, BOOST_PP_NIL)))

BOOST_PP_LIST_FIRST(OLD) == BOOST_PP_LIST_FIRST(NEW)
   // 展开为 a == a

BOOST_PP_LIST_REST(OLD) == BOOST_PP_LIST_RESTK(NEW)
   // 展开为 (b, (c, BOOST_PP_NIL)) == (b, (c, BOOST_PP_NIL))
