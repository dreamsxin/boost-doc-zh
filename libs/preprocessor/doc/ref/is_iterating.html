BOOST_PP_IS_ITERATING 宏在处理 文件迭代 时被定义。

用法

#if !defined(BOOST_PP_IS_ITERATING) // ...

备注

如果正在处理 文件迭代，该宏被定义为 1. 这意味也可以用:
#if !BOOST_PP_IS_ITERATING // ...
该宏的定义是为了防止文件被无限包含。
