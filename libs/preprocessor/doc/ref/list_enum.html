BOOST_PP_LIST_ENUM 宏将一个链表转换为一个逗号分隔的列表。

用法

BOOST_PP_LIST_ENUM(list)

参数

list
被转换的链表。

备注

例如，如果 list 为 (a, (b, (c, BOOST_PP_NIL))), 则该宏生成:
a, b, c
以前，该宏不可用在 BOOST_PP_FOR 中。现在已没有这个限制。不过，在此情况下，使用 BOOST_PP_LIST_ENUM_R 效率更高。

参见

  • BOOST_PP_LIST_ENUM_R

要求

头文件:  <boost/preprocessor/list/enum.hpp>

代码示例

#include <boost/preprocessor/list/enum.hpp>

#define LIST (w, (x, (y, (z, BOOST_PP_NIL))))

BOOST_PP_LIST_ENUM(LIST) // 展开为 w, x, y, z
