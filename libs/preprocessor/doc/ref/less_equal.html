BOOST_PP_LESS_EQUAL 宏比较两个值的小于等于关系。

用法

BOOST_PP_LESS_EQUAL(x, y)

参数

x
用于比较的左操作数。有效值为 0 到 BOOST_PP_LIMIT_MAG.
y
用于比较的右操作数。有效值为 0 到 BOOST_PP_LIMIT_MAG.

备注

如果 x 小于等于 y, 则该宏展开为 1. 否则展开为 0.
以前，该宏不可用在 BOOST_PP_WHILE 中。现在再也没有这个限制。不过在此情形下，使用 BOOST_PP_LESS_EQUAL_D 更为高效。

参见

  • BOOST_PP_LESS_EQUAL_D
  • BOOST_PP_LIMIT_MAG

要求

头文件:  <boost/preprocessor/comparison/less_equal.hpp>

代码示例

#include <boost/preprocessor/comparison/less_equal.hpp>

BOOST_PP_LESS_EQUAL(4, 3) // 展开为 0
BOOST_PP_LESS_EQUAL(5, 5) // 展开为 1
