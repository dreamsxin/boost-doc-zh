BOOST_PP_GREATER_D 宏比较两个值的大于关系。它以最高的效率重入 BOOST_PP_WHILE.

用法

BOOST_PP_GREATER_D(d, x, y)

参数

d
下一个可用的 BOOST_PP_WHILE 迭代。
x
用于比较的左操作数。有效值为 0 到 BOOST_PP_LIMIT_MAG. 
y
用于比较的右操作数。有效值为 0 到 BOOST_PP_LIMIT_MAG. 

备注

如果 x 大于 y, 则该宏展开为 1. 否则展开为 0. 

参见

  • BOOST_PP_GREATER
  • BOOST_PP_LIMIT_MAG

要求

头文件:  <boost/preprocessor/comparison/greater.hpp>

代码示例

#include <boost/preprocessor/comparison/greater.hpp>
#include <boost/preprocessor/list/filter.hpp>

#define LIST (1, (2, (3, (4, (5, BOOST_PP_NIL)))))

#define PRED(d, _, num) BOOST_PP_GREATER_D(d, num, 2)

BOOST_PP_LIST_FILTER(PRED, nil, LIST) // 展开为 (3, (4, (5, BOOST_PP_NIL)))
