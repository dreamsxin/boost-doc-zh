BOOST_PP_INCLUDE_SELF 宏间接包含一个文件。

用法

#include BOOST_PP_INCLUDE_SELF()

参数

filename
带引号或尖括号的文件号，被 BOOST_PP_INCLUDE_SELF 包含。

备注

在使用该宏之前必须定义 BOOST_PP_INDIRECT_SELF。
很多预处理器不允许一个文件直接包含其自身 -- 即使该文件已经防止了这种情形。该宏与 BOOST_PP_INDIRECT_SELF 一起可以让一个文件间接包含其自身。
当 BOOST_PP_INDIRECT_SELF 被包含时，BOOST_PP_INCLUDE_SELF 将宏 BOOST_PP_IS_SELFISH 定义为 1. 当从包含文件返回时，BOOST_PP_IS_SELFISH 被去定义。

参见

  • BOOST_PP_INDIRECT_SELF
  • BOOST_PP_IS_SELFISH

要求

头文件:  <boost/preprocessor/iteration/self.hpp>

代码示例

// file.h
#if !BOOST_PP_IS_SELFISH

   #ifndef FILE_H_
   #define FILE_H_

   #include <boost/preprocessor/iteration/self.hpp>

   #define NAME X
   struct NAME {
      // ...
      #define BOOST_PP_INDIRECT_SELF "file.h"
      #include BOOST_PP_INCLUDE_SELF()
   };

   #define NAME Y
   struct NAME {
      // ...
      #define BOOST_PP_INDIRECT_SELF "file.h"
      #include BOOST_PP_INCLUDE_SELF()
   };

   #define NAME Z
   struct NAME {
      // ...
      #define BOOST_PP_INDIRECT_SELF "file.h"
      #include BOOST_PP_INCLUDE_SELF()
   };

   #endif

#else

      inline bool validate(NAME* p) {
         return true;
      }

      template<class T> bool validate(T* p) {
         return dynamic_cast<NAME*>(p);
      }

   #undef NAME

#endif
