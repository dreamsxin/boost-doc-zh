BOOST_PP_MOD_D 宏展开为其第二和第三参数的模。它以最高效率重入 BOOST_PP_WHILE.

用法

BOOST_PP_MOD_D(d, x, y)

参数

d
下一个可用的 BOOST_PP_WHILE 迭代。
x
操作的被除数(分子)。有效值为 0 到 BOOST_PP_LIMIT_MAG.
y
操作的除数(分母)。有效值为 1 到 BOOST_PP_LIMIT_MAG.

备注

如果 y 为 0, 则结果未定义。

参见

  • BOOST_PP_LIMIT_MAG
  • BOOST_PP_MOD

要求

头文件:  <boost/preprocessor/arithmetic/div.hpp>

代码示例

#include <boost/preprocessor/arithmetic/mod.hpp>
#include <boost/preprocessor/list/filter.hpp>
#include <boost/preprocessor/logical/not.hpp>

#define LIST (1, (2, (3, (4, (5, BOOST_PP_NIL)))))

#define EVEN_P(d, _, num) BOOST_PP_NOT(BOOST_PP_MOD_D(d, num, 2))
#define EVEN(list) BOOST_PP_LIST_FILTER(EVEN_P, nil, list)

EVEN(LIST) // 展开为 (2, (4, BOOST_PP_NIL))

#define ODD_P(d, _, num) BOOST_PP_MOD_D(d, num, 2)
#define ODD(list) BOOST_PP_LIST_FILTER(ODD_P, nil, list)

ODD(LIST) // 展开为 (1, (3, (5, BOOST_PP_NIL)))
