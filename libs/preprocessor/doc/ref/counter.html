BOOST_PP_COUNTER 宏展开为当前计数器的值。

用法

BOOST_PP_COUNTER

备注

该宏展开为一个整数值。其初始值为0。在对 BOOST_PP_UPDATE_COUNTER 的两次使用之间，BOOST_PP_COUNTER 的值为常量。

参见

  • BOOST_PP_UPDATE_COUNTER

要求

头文件:  <boost/preprocessor/slot/counter.hpp>

代码示例

#include <boost/preprocessor/slot/counter.hpp>

BOOST_PP_COUNTER // 0

#include BOOST_PP_UPDATE_COUNTER()

BOOST_PP_COUNTER // 1

#include BOOST_PP_UPDATE_COUNTER()

BOOST_PP_COUNTER // 2

#include BOOST_PP_UPDATE_COUNTER()

BOOST_PP_COUNTER // 3
