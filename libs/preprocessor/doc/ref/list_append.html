BOOST_PP_LIST_APPEND 宏对两个链表进行链接。

用法

BOOST_PP_LIST_APPEND(a, b)

参数

a
第一个链表。
b
第二个链表。

备注

该宏链接两个链表。例如，如果 a 是 (1, (2, (3, BOOST_PP_NIL))) 而 b 是 (4, (5, BOOST_PP_NIL)), 则该宏展开为:
(1, (2, (3, (4, (5, BOOST_PP_NIL)))))
以前，该宏不能在 BOOST_PP_WHILE 中使用。现在已没有这个限制。不过在此情况下，使用 BOOST_PP_LIST_APPEND_D 会更高效。

参见

  • BOOST_PP_LIST_APPEND_D

要求

头文件:  <boost/preprocessor/list/append.hpp>

代码示例

#include <boost/preprocessor/list/append.hpp>

#define L1 (a, (b, (c, BOOST_PP_NIL)))
#define L2 (x, (y, (z, BOOST_PP_NIL)))

BOOST_PP_LIST_APPEND(L1, L2)
   // 展开为 (a, (b, (c, (x, (y, (z, BOOST_PP_NIL))))))
