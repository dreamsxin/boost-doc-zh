BOOST_PP_RPAREN_IF 宏依条件展开为一个右括号。

用法

BOOST_PP_RPAREN_IF(cond)

参数

cond
决定是否展开为一个右括号或展开为空的条件。有效值为 0 到 BOOST_PP_LIMIT_MAG.

备注

如果 cond 展开为 0, 则该宏展开为空。否则展开为一个右括号。
预处理器在宏调用中会将括号解释为分隔符。因此，需要对括号进行特殊的处理。

参见

  • BOOST_PP_LIMIT_MAG
  • BOOST_PP_LPAREN_IF

要求

头文件:  <boost/preprocessor/punctuation/paren_if.hpp>

代码示例

#include <boost/preprocessor/punctuation/paren_if.hpp>

#define MACRO(c, x) BOOST_PP_LPAREN_IF(c) x BOOST_PP_RPAREN_IF(c)

MACRO(0, text) // 展开为 text
MACRO(1, text) // 展开为 (text)
