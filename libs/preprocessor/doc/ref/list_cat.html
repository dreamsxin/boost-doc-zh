BOOST_PP_LIST_CAT 宏将一个链表中的所有元素串接起来。

用法

BOOST_PP_LIST_CAT(list)

参数

list
要将元素进行串接的链表。

备注

元素被从左到右串接起来，从索引 0 开始。
以前，该宏不能在 BOOST_PP_WHILE 中使用。现在已没有这个限制。不过在此情况下，使用 BOOST_PP_LIST_CAT_D 会更高效。

参见

  • BOOST_PP_LIST_CAT_D

要求

头文件:  <boost/preprocessor/list/cat.hpp>

代码示例

#include <boost/preprocessor/list/cat.hpp>

#define LIST (a, (b, (c, BOOST_PP_NIL)))

BOOST_PP_LIST_CAT(LIST) // 展开为 abc
