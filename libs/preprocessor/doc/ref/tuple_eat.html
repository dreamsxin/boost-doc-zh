BOOST_PP_TUPLE_EAT 宏展开为一个宏，该宏将吞掉一个指定大小的 tuple.

用法

BOOST_PP_TUPLE_EAT(size)

参数

size
被吞掉的 tuple 的大小。有效的 tuple 大小范围为 0 到 BOOST_PP_LIMIT_TUPLE. 

备注

size 参数必须为 tuple 的真实大小。

参见

  • BOOST_PP_LIMIT_TUPLE

要求

头文件:  <boost/preprocessor/tuple/eat.hpp>

代码示例

#include <boost/preprocessor/control/if.hpp>
#include <boost/preprocessor/tuple/eat.hpp>

#define OP(a, b) (a b)

#define MACRO(n) BOOST_PP_IF(n, OP, BOOST_PP_TUPLE_EAT(2))(1, 2)

MACRO(0) // 展开为空
MACRO(1) // 展开为 (1, 2)
