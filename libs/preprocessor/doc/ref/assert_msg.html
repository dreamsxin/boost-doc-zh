BOOST_PP_ASSERT_MSG 宏按条件插入调试用的文本。

用法

BOOST_PP_ASSERT_MSG(cond, msg)

参数

cond
决定是否引发断言的条件。有效值为 0 到 BOOST_PP_LIMIT_MAG.
msg
在 cond 求值为 0 时显示的信息。

备注

如果 cond 展开为 0, 该宏展开为 msg. 否则展开为空。

参见

  • BOOST_PP_ASSERT
  • BOOST_PP_LIMIT_MAG

要求

头文件:  <boost/preprocessor/debug/assert.hpp>

代码示例

#include <boost/preprocessor/comparison/equal.hpp>
#include <boost/preprocessor/debug/assert.hpp>

// 假设在编译的第一阶段计算行数

#line 9
BOOST_PP_ASSERT_MSG( \
   BOOST_PP_EQUAL(__LINE__, 9), \
   "incorrect line numbering detected" \
)
