BOOST_PP_INC 宏展开为其参数加一。

用法

BOOST_PP_INC(x)

参数

x
被递增的值。有效值为 0 到 BOOST_PP_LIMIT_MAG.

备注

如果 x 为 BOOST_PP_LIMIT_MAG, 则结果被调整为 BOOST_PP_LIMIT_MAG.

备注

  • BOOST_PP_DEC
  • BOOST_PP_LIMIT_MAG

要求

头文件:  <boost/preprocessor/arithmetic/inc.hpp>

代码示例

#include <boost/preprocessor/arithmetic/inc.hpp>

BOOST_PP_INC(BOOST_PP_INC(6)) // expands to 8
BOOST_PP_INC(4) // expands to 5
