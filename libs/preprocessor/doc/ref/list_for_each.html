BOOST_PP_LIST_FOR_EACH 宏对一个链表中的每个元素重复某个宏。

用法

BOOST_PP_LIST_FOR_EACH(macro, data, list)

参数

macro
形如 macro(r, data, elem) 的三元宏。该宏由 BOOST_PP_LIST_FOR_EACH 对 list 中的每个元素展开。展开的参数为下一个可用的 BOOST_PP_FOR 迭代值、辅助数据 data, 以及当前元素。
data
传给 macro 的辅助数据。
list
macro 将对其每个元素进行调用的链表。

备注

该宏是一个迭代结构。如果 list 为 (a, (b, (c, BOOST_PP_NIL))), 则它展开为序列:
macro(r, data, a) macro(r, data, b) macro(r, data, c)
以前，该宏不可用在 BOOST_PP_FOR 中。现在已没有该限制。不过在此情况下使用 BOOST_PP_LIST_FOR_EACH_R 更为高效。

参见

  • BOOST_PP_LIST_FOR_EACH_R

要求

头文件:  <boost/preprocessor/list/for_each.hpp>

代码示例

#include <boost/preprocessor/cat.hpp>
#include <boost/preprocessor/list/for_each.hpp>

#define LIST (w, (x, (y, (z, BOOST_PP_NIL))))

#define MACRO(r, data, elem) BOOST_PP_CAT(elem, data)

BOOST_PP_LIST_FOR_EACH(MACRO, _, LIST) // 展开为 w_ x_ y_ z_
