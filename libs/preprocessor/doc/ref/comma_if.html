BOOST_PP_COMMA_IF 宏按条件展开为一个逗号。

用法

BOOST_PP_COMMA_IF(cond)

参数

cond
决定将宏展开为逗号或空的条件。有效值为 0 到 BOOST_PP_LIMIT_MAG.

备注

如果 cond 展开为 0, 则该宏展开为空。否则展开为一个逗号。
预处理器在对宏的调用中将逗号解释为参数分隔符。因此，逗号需要特殊的处理。

参见

  • BOOST_PP_LIMIT_MAG

要求

头文件:  <boost/preprocessor/punctuation/comma_if.hpp>

代码示例

#include <boost/preprocessor/punctuation/comma_if.hpp>
#include <boost/preprocessor/repetition/repeat.hpp>

#define MACRO(z, n, text) BOOST_PP_COMMA_IF(n) text

BOOST_PP_REPEAT(3, MACRO, class) // 展开为 class, class, class
