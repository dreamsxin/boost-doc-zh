BOOST_PP_SEQ_TRANSFORM 宏根据指定的转化操作对 seq 中的每个元素进行转化。

用法

BOOST_PP_SEQ_TRANSFORM(op, data, seq)

参数

op
形如 op(d, data, elem) 的三元操作。该转化操作被 BOOST_PP_SEQ_TRANSFORM 对 seq 中的每个元素展开，参数为下一个 BOOST_PP_SEQ_FOLD_LEFT 折叠步骤、辅助数据 data, 以及 seq 中的当前元素。
data
传递给 pred 的辅助数据。
seq
被转化的 seq.

备注

该宏针对 seq 中的每个元素展开 op. 它以每次调用的结果来构建一个新的 seq。例如，如果 seq 为 (a)(b)(c), 则该宏展开为...
(op(d, data, a))(op(d, data, b))(op(d, data, c))
要达到最高效率，请使用 BOOST_PP_SEQ_TRANSFORM_S.

参见

  • BOOST_PP_SEQ_TRANSFORM_S

要求

头文件:  <boost/preprocessor/seq/transform.hpp>

代码示例 

#include <boost/preprocessor/arithmetic/dec.hpp>
#include <boost/preprocessor/seq/transform.hpp>

#define SEQ (1)(3)(2)(5)

#define OP(s, data, elem) BOOST_PP_DEC(elem)

BOOST_PP_SEQ_TRANSFORM(OP, 3, SEQ)
   // 展开为 (0)(2)(1)(4)
