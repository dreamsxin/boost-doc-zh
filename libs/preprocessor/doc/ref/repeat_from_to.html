BOOST_PP_REPEAT_FROM_TO 宏表示一个快速水平重复结构。

用法

BOOST_PP_REPEAT_FROM_TO(first, last, macro, data)

参数

first
迭代的下界。有效值为 0 到 BOOST_PP_LIMIT_MAG.
last
迭代的上界。有效值为 0 到 BOOST_PP_LIMIT_MAG.
macro
形如 macro(z, n, data) 的三元操作。该宏被 BOOST_PP_REPEAT_FROM_TO 展开，参数分别为下一个可用的迭代深度、当前迭代值，以及辅助数据 data 参数。
data
传递给 macro 的辅助数据。

备注

该宏展开为序列:
macro(z, first, data) macro(z, first + 1, data) ... macro(z, last - 1, data)
迭代次数(即 last - first)必须不超过 BOOST_PP_LIMIT_REPEAT.
传给 macro 的 z 值代表下一个可用的迭代维度。其它带 _Z 后缀的宏在内部使用了 BOOST_PP_REPEAT -- 例如 BOOST_PP_ENUM_PARAMS 和 BOOST_PP_ENUM_PARAMS_Z.  并不强制使用这些 _Z 版本，不过将 z 值(传给 macro 的值)传给这些宏可以使它们以最高的效率重入 BOOST_PP_REPEAT.
要直接使用这个 z 值，而不是只将它传给其它宏，请见 BOOST_PP_REPEAT_FROM_TO_z.
以前，该宏不可用在 BOOST_PP_REPEAT 中。现在已没有该限制，因为本库可以自动检测下一个可用的迭代深度。
该宏还使用了 BOOST_PP_WHILE. 因此，该宏还有两个变体用于重入 BOOST_PP_WHILE, 即 BOOST_PP_REPEAT_FROM_TO_D 和  BOOST_PP_REPEAT_FROM_TO_D_z.

要求

头文件:  <boost/preprocessor/repetition/repeat_from_to.hpp>

代码示例

#include <boost/preprocessor/cat.hpp>
#include <boost/preprocessor/repetition/repeat_from_to.hpp>

#define DECL(z, n, text) BOOST_PP_CAT(text, n) = n;

BOOST_PP_REPEAT_FROM_TO(5, 10, DECL, int x)
 /*
 展开为:
 int x5 = 5; int x6 = 6; int x7 = 7;
 int x8 = 8; int x9 = 9;
 */
