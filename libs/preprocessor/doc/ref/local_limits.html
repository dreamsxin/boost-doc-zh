BOOST_PP_LOCAL_LIMITS 宏是一个用户自定义的 命名外部参数，用于 BOOST_PP_LOCAL_ITERATE.

用法

#define BOOST_PP_LOCAL_LIMITS (start, finish)

参数

start
局部迭代 的下界(含该值)。有效值为 0 到 BOOST_PP_LIMIT_ITERATION.
finish
局部迭代 的上界(含该值)。有效值为 0 到 BOOST_PP_LIMIT_ITERATION.

备注

注意在宏标识符后要跟一个空白符。
该宏必须展开为一个 2-元素 tuple.  这个 tuple 的元素表示了 局部迭代 的下界和上界。start 和 finish 均为 已求值参数。这意味着它们可以包含简单的算术表达式(如 1 + 3), 等等..
该宏会被自动去定义，以便于重新调用 BOOST_PP_LOCAL_ITERATE.

参见

  • BOOST_PP_LIMIT_ITERATION
  • BOOST_PP_LOCAL_ITERATE

代码示例

#include <boost/preprocessor/iteration/local.hpp>

template<int> struct sample;

#define BOOST_PP_LOCAL_MACRO(n) \
   template<> struct sample<n> { \
      enum { value = n }; \
   }; \
   /**/

#define BOOST_PP_LOCAL_LIMITS (1, 5)

#include BOOST_PP_LOCAL_ITERATE()
/* 展开为...
template<> struct sample<1> { enum { value = 1 }; };
template<> struct sample<2> { enum { value = 2 }; };
template<> struct sample<3> { enum { value = 3 }; };
template<> struct sample<4> { enum { value = 4 }; };
template<> struct sample<5> { enum { value = 5 }; };
*/
