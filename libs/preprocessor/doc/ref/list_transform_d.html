BOOST_PP_LIST_TRANSFORM_D 宏根据指定的转化操作对链表中的每个元素进行转化。它是最高的效率重入 BOOST_PP_WHILE.

用法

BOOST_PP_LIST_TRANSFORM_D(d, op, data, list)

参数

d
下一个可用的 BOOST_PP_WHILE 迭代值。
op
形如 op(d, data, elem) 的三元操作。该转化操作被 BOOST_PP_LIST_TRANSFORM 对 list 中的每个元素展开，参数为下一个 BOOST_PP_WHILE 迭代值、辅助数据 data, 以及 list 中的当前元素。
data
传递给 pred 的辅助数据。
list
被转化的 list.

备注

该宏针对 list 中的每个元素展开 op. 它以每次调用的结果来构建一个新的链表。例如，如果 list 为 (a, (b, (c, BOOST_PP_NIL))), 则该宏展开为...
(op(d, data, a), (op(d, data, b), (op(d, data, c), BOOST_PP_NIL)))

参见

  • BOOST_PP_LIST_TRANSFORM

要求

头文件:  <boost/preprocessor/list/transform.hpp>

代码示例

#include <boost/preprocessor/arithmetic/inc.hpp>
#include <boost/preprocessor/list/fold_right.hpp>
#include <boost/preprocessor/list/transform.hpp>

#define A (1, (2, (3, (4, BOOST_PP_NIL))))
#define B (A, (A, (A, (A, BOOST_PP_NIL))))

#define OP_T(d, data, x) BOOST_PP_INC(x)
#define OP(d, state, x) (BOOST_PP_LIST_TRANSFORM_D(d, OP_T, 2, x), state)

BOOST_PP_LIST_FOLD_RIGHT(OP, BOOST_PP_NIL, B)
/*
   展开为:
   ((2, (3, (4, (5, BOOST_PP_NIL)))), ((2, (3, (4, (5, BOOST_PP_NIL)))),
   ((2, (3, (4, (5, BOOST_PP_NIL)))), ((2, (3, (4, (5, BOOST_PP_NIL)))),
   BOOST_PP_NIL))))
*/
