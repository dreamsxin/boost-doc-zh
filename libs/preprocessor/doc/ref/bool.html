BOOST_PP_BOOL 宏对其操作数执行布尔类型转换。

用法

BOOST_PP_BOOL(x)

参数

x
被转换的值。有效值为 0 到 BOOST_PP_LIMIT_MAG.

备注

如果 x 为 0, 则该宏展开为 0. 否则展开为 1.

参见

  • BOOST_PP_LIMIT_MAG

要求

头文件:  <boost/preprocessor/logical/bool.hpp>

代码示例

#include <boost/preprocessor/logical/bool.hpp>

BOOST_PP_BOOL(6) // 展开为 1
BOOST_PP_BOOL(0) // 展开为 0
