BOOST_PP_ENUM_TRAILING_PARAMS_Z 宏生成一个以逗号分隔的并以逗号开头的参数列表。它以最高的效率重入 BOOST_PP_REPEAT.

用法

BOOST_PP_ENUM_TRAILING_PARAMS_Z(z, count, param)

参数

z
下一个可用的 BOOST_PP_REPEAT 维度。
count
要生成的参数数量。有效值为 0 到 BOOST_PP_LIMIT_REPEAT.
param
参数的文本。BOOST_PP_ENUM_TRAILING_PARAMS_Z 会将范围 0 到 count - 1 间的数字连接到它以生成参数。

备注

该宏展开为一个以逗号分隔的序列:
, param ## 0, param ## 1, ... param ## count - 1

参见

  • BOOST_PP_ENUM_TRAILING_PARAMS
  • BOOST_PP_LIMIT_REPEAT

要求

头文件:  <boost/preprocessor/repetition/enum_trailing_params.hpp>

代码示例

#include <boost/preprocessor/repetition/enum_trailing_params.hpp>
#include <boost/preprocessor/repetition/repeat.hpp>

#define MACRO(z, n, _) \
   template< \
      class BOOST_PP_ENUM_TRAILING_PARAMS_Z(z, n, class T) \
   > class X ## n { \
      /* ... */ \
   }; \
   /**/

BOOST_PP_REPEAT(2, MACRO, nil)
/*
   展开为...
   template<class> class X0 { };
   template<class, class T0> class X1 { };
*/
