BOOST_PP_ENUM 宏生成一个以逗号分隔的列表。

用法

BOOST_PP_ENUM(count, macro, data)

参数

count
重复调用 macro 的次数。有效值为 0 到 BOOST_PP_LIMIT_REPEAT.
macro
一个三元操作，形如 macro(z, n, data). 该宏被 BOOST_PP_ENUM 展开，参数分别为下一个可用的迭代深度，当前迭代次数，以及辅助的 data 参数。 
data
传给 macro 的辅助数据。

备注

该宏展开为一个以逗号分隔的序列:
macro(z, 0, data), macro(z, 1, data), ... macro(z, count - 1, data)
其中传给 macro 的 z 值表示下一个可用的迭代维度。带有 _Z 后缀的其它宏则在内部使用了 BOOST_PP_REPEAT -- 例如 BOOST_PP_ENUM_PARAMS 和 BOOST_PP_ENUM_PARAMS_Z. 并不强制要求使用这些带 _Z 的版本，不过将 z 值(传给 macro 的值)传给这些宏可以让它们以最高的效率重入 BOOST_PP_REPEAT.
要直接使用这个 z 值，而不是简单地将它传给另一个宏，请见 BOOST_PP_ENUM_z.
以前，这个宏不能在 BOOST_PP_REPEAT 中递归使用。现在这个限制已不存在，因为本库可以自动检测下一个可用的迭代深度。

参见

  • BOOST_PP_ENUM_z
  • BOOST_PP_LIMIT_REPEAT

要求

头文件:  <boost/preprocessor/repetition/enum.hpp>

代码示例

#include <boost/preprocessor/repetition/enum.hpp>

#define TEXT(z, n, text) text

BOOST_PP_ENUM(4, TEXT, class) // 展开为 class, class, class, class
