BOOST_PP_BITAND 宏展开为其操作数的按位 AND.

用法

BOOST_PP_BITAND(x, y)

参数

x
左操作数。其值必须展开为 0 或 1.
y
右操作数。其值必须展开为 0 或 1.

备注

如果 x 和 y 均为 1, 则该宏展开为 1. 否则展开为 0.
该宏在执行按位与的操作之前不会对操作数执行布尔类型转换。如果需要转换，请使用 BOOST_PP_AND.

参见

  • BOOST_PP_AND

要求

头文件:  <boost/preprocessor/logical/bitand.hpp>

代码示例

#include <boost/preprocessor/logical/bitand.hpp>

BOOST_PP_BITAND(0, 0) // 展开为 0
BOOST_PP_BITAND(0, 1) // 展开为 0
BOOST_PP_BITAND(1, 0) // 展开为 0
BOOST_PP_BITAND(1, 1) // 展开为 1
