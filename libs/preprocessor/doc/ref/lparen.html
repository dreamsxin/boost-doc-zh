BOOST_PP_LPAREN 宏展开为一个左括号。

用法

BOOST_PP_LPAREN()

备注

预处理器在宏调用中会将括号解释为分隔符。因此，需要对括号进行特殊的处理。

参见

  • BOOST_PP_RPAREN

要求

头文件:  <boost/preprocessor/punctuation/paren.hpp>

代码示例

#include <boost/preprocessor/punctuation/paren.hpp>

#define X(x) x
#define MACRO(p, x) X p x )

MACRO(BOOST_PP_LPAREN(), abc) // 展开为 abc

#define Y(x)

MACRO((10) Y BOOST_PP_LPAREN(), result) // 展开为 10
