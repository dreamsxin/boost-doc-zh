BOOST_PP_RELATIVE_START 宏展开为 文件迭代 中相对于当前深度的某个迭代帧的下界。

用法

BOOST_PP_RELATIVE_START(i)

参数

i
要取出其下界的迭代帧的相对深度。有效值为 0 到 BOOST_PP_ITERATION_DEPTH() - 1.

备注

该宏仅在 文件迭代 正在处理中有效。
参数 i 被解释为从当前帧向外计的帧数。

要求

头文件:  <boost/preprocessor/iteration/iterate.hpp>
