BOOST_PP_SEQ_REVERSE 宏将一个 seq 反序。

用法

BOOST_PP_SEQ_REVERSE(seq)

参数

seq
被反序的 seq.

备注

要达到最高效率，请使用 BOOST_PP_SEQ_REVERSE_S.

参见

  • BOOST_PP_SEQ_REVERSE_S

要求

头文件:  <boost/preprocessor/seq/reverse.hpp>

代码示例 

#include <boost/preprocessor/seq/reverse.hpp>

#define SEQ (1)(2)(3)

BOOST_PP_SEQ_REVERSE(SEQ) // 展开为 (3)(2)(1)
