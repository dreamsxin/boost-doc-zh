BOOST_PP_IF 宏基于逻辑条件在两个值中进行选择。

用法

BOOST_PP_IF(cond, t, f)

参数

cond
确定选择结果的条件。有效值为 0 到 BOOST_PP_LIMIT_MAG.
t
cond 为非零时的展开结果。
f
cond 为 0 时的展开结果。

备注

该宏对其第一个参数执行布尔类型转换。如果不需要转换，请改用 BOOST_PP_IIF.

参见

  • BOOST_PP_IIF
  • BOOST_PP_LIMIT_MAG

要求

头文件:  <boost/preprocessor/control/if.hpp>

代码示例

#include <boost/preprocessor/control/if.hpp>

BOOST_PP_IF(10, a, b) // 展开为 a
BOOST_PP_IF(0, a, b) // 展开为 b
