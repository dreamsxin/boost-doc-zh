BOOST_PP_NIL 标识符被本库保留用于表示一个非宏的实体。

用法

BOOST_PP_NIL

备注

该标识符是未定义的。其目的是不能用作一个宏。它也被用于表示一个链表的结束。

代码示例

#include <boost/preprocessor/list/adt.hpp>

#define LIST (x, (y, (z, BOOST_PP_NIL)))

BOOST_PP_LIST_FIRST(x) // 展开为 x
BOOST_PP_LIST_REST(x) // 展开为 (y, (z, BOOST_PP_NIL))
