BOOST_PP_ARRAY_INSERT_D 宏向一个数组插入一个元素。它以最高的效率重入 BOOST_PP_WHILE.

用法

BOOST_PP_ARRAY_INSERT_D(d, array, i, elem)

参数

d
下一个可用的 BOOST_PP_WHILE 迭代。
array
要插入元素的数组。
i
元素插入到数组中的位置，从零起计。有效值为 0 到 BOOST_PP_ARRAY_SIZE(array).
elem
要插入的元素。

备注

该宏将元素 elem 插入到索引为 i 的元素之前。
如果该操作试图创建一个大小超过 BOOST_PP_LIMIT_TUPLE 的数组，结果将是未定义的。
