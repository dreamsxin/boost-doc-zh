BOOST_PP_COMPL 宏对其操作数执行按位取反(按位 NOT 或其补足)。

用法

BOOST_PP_COMPL(x)

参数

x
被取反的值。该值必须展开为 0 或 1.

备注

如果 x 为 0, 该宏展开为 1. 如果 x 为 1, 则展开为 0.
该宏在执行操作之前不会对其操作数执行布尔类型转换。如果需要转换，请使用 BOOST_PP_NOT.

参见

  • BOOST_PP_NOT

要求

头文件:  <boost/preprocessor/logical/compl.hpp>

代码示例

#include <boost/preprocessor/logical/compl.hpp>

BOOST_PP_COMPL(1) // 展开为 0
BOOST_PP_COMPL(0) // 展开为 1
