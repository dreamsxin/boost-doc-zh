BOOST_PP_SEQ_TO_ARRAY 宏将一个 seq 转换为数组。

用法

BOOST_PP_SEQ_TO_ARRAY(seq)

参数

seq
被转换的 seq.

要求

头文件:  <boost/preprocessor/seq/to_array.hpp>

代码示例 

#include <boost/preprocessor/seq/to_array.hpp>

#define SEQ (a)(b)(c)

BOOST_PP_SEQ_TO_ARRAY(SEQ) // 展开为 (3, (a, b, c))
