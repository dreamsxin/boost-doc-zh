BOOST_PP_ARRAY_INSERT 宏向一个数组中插入一个元素。

用法

BOOST_PP_ARRAY_INSERT(array, i, elem)

参数

array
要插入元素的数组。
i
元素插入到数组中的位置，从零起计。有效值为 0 到 BOOST_PP_ARRAY_SIZE(array).
elem
插入的元素。

备注

该宏将元素 elem 插入到索引 i 的元素之前。
如果该操作试图创建一个大小超过 BOOST_PP_LIMIT_TUPLE 的数组，结果将是未定义的。
该宏在内部使用了 BOOST_PP_WHILE. 因此，要使用从另一个使用了 BOOST_PP_WHILE 的宏传入的 d 参数，请见 BOOST_PP_ARRAY_INSERT_D.

参见

  • BOOST_PP_ARRAY_INSERT_D

要求

头文件:  <boost/preprocessor/array/insert.hpp>

代码示例

#include <boost/preprocessor/array/insert.hpp>

#define ARRAY (3, (a, b, d))

BOOST_PP_ARRAY_INSERT(ARRAY, 2, c) // 展开为 (4, (a, b, c, d))
