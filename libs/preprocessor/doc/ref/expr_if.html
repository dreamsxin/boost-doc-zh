BOOST_PP_EXPR_IF 宏展开为它的第二个参数，如果第一个参数非零，否则展开为空。

用法

BOOST_PP_EXPR_IF(cond, expr)

参数

cond
确定结果为 expr 或为空的条件。有效值为 0 到 BOOST_PP_LIMIT_MAG.
expr
当 cond 非零时的表达式结果。

备注

该宏对其第一个参数执行布尔类型转换。如果不需要转换，请改用 BOOST_PP_EXPR_IIF.

参见

  • BOOST_PP_EXPR_IIF
  • BOOST_PP_LIMIT_MAG

要求

头文件:  <boost/preprocessor/control/expr_if.hpp>

代码示例

#include <boost/preprocessor/control/expr_if.hpp>
#include <boost/preprocessor/tuple/elem.hpp>

#define CV(n) \
   BOOST_PP_EXPR_IF( \
      n, \
      BOOST_PP_TUPLE_ELEM( \
         4, n, \
         (..., const, volatile, const volatile) \
      ) \
   ) \
   /**/

CV(0) // 展开为空
CV(1) // 展开为 const
