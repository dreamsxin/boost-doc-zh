BOOST_PP_ARRAY_SIZE 宏展开为传入的数组的大小。

用法

BOOST_PP_ARRAY_SIZE(array)

参数

array
要读取大小的数组。

要求

头文件:  <boost/preprocessor/array/size.hpp>

代码示例

#include <boost/preprocessor/array/size.hpp>

#define ARRAY (3, (x, y, z))

BOOST_PP_ARRAY_SIZE(ARRAY) // 展开为 3
