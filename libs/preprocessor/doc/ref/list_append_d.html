BOOST_PP_LIST_APPEND_D 宏对两个链表进行链接。它以最高的效率重入 BOOST_PP_WHILE.

用法

BOOST_PP_LIST_APPEND_D(d, a, b)

参数

d
下一个可用的 BOOST_PP_WHILE 迭代。
a
第一个链表。
b
第二个链表。

备注

该宏链接两个链表。例如，如果 a 是 (1, (2, (3, BOOST_PP_NIL))) 而 b 是 (4, (5, BOOST_PP_NIL)), 则该宏展开为: 
(1, (2, (3, (4, (5, BOOST_PP_NIL)))))

参见

  • BOOST_PP_LIST_APPEND

要求

头文件:  <boost/preprocessor/list/append.hpp>

代码示例

#include <boost/preprocessor/arithmetic/dec.hpp>
#include <boost/preprocessor/control/while.hpp>
#include <boost/preprocessor/list/append.hpp>
#include <boost/preprocessor/tuple/elem.hpp>

#define LIST (1, (2, (3, BOOST_PP_NIL)))

#define PRED(d, state) BOOST_PP_TUPLE_ELEM(3, 1, state)

#define OP(d, state) \
   ( \
      BOOST_PP_LIST_APPEND_D( \
         d, BOOST_PP_TUPLE_ELEM(3, 0, state), \
         BOOST_PP_TUPLE_ELEM(3, 2, state) \
      ), \
      BOOST_PP_DEC( \
         BOOST_PP_TUPLE_ELEM(3, 1, state) \
      ), \
      BOOST_PP_TUPLE_ELEM(3, 2, state) \
   ) \
   /**/

#define LIST_MULTIPLY(c, list) \
   BOOST_PP_TUPLE_ELEM( \
      3, 0, \
      BOOST_PP_WHILE( \
         PRED, OP, \
         (BOOST_PP_NIL, c, list) \
      ) \
   ) \
   /**/

LIST_MULTIPLY(3, LIST)
   // 展开为 (1, (2, (3, (1, (2, (3, (1, (2, (3, BOOST_PP_NIL)))))))))
