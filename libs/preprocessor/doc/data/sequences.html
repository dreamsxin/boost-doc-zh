序列

序列 (简称为 seq)是一组相邻的带括号的元素。例如,
(a)(b)(c)
...就是一个由三个元素组成的 seq -- a, b, 和 c.
序列 是结合了 链表 和 tuples 的特性的一种数据结构，但 seq 不能为空。因此，一个 "空的" seq 被认为是一种特殊情况，必须由C++单独处理。 
#define SEQ (x)(y)(z)
#define REVERSE(s, state, elem) (elem) state
   // 增加到头部

BOOST_PP_SEQ_FOLD_LEFT(REVERSE, BOOST_PP_EMPTY, SEQ)()
   //                           #1                  #2
   // 1) "空" seq 的占位符
   // 2) 删除占位符

#define SEQ_B (1)(2)(3)
#define INC(s, state, elem) state (BOOST_PP_INC(elem))
   // 增加到尾部

BOOST_PP_SEQ_FOLD_RIGHT(INC, BOOST_PP_SEQ_NIL, SEQ)
   //                        ^
   // 特殊的占位符，通过增加到尾部将其"吃掉"
序列 是非常高效的。元素的访问速度接近于随机访问 -- 即使 seqs 中有多达 256 个元素。这是因为元素访问(及其它操作)是以迭代而不是递归的方式来实现的。因此，元素可以被高效访问，即使是在只有很小的最大展开深度的预处理器上。
一个 seq 的元素可以用 BOOST_PP_SEQ_ELEM 来获取。
