读写 Property Map

读写 Property Map 可以通过 get() 函数读取属性值，也可以通过 put() 函数写入属性值。

强化自

可读 Property Map 和 可写 Property Map

符号

PMap 一个类型，符合读写 Property Map.

关联类型

Property Map Category boost::property_traits<PMap>::category 属性的分类，必须是可转换为 read_write_property_map_tag 的类型

概念检查类

  template <class PMap, class Key>
  struct ReadWritePropertyMapConcept
  {
    typedef typename property_traits<PMap>::category Category;
    typedef boost::read_write_property_map_tag ReadWriteTag;
    void constraints() {
      function_requires< ReadablePropertyMapConcept<PMap, Key> >();
      function_requires< WritablePropertyMapConcept<PMap, Key> >();
      function_requires< ConvertibleConcept<Category, ReadWriteTag> >();
    }
  };
