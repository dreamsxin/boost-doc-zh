Description 说明

Non-predicate versions: 不带谓词的版本：

adjacent_find finds the first adjacent elements [x,y] in rng where x == y
adjacent_find 在 rng 中查找第一对满足 x == y 相邻元素 [x,y]

Predicate versions: 带谓词的版本：

adjacent_find finds the first adjacent elements [x,y] in rng where pred(x,y) is true. adjacent_find 在 rng 中查找第一对令 pred(x,y) 为 true 相邻元素 [x,y]

Definition 定义

Defined in the header file boost/range/algorithm/adjacent_find.hpp
定义于头文件 boost/range/algorithm/adjacent_find.hpp 中

Requirements 要求

For the non-predicate versions of adjacent_find: 不带谓词的版本：

  • ForwardRange is a model of the Forward Range Concept.
    ForwardRange 符合 Forward Range 概念。
  • ForwardRange's value type is a model of the EqualityComparableConcept.
    ForwardRange 的值类型符合 EqualityComparableConcept.

For the predicate versions of adjacent_find: 带谓词的版本：

  • ForwardRange is a model of the Forward Range Concept.
    ForwardRange 符合 Forward Range 概念。
  • BinaryPredicate is a model of the BinaryPredicateConcept.
    BinaryPredicate 符合 BinaryPredicateConcept.
  • ForwardRange's value type is convertible to BinaryPredicate's first argument type and to BinaryPredicate's second argument type.
    ForwardRange 的值类型可转换为 BinaryPredicate 的第一参数类型和第二参数类型。
Complexity 复杂度

Linear. If empty(rng) then no comparisons are performed; otherwise, at most distance(rng) - 1 comparisons.
线性。如果 empty(rng) 则不执行比较；否则最多 distance(rng) - 1 次比较。
