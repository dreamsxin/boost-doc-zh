单遍迭代器概念

一个类或内建类型 X 符合 Single Pass Iterator单遍迭代器 概念，如果以下表达式有效且遵从规定的语义。

单遍迭代器要求 (除了可递增迭代器和可等价比较以外)
表达式 返回类型 断言/前提条件/后续条件
++r X& 前提：r 是可提领的；
后续：r 是可提领的或 r 指向末尾
a == b 可转换为 bool == 为该领域的一个等价关系
a != b 可转换为 bool !(a == b)
iterator_traversal<X>::type 可转换为 single_pass_traversal_tag
