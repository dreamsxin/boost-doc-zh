双向遍历概念

一个类或内建类型 X 符合 Bidirectional Traversal Iterator双向遍历迭代器 概念，如果 X 符合前向遍历迭代器的要求，且以下表达式有效并遵从规定的语义。

双向遍历迭代器要求 (除了前向遍历迭代器以外)
表达式 返回类型 断言/前提/后续条件
--r X&

前提：存在某个 s 使得 r == ++s. 

后续：s 为可提领的

++(--r) == r. --r == --s 意味着 r == s. &r == &--r.
r-- 可转换为 const X&   
{
 X tmp = r;
 --r;
 return tmp;
}
iterator_traversal<X>::type 可转换为 bidirectional_traversal_tag
