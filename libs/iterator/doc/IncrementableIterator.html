可递增迭代器概念

一个类或内建类型 X 符合 Incrementable Iterator可递增迭代嚣 概念，如果 X 是可赋值的和可复制构造的，且以下表达式有效并遵从规定的语义。

可递增迭代器要求 (除了可赋值和可复制构造以外)
表达式 返回类型 断言
++r X& &r == &++r
r++ X 
{
   X tmp = r;
   ++r;
   return tmp;
}
iterator_traversal<X>::type 可转换为 incrementable_traversal_tag
