随机数生成器库 --- 杂项：修饰子

  • 类模板 random_number_generator

简介

这 些修饰子的类模板可以把随机数生成器和分布函数适配为C++标准库中的概念，特别是 RandomNumberGenerator 和 InputIterator 概念。后一个概念的适配是很有用的，这是因为，与分布函数相比，基本的随机数生成器本身并不实现 InputIterator 的需求。

头文件 <boost/random.hpp> 中的杂项修饰子概要

namespace boost {
  template<class UniformRandomNumberGenerator, class IntType = long>
  class random_number_generator;
} // namespace boost

类模板 random_number_generator

概要

template<class UniformRandomNumberGenerator, class IntType = long>
class random_number_generator
{
public:
  typedef UniformRandomNumberGenerator base_type;
  typedef IntType argument_type;
  typedef IntType result_type;
  random_number_generator(base_type & rng);
  result_type operator()(argument_type n);
};

说明

类模板 random_number_generator 的实例符合 RandomNumberGenerator 概念(std:25.2.11 [lib.alg.random.shuffle])。每调用一次，它返回区间 [0..n) 中的一个均匀分布的整数。

模板参数 IntType 应指定某个类似于整数的值类型。

注：我认为，C++标准将名字 RandomNumberGenerator 用于别的东西是不幸的。

成员函数

random_number_generator(base_type & rng)

作用：以给定的均匀随机数生成器作为底层的随机数源，构造一个 random_number_generator 仿函数。

result_type operator()(argument_type n)

返回：uniform_int<base_type>(rng, 0, n-1)() 的值。
