将数字类型和枚举类型转换为 double

示范将 数字类型 和枚举类型转换为 double: 

template<class T>
inline double to_double(T const& value)
{
    typedef typename boost::promote<T>::type promoted;
    return boost::numeric::converter<double,promoted>::convert(value);
}
