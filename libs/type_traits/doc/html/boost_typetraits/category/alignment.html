用指定的对齐方式组合类型

某些低层内存管理例程需要按照指定的对齐方式组合一个 POD (Plain Old Data) 类型。模板 type_with_alignment 找出符合一个指定的对齐方式的最小类型，模板 aligned_storage 创建一个具有指定大小和对齐方式的类型。

概要 

template <std::size_t Align>
struct type_with_alignment;

template <std::size_t Size, std::size_t Align>
struct aligned_storage;
