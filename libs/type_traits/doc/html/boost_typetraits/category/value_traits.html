这些 traits 全部都是 value traits，也就是说这些 traits 类全部从 integral_constant 继承，并用来访问一个类型的某些数值属性。通常这是一个简单的 true 或 false 的布尔值，但在某些情况下可能会有一些其他的整数值（例如，处理类型排列，或数组范围时：参见 alignment_of, rank 和 extent）。
