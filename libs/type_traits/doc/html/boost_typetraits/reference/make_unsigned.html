make_unsigned 

template <class T>
struct make_unsigned
{
   typedef see-below type;
};

类型：如果 T 是一个无符号整数类型，则结果类型与 T 相同，如果 T 是一个有符号整数类型，则结果为相应的无符号类型。否则，如果 T 是一个枚举类型或字符类型(char 或 wchar_t)，则结果为与 T 相同宽度的无符号整数类型。

如果 T 带有cv限定符，则结果类型也有。

要求：T 必须是整数或枚举类型，且不能是类型 bool。

C++ 标准参考：3.9.1.

头文件：#include <boost/type_traits/make_unsigned.hpp> 或 #include <boost/type_traits.hpp>
