rank 

template <class T>
struct rank : public integral_constant<std::size_t, RANK(T)> {};

继承：当 RANK(T) 是类型为 T 的数组的维数时，类模板 rank 从 integral_constant<std::size_t,RANK(T)> 继承。

如果 T 不是数组类型，那么 RANK(T) 为 0。

头文件：#include<boost/type_traits/rank.hpp> 或 #include<boost/type_traits.hpp>

示例：

rank<int[]> 从 integral_constant<std::size_t,1> 继承。

rank<double[2][3][4]>::type 是 integral_constant<std::size_t,3> 类型。

rank<int[1]>::value 是求值为 1 的 integral constant expression（整常表达式）。

rank<int[][2]>::value 是求值为 2 的 integral constant expression（整常表达式）。

rank<int*>::value 是求值为 0 的 integral constant expression（整常表达式）。

rank<T>::value_type 是 std::size_t 类型。
