floating_point_promotion 

template <class T>
struct floating_point_promotion
{
   typedef see-below type;
};

类型：如果类型为 T 的右值可以提升为浮点数，则对 T 进行浮点数提供并保持 T 的cv限定符，否则 T 保持不变。

C++ 标准参考：4.6.

头文件：#include <boost/type_traits/floating_point_promotion.hpp> 或 #include <boost/type_traits.hpp>
