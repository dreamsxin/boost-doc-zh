remove_volatile 

template <class T>
struct remove_volatile
{
   typedef see-below type;
};

类型： 与 T 的类型相同，但移除了任何 顶层 volatile 修饰符。

C++ 标准参考：3.9.3。

编译器兼容性：如果编译器不支持类模板的偏特化，这个模板也可以编译，但是成员 type 总是和 T 类型相同，除非编译器 workarounds 可用。

头文件：#include<boost/type_traits/remove_volatile.hpp> 或 #include<boost/type_traits.hpp>
