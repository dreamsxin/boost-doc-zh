is_void 

template <class T>
struct is_void : public true_type-or-false_type {};

继承：如果 T 是一个（可能被 cv 修饰的）void 类型，则从 true_type 继承，否则从 false_type 继承。

C++ 标准参考：3.9.1p9。

头文件：#include<boost/type_traits/is_void.hpp> 或 #include<boost/type_traits.hpp>

示例：

is_void<void> 从 true_type 继承。

is_void<constvoid>::type 是 true_type 类型。

is_void<void>::value 是一个求值为 true 的 integral constant expression（整常表达式）。

is_void<void*>::value 是一个求值为 false 的 integral constant expression（整常表达式）。

is_void<T>::value_type 是 bool 类型。
