aligned_storage 

template <std::size_t Size, std::size_t Align>
struct aligned_storage
{
   typedef see-below type;
};

类型：一个大小为 Size 并对齐到 Align 的倍数的内建或 POD 类型。

头文件：#include <boost/type_traits/aligned_storage.hpp> 或 #include <boost/type_traits.hpp>
