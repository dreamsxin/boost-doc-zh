Boost Exception

tuple/operator<<

#include <boost/exception/info_tuple.hpp>

namespace
boost
    {
    template <class E, class Tag1, class T1, ..., class TagN, class TN>
    E const & operator<<( E const & x,
        tuple<
            error_info<Tag1,T1>,
            ...,
            error_info<TagN,TN> > const & v );
    }

要求:

E 必须是 boost::exception, 或者是从 boost::exception 直接或者间接继承的类型.

效果:

等价于 x << v.get<0>() << ... << v.get<N>().

返回值:

x.

异常抛出:

std::bad_alloc, 或者 T1..TN 拷贝构造函数抛出的异常 .

参见:

Boost Exception
