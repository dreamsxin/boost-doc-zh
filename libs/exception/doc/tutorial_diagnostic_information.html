Boost Exception

Diagnostic Information

Boost Exception 提供了一个命名空间域的函数 diagnostic_information ,以 boost::exception 为参数, 返回的字符串包含以下信息:

  • 通过流式操作符 operator<<向 boost::exception 添加的信息;
  • 标准的异常消息 std::exception::what;
  • 和平台相关的调试信息 .

返回的消息不是用户友好的, 但是因为是自动产生的, 非常适合于调试和日志记录. 下面是一个实例:

#include <boost/exception.hpp>
#include <iostream>

void f(); //throws unknown types that derive from boost::exception.

void
g()
    {
    try
        {
        f();
        }
    catch(
    boost::exception & e )
        {
        std::cerr << diagnostic_information(e);
        }
    }
