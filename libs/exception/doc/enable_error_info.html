Boost Exception

enable_error_info

#include <boost/exception/enable_error_info.hpp>

namespace
boost
    {
    template <class T>
    ---unspecified--- enable_error_info( T const & x );
    }

要求:

T 的copy构造函数不能抛出异常,并且是可访问.参见(15.5.1).

返回值:

  • 如果T继承自 boost::exception, 那么返回的对象是x的一个拷贝.
  • 否则, 返回的对象类型不确定,但是确保共有继承自 T 和 boost::exception. T类型的子对象通过T的构造函数初始化.

异常抛出:

没有异常.
