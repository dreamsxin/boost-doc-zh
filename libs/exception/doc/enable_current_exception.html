Boost Exception

enable_current_exception

#include <boost/exception/enable_current_exception.hpp>

namespace
boost
    {
    template <class T>
    ---unspecified--- enable_current_exception( T const & e );
    }

要求:

  • T 的复制构造函数不能抛出异常,并且是可访问。
  • 如果 T 有任何虚拟基类，则那些类型必须具有一个可访问的缺省构造函数。

返回值:

是从T继承的不确定类型. 也就是说, 可以通过语句 catch(T &) 截获异常.

描述:

这个函数可以直接使用在throw语句中,从而使得exception_ptr支持 Boost 异常. 例如:

class
my_exception:
    public std::exception
    {
    };

....
throw boost::enable_current_exception(my_exception());

除非在抛出异常的语句中调用了 enable_current_exception , 否则异常对象的拷贝会使得函数 current_exception 的返回值指向一个 unknown_exception类型的对象.  更详细的信息参见 current_exception .

注意:

相对于直接使用throw关键字, 更好的方式是调用 boost::throw_exception. 这个函数确保抛出的异常对象继承自 boost::exception , 从而也支持 exception_ptr 的功能.
