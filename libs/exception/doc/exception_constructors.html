Boost Exception

exception::exception

exception();
exception( exception const & x );

效果:

  • 默认构造器,初始化一个空的 boost::exception 对象.
  • 拷贝构造器,初始化一个 boost::exception 对象,共享x的所有数据,包括通过流式操作符operator<< 添加的数据, 以及以后添加的数据.

异常抛出:

没有异常.
