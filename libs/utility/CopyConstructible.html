Copy Constructible 可复制构造

描述

一个类型是 Copy Constructible 的如果可以复制该类型的对象。

符号

T 符合 Copy Constructible 的类型
t 类型 T 的一个对象
u 类型 const T 的一个对象

定义

有效表达式

名字 表达式 返回类型 语义

 T u 等价于 T(u)

概念检查类

  template <class T>
  struct CopyConstructibleConcept
  {
    void constraints() {
      T a(b);            // 要求复制构造函数
      T* ptr = &a;       // 要求取址操作符
      const_constraints(a);
      ignore_unused_variable_warning(ptr);
    }
    void const_constraints(const T& a) {
      T c(a);            // 要求常量复制构造函数
      const T* ptr = &a; // 要求常量取址操作符
      ignore_unused_variable_warning(c);
      ignore_unused_variable_warning(ptr);
    }
    T b;
  };
