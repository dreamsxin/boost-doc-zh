throw_exception.hpp
 

头文件 <boost/throw_exception.hpp> 定义了辅助函数 boost::throw_exception. 它被用于需要抛出异常的 Boost 库，可以配置并支持不使用异常的平台，用 BOOST_NO_EXCEPTIONS 配置宏 来指明。

如果 BOOST_NO_EXCEPTIONS 没有被定义，则 boost::throw_exception(e) 等同于 throw e. 否则，该函数为未定义，用户应该给出一个合适的定义。throw_exception 的调用者被允许假定该函数不会返回；因此，如果用户定义的 throw_exception 会返回，则行为也是未定义的。

概要

namespace boost
{

#ifdef BOOST_NO_EXCEPTIONS

void throw_exception(std::exception const & e); // 用户定义

#else

template<class E> void throw_exception(E const & e)
{
    throw e;
}

#endif

}
