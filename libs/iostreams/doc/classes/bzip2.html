简介

类模板 basic_bzip2_compressor 和 basic_bzip2_decompressor 使用 Julian Seward 的 libbzip2压缩库([Seward])执行压缩和解压。

bzip2 过滤器是 DualUseFilters，因此它们可用于输入或输出。不过通常来说，压缩过滤器用于输出而解压过滤器用于输入。

感谢

bzip2 过滤器受到 Jeff Garland ([Garland]) 和 Jonathan de Halleux ([de Halleux]) 的工作所影响。

感谢 Julian Seward 将他的卓越的库以 Boost-compatible 的授权公开。

参考

名字空间 boost::iostreams::bzip2

名字空间 boost::iostreams::bzip2 包含整型的错误代码。这些常量的解释如下。更多详情请见 [Seward] .

常量 解释
data_error 表示压缩数据流被损坏。等同于 BZ_DATA_ERROR.
data_error_magic 表示压缩数据流不是以 'magic' 序列 'B' 'Z' 'h' 开始。等同于 BZ_DATA_ERROR_MAGIC.
config_error 表示 libbzip2 在当前平台上的配置不正确。等同于 BZ_CONFIG_ERROR.

类 bzip2_params

描述

封装用于配置 basic_bzip2_compressor 和 basic_bzip2_decompressor 参数。

摘要

struct bzip2_params {

    // 用于压缩的非显式构造函数
    bzip2_params( int block_size   = default value,
                  int work_factor  = default value );

    // 用于解压的构造函数
    bzip2_params( bool small );

    union {
        int   block_size;    // 用于压缩
        bool  small;         // 用于解压
    };
    int       work_factor;   // 用于压缩
};

bzip2_params::bzip2_params

    bzip2_params( int block_size   = default value,
                  int work_factor  = default value );
    bzip2_params( bool small );

第一个成员构造一个 bzip2_params 对象，用于配置 basic_bzip2_compressor.

block_size - 用于压缩的块大小；实际的块大小为 100000 * block_size. 必须在范围 1-9 中。
work_factor - 控制最差情况下的压缩行为。必须在范围 0-250 中。

第二个成员构造一个 bzip2_params 对象，用于配置 basic_bzip2_decompressor.

small - 表示是否一个慢解压算法，如是则使用更少的内存。

更多详情请见 [Seward] .

类模板 basic_bzip2_compressor

描述

DualUseFilter 的 model，使用 libbzip2 ([Seward]) 执行压缩。

摘要

template<typename Alloc = std::allocator<char> >
struct basic_bzip2_compressor {
    typedef char                    char_type;
    typedef implementation-defined  category;

    basic_bzip2_compressor( const bzip2_params& = bzip2::default_block_size, 
                            std::streamsize buffer_size = default value );
    ...
};

typedef basic_bzip2_compressor<> bzip2_compressor;

模板参数

Alloc - 一个C++标准库的分配器类型([ISO], 20.1.5), 用于分配一个字符缓冲区及配置 libbzip2.

basic_bzip2_compressor::basic_bzip2_compressor

    basic_bzip2_compressor(const bzip2_params&, std::streamsize buffer_size);

以给定参数及缓冲区大小构造一个 basic_bzip2_compressor 实例。由于 bzip2_params 对象可以由一个表示块大小的 int 隐式构造，所以可以用一个 int 作为第一个参数来传递。

例子

以下代码从一个文件解压数据并写出到标准输出。 
#include <fstream>
#include <iostream>
#include <boost/iostreams/filtering_streambuf.hpp>
#include <boost/iostreams/copy.hpp>
#include <boost/iostreams/filter/bzip2.hpp>

int main() 
{
    using namespace std;
    using namespace boost::iostreams;

    ifstream file("hello.bz2", ios_base::in | ios_base::binary);
    filtering_streambuf<input> in;
    in.push(bzip2_decompressor());
    in.push(file);
    boost::iostreams::copy(in, cout);
}

安装

bzip2 过滤器依赖于第三方的 libbz2 库，该库并不包含于 Boost 的分发包中。在多数UNIX和类-UNIX系统上，预构建的 libbz2 二进制文件都是可用的，并可以由 Boost 构建系统自动发现。用户也可以配置 Boost Iostream 库从源代码构建 libbz2，源代码在 libbz2 主页。关于配置构建系统以发现你的 libbz2 安装的详情，请看 Installation.
