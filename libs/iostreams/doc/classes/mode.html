描述

一个 Metafunction元函数，返回某个 Filter 或 Device 概念的 model 的 mode 所对应的 mode tag .

头文件

<boost/iostreams/traits.hpp>

参考

摘要

namespace boost { namespace iostreams {

template<typename T>
struct mode_of {
    typedef see below type;
};

} } // End namespace boost::io

模板参数

T - 某个 Filter 或 Device 概念的 model

mode_of::type

    typedef implementation-defined type;

可从 category_of<T>::type 转换为的，位于派生树最底层的 mode tag . 如果没有这样的最底层标志，则产生一个编译期错误。

下表列出了分配给标准库类型的 Modes.

T category
std::basic_iostream 的特化类及其派生类 seekable
std::basic_istream 的特化类及其派生类 input_seekable
std::basic_ostream 的特化类及其派生类 output_seekable
std::basic_streambuf 的特化类及其派生类 seekable
std::back_insert_iterator 的特化类及其派生类 output
