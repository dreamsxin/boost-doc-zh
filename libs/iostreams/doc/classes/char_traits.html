描述

类模板 boost::iostreams::char_traits 是 std::char_traits 的派生类，增加了一个静态函数 would_block，用于表示某次读入字符的尝试由于当前没有输入可用而失败。

为方便使用，提供了常量 WOULD_BLOCK 和 WWOULD_BLOCK .

头文件

<boost/iostreams/char_traits.hpp>

参考

摘要

namespace boost { namespace iostreams {

const int WOULD_BLOCK;

const std::wint_t WWOULD_BLOCK;

template<typename Ch>
class char_traits : public std::char_traits<Ch> {
    static typename std::char_traits<Ch>::int_type would_block();
};

} } // End namespace boost::io

常量 WOULD_BLOCK

    const int WOULD_BLOCK;

char_traits<char>::would_block 的返回值。被函数模板 get 用于表示由于输入暂时不可用而没有字符可以返回。

常量 WWOULD_BLOCK

    const std::wint_t WWOULD_BLOCK;

char_traits<wchar_t>::would_block 的返回值。被函数模板 get 用于表示由于输入暂时不可用而没有字符可以返回。

类模板 boost::iostreams::char_traits

模板参数

Ch - 字符类型

char_traits::would_block

    static typename std::char_traits<Ch>::int_type would_block();

返回一个 int_type 实例，被函数模板 get 用于表示从一个 Source 读入字符的尝试由于当前没有输入可用而失败。后续的读取字符尝试可能会成功。
