3.4 过滤的流和流缓冲

类模板 filtering_streambuf 和 filtering_stream 是 Iostreams 库提供的用于过滤数据的基础组件。每个 filtering_streambuf 或 filtering_stream 包含一条由零个或多个 过滤器 后跟一可选 设备 所组成的链，以类似于 std::stack 的接口来访问。该链以类模板 chain 表示。如果该链以一个设备结束，则称之为 完全的 并可用于执行 i/o.

该链中的每个 stream_buffer 的 模式 必须强化自 filtering_streambuf 或 filtering_stream 的模式。例如，你不能将一个 输出 模式的过滤器或设备用于一个 输入 模式的 filtering_streambuf.

如果一个 filtering_streambuf 或 filtering_stream 是 输入 模式的，数据将从链的末端流向始端 — 从一个 来源 开始并通过零个或多个 输入过滤器。如果是 输出 模式的，则数据流为相反方向 — 从链的始端开始，通过零个或多个 输出过滤器，流向链末端的 接收器。如果一个 filtering_streambuf 或 filtering_stream 既执行输入也执行输出，则数据在双方向流到。
