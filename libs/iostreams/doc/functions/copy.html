函数模板 copy

描述
头文件
摘要

描述

函数模板 copy 从一个给定的 Source 的 model 读入数据，并写出到一个给定的 Sink 的 model，直至到达流末尾。接着用函数模板 close 关闭两个组件，并返回写出的字符数。

临时缓冲区的大小可以通过一个可选参数指定。

有四个 copy 的重载，以适应所有的 Sources 和 Sinks 组合，分别以 const 引用的方式传递，或者对于流和流缓冲，以非-const 引用的方式传递。

头文件

摘要

namespace boost { namespace iostreams {
              
template<typename Source, typename Sink>
std::streamsize

      std::streamsize buffer_size = default value );

} } // End namespace boost::io
