描述

类模板 combination 接受一对 InputFilter/OutputFilter 或者一对 Source/Sink，并生成一个过滤器或设备，用第一个组件来执行输入同时用第二个来输出。函数模板 combine 接受一对 InputFilter/OutputFilter 或者一对 Source/Sink，返回一个合适的 combination 的特化。

头文件

<boost/iostreams/combine.hpp>

摘要

namespace boost { namespace iostreams {
   
template<typename In, typename Out>
class combination;

template<typename In, typename Out>
combination<In, Out> combine(const In& in, const Out& out);

} } // End namespace boost::io
