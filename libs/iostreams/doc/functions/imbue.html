函数模板 imbue

描述
头文件
参考

描述

当使用 std::basic_ios::imbue 或 std::basic_streambuf::pubimbue 设置 Iostreams 库的流或流缓冲的 std::locale 时，将自动调用函数模板 imbue . 当一个 stream_buffer 或 stream 实例的 std::locale 被设置时，会对底层设备调用 imbue ; 当一个 filtering_streambuf 或 filtering_stream 的 std::locale 被设置时，会对底层链中每个过滤器和设备调用 imbue .

头文件

<boost/iostreams/detail/imbue.hpp>
<boost/iostreams/detail/operations.hpp>

参考

摘要

namespace boost { namespace iostreams {
              
template<typename T>     
void imbue(T& t, const std::locale& loc);

} } // End namespace boost::io

模板参数

T - Filter 或 Device 概念的 model

语义

对于一个过滤器或设备类型 T，imbue 的语义取决于 T 的 category 如下：

category_of<T>::type 语义
可转换为 streambuf_tag 调用 t.pubimbue(loc)
可转换为 localizable_tag 但不可转换为 streambuf_tag 调用 t.imbue(loc)
其它情况 空操作
