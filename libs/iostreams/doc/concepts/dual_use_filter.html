DualUseFilter两用过滤器

描述

DualUseFilter 是一种过滤器，它即可用作 InputFilter 也可用作 OutputFilter, 但不能同时用作两者。

DualUseFilters 有助于减少对同一个过滤算法实现不同类型过滤器的数量。例如，将 basic_regex_filter 实现为一个两用过滤器，Iostreams 库就无需定义独立的模板 basic_regex_input_filter 和 basic_regex_output_filter 了。

强化自

InputFilter, OutputFilter.

关联类型

Character type 被过滤序列的字符类型
Category 一个类型，可转换为 filter_tag 和 dual_use
Mode Category 可转换为的，唯一的，位于派生树最底层的 mode tag

有效表达式 / 语义

与 BidirectionalFilter 一致。

异常

与 BidirectionalFilter 一致。

模型

  • 文本处理过滤器
  • 压缩与解压过滤器，除了 basic_gzip_decompressor.
