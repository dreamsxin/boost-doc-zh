Pipable可管道的

描述

一个可管道的过滤器可以用作某个 pipline 的组件。

要使得一个过滤器成为可管道的，必须重载相应的 operator| ; 这可通过调用 BOOST_IOSTREAMS_PIPABLE 来达到。

强化自

Filter.

关联类型

与 Filter 一致。

符号

P - 一个类型，是 Pipable 的 model
C - 一个类型，是 Filter 或 Device 的 model，且是 CopyConstructible[1]
p - 类型 P 的对象
c - 类型 C 的对象

有效表达式 / 语义

表达式 表达式类型 语义 
p | c
 由实现定义 返回由序列 p, c 组成的一个管道

异常

无。

模型

  • 由 Iostreams 库提供的所有过滤器。

[1]该要求防止流和流缓冲没有使用 reference wrappers 就用于管道。该限制可能会在以后版本的 Iostreams 库中被去除。
