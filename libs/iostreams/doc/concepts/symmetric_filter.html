SymmetricFilter对称过滤器

描述

对称过滤器按以下方式处理字符序列，逐渐从某个指定的缓冲取出字符，并逐渐将已过滤数据填入另一个指定的缓冲区。它可用于定义基于C语言API的过滤器。

对称过滤器并不是强化自 Filter 的。模板 symmetric_filter 可用于从对称过滤器生成一个过滤器。以该方式生成的过滤器将是 DualUseFilters 和 CopyConstructible 的，无论它所基于的对称过滤器是否 CopyConstructible.

请注意，概念 SymmetricFilter 的定义并不包括 Source 或 Sink 的概念。

表达式 表达式类型 语义 后续条件 
f.filter(i1, i2, o1, o2, flush)
 bool 尝试过滤序列 [i1, i2), 将结果保存到序列 [o1, o2). 如果 flush 为 true, 写出尽可能多的输出到 [o1, o2). 如果 flush 为 false, 返回 false 表示检测到一个“自然”的流结束。否则，返回 true 表示有可供输出的额外字符未保存至 [o1, o2).

i1 指向原始序列 [i1, i2) 中的第一个未消耗字符。

o1 指向原始序列 [o1, o2) 中的第一个未消耗字符。

要么 i1 == i2，要么 o1 == o2 

f.close()
 void - f 准备好过滤新的序列

异常

在 filter 或 close 的执行过程中发生的错误将通过抛出异常来表示。
