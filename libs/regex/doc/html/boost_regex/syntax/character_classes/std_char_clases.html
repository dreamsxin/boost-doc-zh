Character Classes that are Always Supported 均被支持的字符类

The following character class names are always supported by Boost.Regex:
下面的字符类名称总被Boost.Regex支持：

Name 名称

POSIX-standard name 是否POSIX标准名称

Description 说明

alnum

Yes

Any alpha-numeric character.
任意单词-数字字符。

alpha

Yes

Any alphabetic character.
任意单词字符。

blank

Yes

Any whitespace character that is not a line separator.
非行分隔符的任意空白字符。

cntrl

Yes

Any control character.
任意控制字符。

d

No

Any decimal digit
任意数字字符。

digit

Yes

Any decimal digit.
任意数字字符。

graph

Yes

Any graphical character.
任意图形字符。

l

No

Any lower case character.
任意小写字符。

lower

Yes

Any lower case character.
任意小写字符

print

Yes

Any printable character.
任意可打印字符。

punct

Yes

Any punctuation character.
任意标点字符。

s

No

Any whitespace character.
任意空白字符。

space

Yes

Any whitespace character.
任意空白字符。

unicode

No

Any extended character whose code point is above 255 in value.
任意码点高于255的扩展字符。

u

No

Any upper case character.
任意大写字符。

upper

Yes

Any upper case character.
任意大写字符。

w

No

Any word character (alphanumeric characters plus the underscore).
任意单词字符(字母、数字和下划线)。

word

No

Any word character (alphanumeric characters plus the underscore).
任意单词字符(字母、数字和下划线)。

xdigit

Yes

Any hexadecimal digit character.
任意十六进制数字字符。
