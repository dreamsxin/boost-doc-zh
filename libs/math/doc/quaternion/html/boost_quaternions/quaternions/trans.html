Quaternion 超越函数

在这个四元数的实现中并没有提供log 或 sqrt函数, 并且函数pow 局限于指数的整数次幂. 关于这一点有如下理由: 一方面, 与四元数对应的解析开拓("分支切割")仍然有需要彻底的研究 (至少对我来说...), 并且我们希望避免重述在标准中介绍的复数的复数次幂(这是有良好定义的，但不在标准中...). 说到无稽之谈, 象说 pow(0,0) 的结果是由 "具体实现规定" 的，就是坏了脑的...

然而，我们确实提供了一些超越函数, 最主要的是指数运算. 作者声明这个 "闭公式"(closed formula)的证明是他自己完成的 也是他的独自的发明(在这个公式被发明之前已经有了一些声明, 比如一份由 Shoemake 教授发表的声明, 有可能这个公式在几百年以前已经被人所知晓, 但是由于缺少参考文献,这件事情一直悬而未决， 等待以后的调查; 另一方面, 四元数的指数运算的定义和存在, 是一个已经被知道很久的事实). 基本上, 任意收敛的实系数幂级数, 在 C 上满足闭公式的, 都能够被转化为对应的H上的幂级数. 这个四元数类型中的更多的超越函数可能会依据需求在将来添加. 需要指出的是，这些函数增加了对boost/math/special_functions/sinc.hpp 和 boost/math/special_functions/sinhc.hpp 头文件 的依赖.

exp 
template<typename T>	quaternion<T> exp(quaternion<T> const & q);

计算四元数的exp(指数).

cos 
template<typename T>	quaternion<T>  cos(quaternion<T> const & q);

计算四元数的cosine(余弦)

sin 
template<typename T>	quaternion<T>  sin(quaternion<T> const & q);

计算四元数的sine(正弦).

tan 
template<typename T>	quaternion<T>  tan(quaternion<T> const & q);

计算四元数的tangent(正切).

cosh 
template<typename T>	quaternion<T>  cosh(quaternion<T> const & q);

计算四元数的 hyperbolic cosine（双曲余弦）.

sinh 
template<typename T>	quaternion<T>  sinh(quaternion<T> const & q);

计算四元数的 hyperbolic sine（双曲正弦）.

tanh 
template<typename T>	quaternion<T>  tanh(quaternion<T> const & q);

计算四元数的 hyperbolic tangent（双曲正切）.

pow 
template<typename T>	quaternion<T>  pow(quaternion<T> const & q, int n);

计算四元数 q 的 n 次幂.

译注: 关于 闭公式(closed formula) 以及 数域 的概念在 wikipedia中有相关的描述.
