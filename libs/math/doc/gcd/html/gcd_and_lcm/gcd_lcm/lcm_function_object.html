LCM（最小公倍数）函数对象

头文件: <boost/math/common_factor_rt.hpp> 

template < typename IntegerType >
class boost::math::lcm_evaluator
{
public:
   // Types
   typedef IntegerType  result_type;
   typedef IntegerType  first_argument_type;
   typedef IntegerType  second_argument_type;

   // Function object interface
   result_type  operator ()( first_argument_type const &a,
   second_argument_type const &b ) const;
};

boost::math::lcm_evaluator 类模板定义了一个用于返回两个整数的最小公倍数的函数对象。 这个模板类通过单个类型进行类型参数化，在这里被称作 IntegerType . 这个类型应当是一个用于表示整数的数值类型。这个函数对象的返回值永远都是非负的， 即便运算符的参数中的任意一个是负数. 如果最小公倍数超过了这个整数类型的范围，那么结果是未定义的.

这个函数对象类模板被用在对应版本的LCM（最小公倍数）函数模板中。 如果某个数值类型想要定制它的最小公倍数的计算， 那么这种类型应当特化LCM(最小公倍数)类模板。
