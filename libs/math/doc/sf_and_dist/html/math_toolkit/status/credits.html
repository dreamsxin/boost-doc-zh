致谢(Credits and Acknowledgements)

Hubert Holin 开启了 Boost.Math 库。反双曲函数和正弦函数是他的。

John Maddock开启了这个库, β函数， γ函数，erf函数，多项式，以及阶乘函数是他的，以及“工具”部分和大多数的统计分布类

Paul A. Bristow threw 抛出了一个挑战A Proposal to add Mathematical Functions for Statistics to the C++ Standard Library ，尤其是对于统计分布的基本函数。 JM 接受这个挑战并解决了这个问题，不只是在数值上的，而是以完全的C++模板风格，PAB 实现了一些统计分布类，PAB也不知疲倦地阅读 JM 丢给他的文档和代码 (所以所有的编辑错误都是他的错误)。

Xiaogang Zhang 为他的Google Summer of Code project 2006实现了 Bessel 函数和椭圆积分。

Bruno Lalande 提交了"运行时基础的编译时幂函数(compile time power of a runtime base)"代码。

Nico Temme 在不完全β函数反函数上提出了宝贵意见。

Victor Shoup for NTL, 没有这个库将很难生成高精度的数值常量，尤其是进行精度测试的数据。

我们感谢Joel Guzman 帮助我们对他的Boost.Quickbook 程序进行强度测试(stress-test)，并使用这个程序来生成这个文档的html 和 pdf 版本，并增加了一些新的特性。

这些函数和分布的图像是基于W3C 标准Scalable Vector Graphic (SVG) 格式，使用Jacob Voytko 在 Google Summer of Code (2007)中开发的应用程序生成。因为浏览器对SVG的支持还不是很广泛的(Microsoft Internet Explorer，甚至是 IE 8 beta版本，仍然缺省对SVG 的本地化支持(native support)，但可以使用Adobe's free SVG viewer 插件来弥补)，因此这些SVG文件都使用Inkscape批处理成JPEG文件。

我们也要感谢 Matthias Schabel 管理对这个库的正式的 Boost-复审 过程，并感谢所有参与复审的人－包括 Guillaume Melquiond， Arnaldur Gylfason， John Phillips， Stephan Tolksdorf 以及 Jeff Garland －感谢他们的许多有帮助的意见。
