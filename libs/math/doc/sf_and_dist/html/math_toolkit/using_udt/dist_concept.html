分布类型的概率性要求(Conceptual Requirements for Distribution Types)

一个 DistributionType 类型是一个实现下面的概念要求的类型，并且包含一种数学统计分布类型。

请注意，这个文档不应当做为参考文档和统计分布手册 的替代品(substitute)。

在下面的表中，d 是一个 DistributionType类型的对象，cd 是一个 const DistributionType 类型的对象且cr 是一个可以转换为RealType类型的对象。

表达式

返回值类型

说明

DistributionType::value_type

RealType

分布所操作的实数类型RealType 。

DistributionType::policy_type

RealType

当计算依赖于这个分布的函数时所使用的策略 。

d = cd

Distribution&

分布类型是可赋值的。

Distribution(cd)

Distribution

分布类型是可拷贝构造的。

pdf(cd, cr)

RealType

返回分布的PDF。

cdf(cd, cr)

RealType

返回分布的CDF。

cdf(complement(cd, cr))

RealType

返回分布的CDF的补集，这与：1-cdf(cd, cr) 是一样的。

quantile(cd, cr)

RealType

返回分布的分位点。

quantile(complement(cd, cr))

RealType

返回分布的分位点，从概率的补集开始，与：quantile(cd, 1-cr) 是一样的。

chf(cd, cr)

RealType

返回分布的累积危险函数(cumulative hazard function)。

hazard(cd, cr)

RealType

返回分布的故障率函数(hazard function)。

kurtosis(cd)

RealType

返回分布的峰态(kurtosis)。

kurtosis_excess(cd)

RealType

返回分布的峰态超越(kurtosis excess)。

mean(cd)

RealType

返回分布的均值(mean)。

mode(cd)

RealType

返回分布的众数(mode)。

skewness(cd)

RealType

返回分布的偏斜(skewness)。

standard_deviation(cd)

RealType

返回分布的标准差。

variance(cd)

RealType

返回分布的方差。
