迭代限制策略(Iteration Limits Policies)

有两个等价的策略来影响这个库中用于实现特殊函数的迭代算法：

template <unsigned long limit = BOOST_MATH_MAX_SERIES_ITERATION_POLICY>
class max_series_iterations;

template <unsigned long limit = BOOST_MATH_MAX_ROOT_ITERATION_POLICY>
class max_root_iterations;

类max_series_iterations 确定在级数计算完成之前所能使用的迭代次数的最大值，并返回计算错误(evaluation_error)。

类max_root_iterations确定在根查找算法完成之前所能使用的迭代次数的最大值，并返回计算错误(evaluation_error)。
