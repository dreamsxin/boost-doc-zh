template <class T>

template <class T, class Policy>
calculated-result-type tgamma(T z, const Policy&);

返回z值的"true gamma" (也即是函数名 tgamma) :

最后一个策略 参数是可选的并且可以用来控制函数的行为: 如何处理错误, 使用哪种层次的精度等等. 参见策略文档了解更多信息。

在库的内部，这个函数有两个高效的版本：一个完全通用的版本，运行速度慢一些但是更精确一些, 以及一个更高效的使用逼近方法的版本，当T的有效数字的个数于一个特定的 兰克泽斯逼近对应时。实际上，任何一个你所遇到的内建的浮点类型都有一个适当的兰克泽斯逼近 定义. 给定足够的运算时间，也有可能使用libs/math/tools/lanczos_generator.cpp产生其它的兰克泽斯逼近 .

函数返回值的类型使用返回值推导法则:来确定。如果T是整型，那么结果类型为double，否则为T。

template <class T>
calculated-result-type tgamma1pm1(T dz);

template <class T, class Policy>
calculated-result-type tgamma1pm1(T dz, const Policy&);

返回tgamma(dz + 1) - 1. 在实现的内部并没有使用加法和减法，这就可以使得即使对很小的d值也会产生精确的结果。然而，这个实现的精度交叉于35个数字或与T类型相关的兰克泽斯逼近 相关，无论哪一个更精确一些。

测试

γ函数很容易测试：阶乘与半整数阶乘可以精确的计算并且与γ函数相比较。此外，在一些独特的区域中的精确测试使用这个函数的一般版本在高精度情况下计算。

函数tgamma1pm1 使用基于方程tgamma(1+dz)-1使用兰克泽斯逼近至精度为100个十进制数字来计算的数据.
