注意

这里所使用的连带勒让德多项式的定义包含一个前导的位相(-1)m。这与Abramowitzt和Stegun(8.6.6)给定的定义相符合，并且被Mathworld和 Mathematica's 勒让德 P 函数使用。然而，在这些使用的定义并没有一直包含这个前导的位相，同时，比较奇怪的是C++ TR1 (assoc_legendre) 也忽略了这一点，除了说明使用Abramowitz和Stegun作为这些问题的权威参考.

template <class T1, class T2, class T3>
calculated-result-type legendre_next(unsigned l, T1 x, T2 Pl, T3 Plm1);

使用针对于勒让德多项式的三项针对关系来实现，这个函数可以用来生成在同一个点x处增长n值的数列。 这种递推关系对于第一类和第二类苗让德多项式都成立.

例如我们可以使用下面的代码来产生最开始的10个多项式值的数组: 

double x = 0.5;  // 横坐标值
vector<double> v;
v.push_back(legendre_p(0, x)).push_back(legendre_p(1, x));
for(unsigned l = 1; l < 10; ++l)
   v.push_back(legendre_next(l, x, v[l], v[l-1]));

形式上，参数为:

l

最后一个计算的多项式的次数l

x

横坐标值

Pl

多项式在次数为 l时的值.

Plm1

多项式在次数n-1为时的值. 

template <class T1, class T2, class T3>
calculated-result-type legendre_next(unsigned l, unsigned m, T1 x, T2 Pl, T3 Plm1);

实现针对于连带勒让德多项式的三项递推关系，这个函数可以用来生成在同一个点x处增长n值的数列。

例如我们可以使用下面的代码来产生最开始的m + 10个多项式值的数组: 

double x = 0.5;  // 横坐标
int m = 10;      // 次数
vector<double> v;
v.push_back(legendre_p(m, m, x)).push_back(legendre_p(1 + m, m, x));
for(unsigned l = 1 + m; l < m + 10; ++l)
   v.push_back(legendre_next(l, m, x, v[l], v[l-1]));

形式上，参数为:

l

最后一个计算的多项式的次数l

m

连带多项式的次数

x

横坐标值

Pl

多项式在次数为 l时的值.

Plm1

多项式在次数n-1为时的值. .

测试

混合的抽样测试使用 functions.wolfram.com产生的数据, 而随机测试数据使用精度为1000-bit的NTL::RR 库产生的数据。

实现

这些函数使用稳定的三项递推关系来实现。这些函数可以保证小的绝对误差，但是在多项式的根的附近不能保证小的相对误差。
