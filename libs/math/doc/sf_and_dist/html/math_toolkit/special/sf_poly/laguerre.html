拉格朗日 (及连带) 多项式

概要 

#include <boost/math/special_functions/laguerre.hpp>


namespace boost{ namespace math{

template <class T>
calculated-result-type laguerre(unsigned n, T x);

template <class T, class Policy>
calculated-result-type laguerre(unsigned n, T x, const Policy&);

template <class T>
calculated-result-type laguerre(unsigned n, unsigned m, T x);

template <class T, class Policy>
calculated-result-type laguerre(unsigned n, unsigned m, T x, const Policy&);

template <class T1, class T2, class T3>
calculated-result-type laguerre_next(unsigned n, T1 x, T2 Ln, T3 Lnm1);

template <class T1, class T2, class T3>
calculated-result-type laguerre_next(unsigned n, unsigned m, T1 x, T2 Ln, T3 Lnm1);


}} // namespaces
说明

函数返回值的类型使用返回值推导法则:来确定。注意，如果有一个单一的模板参数，那么结果的类型与这个模板参数的类型一样；或者如果模板参数的类型为整型，那么结果的类型为double。

最后一个策略 参数是可选的并且可以用来控制函数的行为: 如何处理错误, 使用哪种层次的精度等等. 参见策略文档了解更多信息。

template <class T>
calculated-result-type laguerre(unsigned n, T x);

template <class T, class Policy>
calculated-result-type laguerre(unsigned n, T x, const Policy&);

返回在点x处次数为n的拉格朗日式项式的值:

面的图像显示了最开始的一些拉格朗日多项式的特性：

template <class T>
calculated-result-type laguerre(unsigned n, unsigned m, T x);

template <class T, class Policy>
calculated-result-type laguerre(unsigned n, unsigned m, T x, const Policy&);

返回在点x处次数为m度数为n的连带拉格朗日多项式的值: 

template <class T1, class T2, class T3>
calculated-result-type laguerre_next(unsigned n, T1 x, T2 Ln, T3 Lnm1);

实现针对于拉格朗日多项式的递推关系，这个函数可以用来生成在同一个点x处增长n值的数列。

例如我们可以使用下面的代码来产生最开始的10个多项式值的数组: 

double x = 0.5;  // 横坐标值
vector<double> v;
v.push_back(laguerre(0, x)).push_back(laguerre(1, x));
for(unsigned l = 1; l < 10; ++l)
   v.push_back(laguerre_next(l, x, v[l], v[l-1]));

形式上，参数为:

n

最后一个计算的多项式的次数n.

x

横坐标值

Ln

多项式在次数为 n时的值.

Lnm1

多项式在次数n-1为时的值

template <class T1, class T2, class T3>
calculated-result-type laguerre_next(unsigned n, unsigned m, T1 x, T2 Ln, T3 Lnm1);

实现针对于连带拉格朗日多项式的三项递推关系，这个函数可以用来生成在同一个点x处增长n值的数列。.

例如我们可以使用下面的代码来产生最开始的10个多项式值的数组: 

double x = 0.5;  // 横坐标值
int m = 10;      // 次数
vector<double> v;
v.push_back(laguerre(0, m, x)).push_back(laguerre(1, m, x));
for(unsigned l = 1; l < 10; ++l)
   v.push_back(laguerre_next(l, m, x, v[l], v[l-1]));

形式上，参数为:

n

最后一个计算的多项式的次数n.

m

连带多项式的次数.

x

横坐标值

Ln

多项式在次数为 n时的值.

Lnm1

多项式在次数n-1为时的值.

测试

混合的抽样测试使用 functions.wolfram.com产生的数据, 而随机测试数据使用精度为1000-bit的NTL::RR 库产生的数据。

实现

这些函数使用稳定的三项递推关系来实现。这些函数可以保证小的绝对误差，但是在多项式的根附近不能保证小的相对误差。
