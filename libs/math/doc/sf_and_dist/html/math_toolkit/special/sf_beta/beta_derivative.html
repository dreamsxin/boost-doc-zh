不完全β函数的导数

概要 

#include <boost/math/special_functions/beta.hpp>


namespace boost{ namespace math{ 

template <class T1, class T2, class T3>
calculated-result-type ibeta_derivative(T1 a, T2 b, T3 x);

template <class T1, class T2, class T3, class Policy>
calculated-result-type ibeta_derivative(T1 a, T2 b, T3 x, const Policy&);

}} // namespaces
说明

这个函数在统计分布中具有一定的作用: 它计算不完全β函数ibeta关于x的偏导数。

当T1,T2和T3是不同类型的时候，函数的返回值的类型使用函数返回值推导法则 来确定。

最后一个策略 参数是可选的并且可以用来控制函数的行为: 如何处理错误, 使用哪种层次的精度等等. 参见策略文档了解更多信息。

精确度

几乎和不完全β函数ibeta的精确度相同。

实现

这个函数正好暴露了不完全β函数ibeta的一些内部细节： 参见相关文档了解更多信息。
