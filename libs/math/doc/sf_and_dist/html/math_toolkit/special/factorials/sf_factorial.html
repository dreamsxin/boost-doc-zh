阶乘

概要 

#include <boost/math/special_functions/factorials.hpp>


namespace boost{ namespace math{

template <class T>
T factorial(unsigned i);

template <class T, class Policy>
T factorial(unsigned i, const Policy&);

template <class T>
T unchecked_factorial(unsigned i);

template <class T>
struct max_factorial;

}} // namespaces
说明 
template <class T>
T factorial(unsigned i);

template <class T, class Policy>
T factorial(unsigned i, const Policy&);

返回i!.

最后一个策略 参数是可选的并且可以用来控制函数的行为: 如何处理错误, 使用哪种层次的精度等等. 参见策略文档了解更多信息。

当i <= max_factorial<T>::value时，阶乘的值是通过表查找来计算的, 对于较大的i值, 阶乘函数是以 tgamma函数的方式来实现的。

如果i的值太大以至于结果无法在T类型中进行表示, 那么就调用 溢出错误. 

template <class T>
T unchecked_factorial(unsigned i);

返回i!.

在这个函数内部是通过表查找来计算阶乘结果. 更进一步，它并没有对i的值进行范围检查：由用户来确保i <= max_factorial<T>::value. 这个函数打算用在需要快速表查找的内部循环中, 但是要求注意确保i的值永远都不会增长得太在。

template <class T>
struct max_factorial
{
   static const unsigned value = X;
};

特性类(traits class,也称作萃取类) 定义了可以传递给unchecked_factorial的最大值。成员变量value 可以用在需要整型常量的地方: 用于定义依赖于这个阶乘函数的查找表的大小.

精确性

对于小于max_factorial<T>::value 的变量，结果应当进行正确的舍入. 对于大的参数值，精度和tgamma一样。

测试

基本的合理性检查(sanity check)以及用于检测阶乘查找表正确性的抽样数据: 对于tgamma 函数的主要测试已经处理了这些情况.

Implementation

对于较小的参数，阶乘函数是以查找表的形式来实现的, 对于较大的参数是以tgamma 函数来实现的.
