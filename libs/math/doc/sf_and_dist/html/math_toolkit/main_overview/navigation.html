导航(Navigation)

与配置的浏览器键( browser key )混合使用，下面的键可以作为常见导航任务的方便的快捷键：

快捷键

p - 前一页

n - 下一页

h - home

u - Up

下面的表中显示了如何在常见的浏览器中使用这些快捷键：

浏览器

访问方法

Internet Explorer

Alt+Key 只高亮链接，所以为了移动到下一个链接，你需要使用“Alt + n”然后单击“Enter” 。

Firefox 2.0 and later

Alt+Shift+ 跟随链接的键，例如，“Alt+Shift+n”将会把你带到下一个主题。

Opera

点击 Shift+Esc +访问链接快捷键

Konqueror

点击并松开 Ctrl 键，然后点击访问链接快捷键。

有一些浏览器在它们的网站导航工具栏中使得这些链接可用：例如，在 Opera 中，你可以使用 Ctrl 加上 左方向键和右方向键来在“下一个”和“上一个”主题之间进行移动。
