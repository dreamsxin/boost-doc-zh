返回在特定方向上的下一个可表示值 (函数nextafter)

概要 

#include <boost/math/special_functions/next.hpp>


namespace boost{ namespace math{
 
template <class FPT>
FPT nextafter(FPT val, FPT direction);
 
}} // namespaces
nextafter函数说明

这是包含在C99标准中的nextafter函数的实现 (它同样也是C99中'nexttoward'函数的实现，区别在于仅有一个long double的方向，如果要求的话，通常能够适合它自己的位置)。

注意

C99中的函数必须使用后缀 f 和 l 来区分 float 和 long double 版本的函数. 而C++ 使用函数模板来代替。

返回在y方向上的x的下一个可表示的值。如果x == y ，那么就返回x. 如果x 是无限的，那么返回定义域错误.如果在y 方向上没有这个值，那么就返回返回同错误.

例子 - nextafter

这两个表示使用任意一个方向上的32-bits浮点单位元素: 

1.F最接近(精确的)表示是                             1.00000000
nextafter(1.F, 999) 是                            1.00000012
nextafter(1/f, -999) 是                           0.99999994

0.1F的最接近的(不精确的)表示是                      0.100000001
nextafter(0.1F, 10) 是                            0.100000009
nextafter(0.1F, 10) 是                            0.099999994
