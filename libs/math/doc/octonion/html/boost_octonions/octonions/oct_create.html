八元数创建函数 

template<typename T> octonion<T> spherical(T const & rho, T const & theta, T const & phi1, T const & phi2, T const & phi3, T const & phi4, T const & phi5, T const & phi6);
template<typename T> octonion<T> multipolar(T const & rho1, T const & theta1, T const & rho2, T const & theta2, T const & rho3, T const & theta3, T const & rho4, T const & theta4);
template<typename T> octonion<T> cylindrical(T const & r, T const & angle, T const & h1, T const & h2, T const & h3, T const & h4, T const & h5, T const & h6);

这些函数通过类似于极坐标构造复数的方式来构造八元数, 因为没有严格等价的相对应于八元数的极坐标系.

球面 是一种简单的 极坐标变换, 它以一个距离(magnitude)（正的），以及一个超球面上的由三个角度给定的点作为输入 第一个角度, theta 有一个固有的 从-pi 到 +pi 的范围, 而其它的两个角度有一个固有的从 -pi/2 到 +pi/2的范围 (类似于R3 空间上的球面坐标系). 由于对称性和周期性, 由于存在大量的对称性和周期性，如果这个距离是负数 ( 译注：严格意义上来讲，距离是非负的，作者这里所说的 距离为负数是指使用这个函数的程序员将一个负值作为距离值传递给这个函数) 或者这个角度超过 了它的自然的分布范围也不会有什么麻烦的问题产生. 然而，预期的退化仍然会发生(一个大小为0的距离将会使得角度设置值被忽略...).

圆柱 也是通常的柱面坐标系在R3 空间(译注：就是数学中通常所说的三维坐标空间)上的转换, 柱面坐标反过来也是平面极坐标的另一个派生. 最开始的两个输入(函数的最开始的两个参数)是这个八元数的第一个 C ( 译注：:在数学中 C 表示复数域) 成员的极坐标. 第三个和第四个输入（函数的第三个和第四个参数）分别放入这个八元数的R ( 译注：:在数学中 R 表示实数域) 成员中.

多极（multipolar） 是极坐标系的另一个泛化. 这一次, 八元数的两个C 成员都由极坐标给定.

在这种实现版本的八元数中, 并没有以复数值为参数的构造八元数的操作，因为这种情况颇为复杂.

译注: 关于各种坐标系的概念在 wikipedia中有相关的描述.
