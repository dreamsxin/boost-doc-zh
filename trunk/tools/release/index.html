Release Scripts发行版脚本

 

Introduction 介绍

The release scripts are used by the release management team to build the release distribution files, and perform related release management functions.

发行版脚本用于发行版管理团队建立发行版文件，并实施相关的发行管理功能。

The files being built include:

建立的文件包括：

Files
boost_x_xx_x.7z  
boost_x_xx_x.tar.bz2 Unix-style line endings
boost_x_xx_x.tar.gz Unix-style line endings
boost_x_xx_x.zip Windows-style line endings

文件
boost_x_xx_x.7z  
boost_x_xx_x.tar.bz2 Unix风格的行结尾
boost_x_xx_x.tar.gz Unix风格的行结尾
boost_x_xx_x.zip Windows风格的行结尾

The content of all files is identical except for the line endings.

除了行结尾外，所有文件的内容是一致的。
