BoostBook element template-type-parameter

template-type-parameter — Declares a template type parameter
template-type-parameter — 声明一个模板类型参数

Synopsis 概要

template-type-parameter ::= (default?, purpose?)

Attributes 属性

Name 名字 Type 类型 Value 值 Purpose 用途
last-revision #IMPLIED CDATA Set to $Date: 2008-06-15 15:48:40 -0400 (Sun, 15 Jun 2008) $ to keep "last revised" information in sync with CVS changes
设置为 $Date: 2008-06-15 15:48:40 -0400 (Sun, 15 Jun 2008) $，以保持 "最后修订" 信息与CVS的变化同步
pack#IMPLIEDCDATASet to '1' if the parameter is a parameter pack.
如果参数是一个参数包，则设置为 '1'
name #REQUIRED CDATA The name of the element being declared to referenced
被声明为被引用元素的名字
id #IMPLIED CDATA A global identifier for this element
该元素的全局标识符
xml:base #IMPLIED CDATA Implementation detail used by XIncludes
用于 XIncludes 的实现细节
