BoostBook element inherit

inherit — Declares a base class of the enclosing class or struct
inherit — 声明所属类或结构的一个基类

Synopsis 概要

inherit ::= (type, purpose?)

Description 说明

This element contains the type of the class inherited.
该元素包含所继承类的类型。

Attributes 属性

Name 名字 Type 类型 Value 值 Purpose 用途
last-revision #IMPLIED CDATA Set to $Date: 2008-06-15 15:48:40 -0400 (Sun, 15 Jun 2008) $ to keep "last revised" information in sync with CVS changes
设置为 $Date: 2008-06-15 15:48:40 -0400 (Sun, 15 Jun 2008) $，以保持 "最后修订" 信息与CVS的变化同步
access#REQUIREDCDATAThe access specifier ("public", "private", or "protected") of the inheritance.
继承的访问指示符("public", "private", 或 "protected")
id #IMPLIED CDATA A global identifier for this element
该元素的全局标识符
pack#IMPLIEDCDATASet to '1' if this is a pack exapansion.
如果这是一个包扩展，则设置为 '1'
xml:base #IMPLIED CDATA Implementation detail used by XIncludes
用于 XIncludes 的实现细节
