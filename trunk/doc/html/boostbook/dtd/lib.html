BoostBook element lib

lib — A library dependency
lib — 一个库的依赖关系

Synopsis 概要

lib ::= (#PCDATA)

Description 说明

Declares a library dependency on the library named by the content of this element, to be emitted in a Jamfile.
声明一个库的依赖关系，该库由元素的内容给出，将被发表到 Jamfile。
