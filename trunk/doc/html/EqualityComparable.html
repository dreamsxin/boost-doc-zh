Concept EqualityComparable 可等价比较概念

EqualityComparable 可等价比较

Description 说明

Equality Comparable types must have == and != operators.
可等价比较的类型必须有 == 和 != 操作符。

Notation 符号

X
A type playing the role of comparable-type in the EqualityComparable concept.
在 可等价比较 概念中担任可比较类型角色的类型。
x, y
Objects of type X
类型 X 的对象

可转换为 bool
