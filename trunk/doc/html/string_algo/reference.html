Header <boost/algorithm/string/formatter.hpp>

定义 Formatter 生成器。Formatter 是一个根据给定参数格式化一个 string 的仿函数。Formatter 总是和 Finder 协力工作。Finder 能为特定的 Formatter 提供额外信息。这种合作的一个例子就是 regex_finder 和 regex_formatter。

Formatters 通常用作替换部件的可插入组件。这个头文件包含本库提供的 Formatters 的生成器函数。

namespace boost {
  namespace algorithm {
    template<typename RangeT> unspecified const_formatter(const RangeT &);
    template<typename RangeT> unspecified identity_formatter();
    template<typename RangeT> unspecified empty_formatter(const RangeT &);
  }
}

Header <boost/algorithm/string/iter_find.hpp>

定义通用的分割算法。分割算法用来将一个序列根据给定规则分为若干个部分。结果以“容器的容器”的方式给出，它的元素是选取部分的拷贝或引用。

提供连个算法。一个迭代遍历匹配的 substring，另一个遍历这些匹配项之间的间隔。

namespace boost {
  namespace algorithm {
    template<typename SequenceSequenceT, typename RangeT, typename FinderT> 
      SequenceSequenceT & iter_find(SequenceSequenceT &, RangeT &, FinderT);
    template<typename SequenceSequenceT, typename RangeT, typename FinderT> 
      SequenceSequenceT & iter_split(SequenceSequenceT &, RangeT &, FinderT);
  }
}

Header <boost/algorithm/string/predicate.hpp>

定义 string 相关的判断式。判断式用于判定在各种条件下，一个 substring 是否包含于输入 string，条件为：一个 string 以这个 substring 开始，以这个 substring 结尾，简单地包含这个 substring 以及两个 string 是否相等。另外，算法 all() 检查一个容器中的所有元素是否满足一个条件。

Header <boost/algorithm/string/sequence_traits.hpp>

定义在这个头文件中的 traits 被各种不同的算法使用，可以使特定的容器达到更好的性能。traits 缺省提供自动保险。如果一个容器支持这些特性中的某一些，它就有可以为这个容器指定特定的 trait。对于先天不足的编译器，为一个特定的测试函数定义一个重载也是有可能的。

由于一个语言约束，现在还不可能不包含相应的头文件就为 stl 容器定义特化。为了减少这种包含的成本，用户可以为一个特定的容器选择性地包含一个特化的头文件。它们位于 boost/algorithm/string/stl 目录。或者还可以包含 boost/algorithm/string/std_collection_traits.hpp 头文件，它包含所有 stl 容器的特化。

namespace boost {
  namespace algorithm {
    template<typename T> class has_native_replace;
    template<typename T> class has_stable_iterators;
    template<typename T> class has_const_time_insert;
    template<typename T> class has_const_time_erase;
  }
}

Header <boost/algorithm/string/split.hpp>

定义基本的分割算法。分割算法可以根据给定的规则将一个 string 分为若干个部分。

每一个部分被拷贝并作为一个新的元素加入到输出容器中。因此结果容器必须能够持有匹配项（类似于 std::string 这样的兼容结构中）的拷贝或引用（例如，使用迭代器范围类）。这样的容器的例子包括 std::vector<std::string> 或 std::list<boost::iterator_range<std::string::iterator>>。

Header <boost/algorithm/string/trim.hpp>

定义修剪算法。修剪算法用于从序列（string）中移除末尾的和开头的空格。空格使用给定的地区设置进行识别。
