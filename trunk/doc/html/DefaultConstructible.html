Concept DefaultConstructible 可缺省构造概念

DefaultConstructible 可缺省构造

Description 说明

DefaultConstructible objects only need to have a default constructor.
可缺省构造的对象只需具有一个缺省构造函数。

Notation 符号

X
A type playing the role of default-constructible-type in the DefaultConstructible concept.
在 可缺省构造 概念中担任可缺省构造类型角色的类型。

Valid expressions 有效表达式

Name 名字 Expression 表达式 Type 类型 Semantics 语义

Construction 构造

X()

X

Construct an instance of the type with default parameters.

以缺省参数构造该类型的一个实例。

Models 模型
