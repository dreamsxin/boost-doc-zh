    If what.prefix().first != what[0].second and if the element match_prev_avail is not set in flags then sets it. Then behaves as if by calling regex_search(what[0].second, end, what, *pre, flags), with the following variation: in the event that the previous match found was of zero length (what[0].length() == 0) then attempts to find a non-zero length match starting at what[0].second, only if that fails and provided what[0].second != suffix().second does it look for a (possibly zero length) match starting from what[0].second + 1. If no further match is found then sets this equal to the end of sequence iterator.
    如 果 what.prefix().first != what[0].second 且元素 match_prev_avail 在 flags 中未置位，则将它置位。然后其行为就象调用 regex_search(what[0].second, end, what, *pre, flags)，但有以下不同：当前一个匹配的长度为零(what[0].length() == 0)则尝试找出一个从 what[0].second 开始的非零长度的匹配，仅当失败且 what[0].second != suffix().second 时，从 what[0].second + 1 开始查找一个(可能长度为零)的匹配。如果找不到更多的匹配，则将它设为等同结束迭代器。

    Postconditions:

    (*this)->size() == pre->mark_count() + 1

    (*this)->empty() == false

    (*this)->prefix().first == An iterator denoting the end point of the previous match found 表示前一个匹配的结束点的迭代器

    (*this)->prefix().last == (**this)[0].first

    (*this)->prefix().matched == (*this)->prefix().first != (*this)->prefix().second

    (*this)->suffix().first == (**this)[0].second

    (*this)->suffix().last == end

    (*this)->suffix().matched == (*this)->suffix().first != (*this)->suffix().second

    (**this)[0].first == The starting iterator for this match. 此次匹配的开始迭代器

    (**this)[0].second == The ending iterator for this match. 此次匹配的结束迭代器

    (**this)[0].matched == true if a full match was found, and false if it was a partial match (found as a result of the match_partial flag being set). 如果找到一个全匹配则为 true，如果是部分匹配则为 false

    (**this)[n].first == For all integers n < (*this)->size(), the start of the sequence that matched sub-expression n. Alternatively, if sub-expression n did not participate in the match, then end. 对于所有 n < (*this)->size() 的整数，则为匹配子表达式n的序列的始端。此外，如果子表达式n没有参与匹配，则为 end。

    (**this)[n].second == For all integers n < (*this)->size(), the end of the sequence that matched sub-expression n. Alternatively, if sub-expression n did not participate in the match, then end. 对于所有 n < (*this)->size() 的整数，则为匹配子表达式n的序列的末端。此外，如果子表达式n没有参与匹配，则为 end。

    (**this)[n].matched == For all integers n < (*this)->size(), true if sub-expression n participated in the match, false otherwise.对于所有 n < (*this)->size() 的整数，如果子表达式n参与了匹配则为 true，否则为 false。

    (*this)->position() == The distance from the start of the original sequence being iterated, to the start of this match. 从原有序列的始端至此次匹配的始端的距离。
