Struct template matches

boost::proto::matches — A Boolean metafunction that evaluates whether a given expression type matches a grammar.
boost::proto::matches — 一个布尔元函数，判断一个表达式类型是否匹配一个语法。

Synopsis

template<typename Expr, typename Grammar> 
struct matches :  mpl::bool_<true-or-false> {
};

Description

proto::matches<Expr, Grammar> inherits from mpl::true_ if Expr::proto_base_expr matches Grammar::proto_base_expr, and from mpl::false_ otherwise.
proto::matches<Expr, Grammar> 继承自 mpl::true_， 如果 Expr::proto_base_expr 匹配 Grammar::proto_base_expr， 否则继承自 mpl::false_。

Non-terminal expressions are matched against a grammar according to the following rules:
非终结符表达式按以下规则匹配一个语法：

  • The wildcard pattern, proto::_ , matches any expression.
    通配符模板 proto::_ 匹 配任意表达式。

  • An expression proto::expr<AT, proto::listN <A0,...An> > matches a grammar proto::expr<BT, proto::listN <B0,...Bn> > if BT is proto::_ or AT, and if Ax matches Bx for each x in [0,n].
    表达式 proto::expr<AT, proto::listN <A0,...An> > 匹配语法 proto::expr<BT, proto::listN <B0,...Bn> > 如果 BT 为 proto::_ 或 AT，且如果对于 [0,n] 中的每个 x 有 Ax 匹配 Bx。

  • An expression proto::expr<AT, proto::listN <A0,...An,U0,...Um> > matches a grammar proto::expr<BT, proto::listM <B0,...Bn,proto::vararg<V> > > if BT is proto::_ or AT, and if Ax matches Bx for each x in [0,n] and if Ux matches V for each x in [0,m].
    表达式 proto::expr<AT, proto::listN <A0,...An,U0,...Um> > 匹配语法 proto::expr<BT, proto::listM <B0,...Bn,proto::vararg<V> > > 如果 BT 为 proto::_ 或 AT，且如果对于 [0,n] 中的每个 x 有 Ax 匹 配 Bx， 且如果对于 [0,m] 中的每个 x 有 Ux 匹配 V。

  • An expression E matches proto::or_<B0,...Bn> if E matches some Bx for x in [0,n].
    表达式 E 匹配 proto::or_<B0,...Bn> 如果对 于 [0,n] 中的某个 x 有 E 匹配 Bx。

  • An expression E matches proto::and_<B0,...Bn> if E matches all Bx for x in [0,n].
    表达式 E 匹配 proto::and_<B0,...Bn> 如 果对于 [0,n] 中的每个 x 有 E 匹配 Bx。

  • An expression E matches proto::if_<T,U,V> if:
    表达式 E 匹配 proto::if_<T,U,V> 如 果：

    • boost::result_of<proto::when<proto::_,T>(E)>::type::value is true and E matches U, or
      boost::result_of<proto::when<proto::_,T>(E)>::type::value 为 true 且 E 匹配 U，或者
    • boost::result_of<proto::when<proto::_,T>(E)>::type::value is false and E matches V.
      boost::result_of<proto::when<proto::_,T>(E)>::type::value 为 false 且 E 匹配 V.

    Note: U defaults to proto::_ and V defaults to proto::not_<proto::_> .
    注意：U 缺省为 proto::_ 而 V 缺省为 proto::not_<proto::_> .

  • An expression E matches proto::not_<T> if E does not match T.
    表达式 E 匹配 proto::not_<T> 如果 E 不匹配 T.

  • An expression E matches proto::switch_<C> if E matches C::case_<E::proto_tag>.
    表达式 E 匹配 proto::switch_<C> 如果 E 匹配 C::case_<E::proto_tag>.

A terminal expression proto::expr<proto::tag::terminal, proto::term<A> > matches a grammar proto::expr<BT, proto::term<B> > if BT is proto::_ or proto::tag::terminal and one of the following is true:
终结符表达式 proto::expr<proto::tag::terminal, proto::term<A> > 匹配语法 proto::expr<BT, proto::term<B> > 如果 BT 为 proto::_ 或 proto::tag::terminal 且以下一条为真：

  • B is the wildcard pattern, proto::_
    B 为通配符模板 proto::_

  • A is B
    A 为 B

  • A is B &
    A 为 B &

  • A is B const &
    A 为 B const &

  • B is proto::exact<A>
    B 为 proto::exact<A>

  • B is proto::convertible_to<X> and boost::is_convertible<A,X>::value is true.
    B 为 proto::convertible_to<X> 且 boost::is_convertible<A,X>::value 为 true.

  • A is X[M] or X(&)[M] and B is X[proto::N] .
    A 为 X[M] 或 X(&)[M] 且 B 为 X[proto::N] .

  • A is X(&)[M] and B is X(&)[proto::N] .
    A 为 X(&)[M] 且 B 为 X(&)[proto::N] .

  • A is X[M] or X(&)[M] and B is X*.
    A 为 X[M] 或 X(&)[M] 且 B 为 X*.

  • B lambda-matches A (see below).
    B 以lambda方式匹配 A (见后)。

A type B lambda-matches A if one of the following is true:
类型 B 以lambda方式匹配 A，如果以下一条为真：

  • B is A
    B 为 A

  • B is the wildcard pattern, proto::_
    B 为通配符模板 proto::_

  • B is T<B0,...Bn> and A is T<A0,...An> and for each x in [0,n], Ax and Bx are types such that Ax lambda-matches Bx
    B 为 T<B0,...Bn> 且 A 为 T<A0,...An> 且对于 [0,n] 中的每个 x，Ax 和 Bx 的类型均满足 Ax 以lambda方式匹配 Bx
