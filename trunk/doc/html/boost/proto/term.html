Struct template term

boost::proto::term — A type sequence, for use as the 2nd parameter to the proto::expr<> class template.
boost::proto::term — 一个类型序列，用作 proto::expr<> 类模板的第二个参数。

Synopsis 概要

template<typename Arg0> 
struct term {
  // types
  typedef Arg0 child0;
  static const long arity;
};

Description 说明

A type sequence, for use as the 2nd parameter to the proto::expr<> class template. The types in the sequence correspond to the children of a node in an expression tree.
一个类型序列，用作 proto::expr<> 类模板的第二个参数。该序列中的类型对应于表达式树中某个节点的子节点。
