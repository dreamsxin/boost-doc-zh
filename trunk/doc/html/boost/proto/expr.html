Struct template expr

boost::proto::expr — Representation of a node in an expression tree.
boost::proto::expr — 表达式树中一个节点的表示法。

  // types

Description

proto::expr<> is a node in an expression template tree. It is a container for its child sub-trees. It also serves as the terminal nodes of the tree.
proto::expr<> 是表达式模板树中的一个节点。它是其子树的容器。它也可作为树的终结符节点。

Tag is type that represents the operation encoded by this expression. It is typically one of the structs in the boost::proto::tag namespace, but it doesn't have to be. If Arity is 0 then this expr<> type represents a leaf in the expression tree.
Tag 是表示该表达式所代表的操作的类型。它通常是一个位于 boost::proto::tag 名字空间中的结构，但不需要一定是。如果 Arity 为0，则这个 expr<> 类型表示表达式树中的一个叶节点。

Args is a list of types representing the children of this expression. It is an instantiation of one of proto::list1<>, proto::list2<>, etc. The child types must all themselves be either expr<> or proto::expr<>&, unless Arity is 0, in which case Args must be proto::term<T>, where T can be any type.
Args 是表示该形式各子节点的一个类型链表。它是 proto::list1<>, proto::list2<>, 等类型之一的实例。子节点的类型必须全部都是 expr<> 或 proto::expr<>&，除非 Arity 为0，这时 Args 必须为 proto::term<T>，其中 T 可以是任意类型。

proto::expr<> is a valid Fusion random-access sequence, where the elements of the sequence are the child expressions.
proto::expr<> 是一个有效的Fusion随机访问序列，该序列的各元素为各个子表达式。
