Function eval

boost::proto::eval — Evaluate a given Proto expression with a given context.
boost::proto::eval — 以给定上下文对给定的Proto表达式求值。

Synopsis

  eval(Expr & expr, Context & context);

  eval(Expr & expr, Context const & context);

Description

Parameters:
