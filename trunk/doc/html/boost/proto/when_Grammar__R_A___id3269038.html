boost::proto::when<Grammar, R(A...)> — A grammar element and a PrimitiveTransform that associates a transform with the grammar.

Description

Use proto::when<> to override a grammar's default transform with a custom transform. It is for used when composing larger transforms by associating smaller transforms with individual rules in your grammar.
使用 proto::when<> 可以将一个语法的缺省变换覆盖为一个定制的变换。它用于通过将较小的变换至你的语法中的个别规则组合出较大的变换。

The when<G, R(A...)> form accepts either a CallableTransform or an ObjectTransform as its second parameter. proto::when<> uses proto::is_callable<R>::value to distinguish between the two, and uses proto::call<> to evaluate CallableTransforms and proto::make<> to evaluate ObjectTransforms.
when<G, R(A...)> 形式接受一个 可调用变换 或一个 对象变换 作为其第二个参数。proto::when<> 使用 proto::is_callable<R>::value 来区分这两种参数，使用 proto::call<> 来对 可调用变换 求值，使用 proto::make<> 来对 对象变换 求值。
