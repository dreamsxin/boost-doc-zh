函数 any_cast

boost::any_cast — 定制转型用于从 any 取出给定类型的值。

Synopsis摘要

template<typename T> T any_cast(any & operand);
template<typename T> T any_cast(const any & operand);
template<typename ValueType> const ValueType * any_cast(const any * operand);
template<typename ValueType> ValueType * any_cast(any * operand);

Description描述

Returns: 如果传入一个指针，则在成功时返回一个具有相同限定的指针，指向所持内容的值，失败则返回null。如果 T 是 ValueType，则返回所持值的拷贝；如果 T 是一个 ValueType引用(可能是 const 限定的)，则返回一个所持值的引用。
Throws: 对于 any 指针的重载方式不会抛出异常；而对于 any 值或引用的重载方式则在失败时抛出 bad_any_cast。
