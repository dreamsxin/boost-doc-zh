类 greg_day

boost::gregorian::greg_day — 表示月份中的日期(范围 1 - 31).

概要

class greg_day {
public:
  // construct/copy/destruct
  greg_day(unsigned short);

  // public member functions
  unsigned short as_number() const;
  operator unsigned short() const;
};

说明

这个小型类允许将一个整数值简单地转换为标准格里历的月份中的日期。该类型会自动检测，超出范围 1-31 的值将引起 bad_day_of_month 异常。

greg_day 构造/复制/析构

  1. greg_day(unsigned short day_of_month);

greg_day 公有成员函数

  2. operator unsigned short() const;
