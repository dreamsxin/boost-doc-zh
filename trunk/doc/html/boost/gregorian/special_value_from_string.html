函数 special_value_from_string

boost::gregorian::special_value_from_string — 由字符串参数返回 special_value.

概要

BOOST_DATE_TIME_DECL special_values 
special_value_from_string(const std::string & s);

说明

由字符串参数返回 special_value. 如果传入的参数不是特殊值名之一(在 src/gregorian/names.hpp 中定义)，则返回 'not_special'
