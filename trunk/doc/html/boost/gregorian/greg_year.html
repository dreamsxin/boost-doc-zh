类 greg_year

boost::gregorian::greg_year — 表示年份(范围 1900 - 10000).

概要

class greg_year {
public:
  // construct/copy/destruct
  greg_year(unsigned short);

  // public member functions
  operator unsigned short() const;
};

说明

这个小类允许将一个整数值转换为格里历的年份。当前允许的范围是从 1900 到 10000。范围的两端都可能会有变化，它们只是本库当前测试的限制值。以后它们有可能会被加大。
