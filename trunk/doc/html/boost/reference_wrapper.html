类模板 reference_wrapper

boost::reference_wrapper —— 容纳一个引向类型为 T 的对象的引用。

Synopsis 概要

// In header: <boost/ref.hpp>

template<typename T> 
class reference_wrapper {
public:
  // types
  typedef T type;

  // construct/copy/destruct
  explicit reference_wrapper(T&);

  // access
  operator T&() const;
  T& get() const;
  T* get_pointer() const;
};

// constructors
reference_wrapper<T> ref(T&);
reference_wrapper<T const> cref(T const&);

// access
unwrap_reference<T>::type& unwrap_ref(T&);

Description 说明

reference_wrapper 主要用于把引用“喂”给那些以传值方式持有它们的参数的函数模板（算法）。它提供一个到 T& 的隐式转换，这通常允许函数模板可以不加改变地工作在引用上。

reference_wrapper construct/copy/destruct

  1. explicit reference_wrapper(T& t);
    作用： 构造一个存储引向 t 的引用的 reference_wrapper 对象。
    抛出： 不会抛出异常。

reference_wrapper access

  1. operator T&() const;
    返回： 所存储的引用。
    抛出： 不会抛出异常。 
  2. T& get() const;
    返回： 所存储的引用。
    抛出： 不会抛出异常。 
  3. T* get_pointer() const;
    返回： 一个指向所存储的引用引向的对象的指针。
    抛出： 不会抛出异常。

reference_wrapper constructors

  1. reference_wrapper<T> ref(T& t);
    返回： reference_wrapper<T>(t)
    抛出： 不会抛出异常。 
  2. reference_wrapper<T const> cref(T const& t);
    返回： reference_wrapper<T const>(t)
    抛出： 不会抛出异常。

reference_wrapper access

  1. unwrap_reference<T>::type& unwrap_ref(T& t);

    返回：

    		 unwrap_reference<T>::type&(t)

    抛出：

    		 不会抛出异常。
