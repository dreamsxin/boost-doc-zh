Class template period_parser

boost::date_time::period_parser — 不是一个 facet, 是一个指定和控制时间段分析的类。

Description

提供以下设置：

  • period_separator -- 缺省为 '/'

  • period_open_start_delimeter -- 缺省为 '['

  • period_open_range_end_delimeter -- 缺省为 ')'

  • period_closed_range_end_delimeter -- 缺省为 ']'

  • display_as_open_range, display_as_closed_range -- 缺省为 closed_range

对于典型的 date_period, 输入流中的内容应该是 

  [2004-Jan-04/2004-Feb-01]
   *

其中 date 的格式由 date facet 控制。

    这个泛型代码使用 facet 分析一个时间段，取得各'元素'。例如，对于一个 date_period，元素是 date 实例，依据传入的 facet 参数中的设置进行分析。

    对一个时间段进行分析的步骤是一致的：

    • 消耗 start 界定符

    • 取出 start 元素

    • 消耗分隔符

    • 根据区间的设置取出 last 或 end 元素

    • 根据区间的设置消耗 end 界定符

    因此，一个典型的 date period，输入流的内容应该如下：

        [March 01, 2004/June 07, 2004]   <-- closed range
        [March 01, 2004/June 08, 2004)   <-- open range

         *
