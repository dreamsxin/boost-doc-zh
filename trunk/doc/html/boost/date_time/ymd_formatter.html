类模板 ymd_formatter

boost::date_time::ymd_formatter — 使用格式化策略将 ymd 转换为标准字符串。

概要

template<typename ymd_type, typename format_type, typename charT = char> 
class ymd_formatter {
public:

  // public static functions
  static std::basic_string< charT > ymd_to_string(ymd_type) ;
  static std::string ymd_to_string(ymd_type) ;
};

说明

ymd_formatter 公有静态函数

  1. static std::basic_string< charT > ymd_to_string(ymd_type ymd) ;

    这是以基于年-月-日的日期信息处理日期格式化的标准代码。该函数使用 format_type 来控制字符串是否包含分隔符，以及如果有的话是什么字符。另外，它也可以将月份格式化为整数或字符串，这由格式化策略控制。
