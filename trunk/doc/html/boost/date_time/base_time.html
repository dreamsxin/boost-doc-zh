Class template base_time

boost::date_time::base_time — 表示时间上的一个精确时刻，包含日期。

Description

该类是解析度高于天级的时间类型的接口的一个骨架。其目的是以此类为基类，真实的时间类以BN模式派生。以此方法，派生类可以决定象'是否具有缺省构造函数'以及缺省构造函数应该如果设置其值，是否具有可选的构造函数允许只有一个 time_durations 并且从时钟生成一个时间，等等问题。因此，实际上可以以不同的构造策略从一个 time_system 创建多种时间类型，这些类型可以只编写一个复制构造函数就执行基本的操作。最后，编译错误也更短。

时间类的真正行为由 time_system 的模板参数提供。该类必须提供所有加、减逻辑，并定义所有接口类型。
