Description

date 模板表示一个接口外壳，用于基于‘年-月-日’系统的日期类，如格里历或iso系统。它提供了计算和比较的基本操作。

理论

这个日期表示法基本上来自于 C 的 tm 结构。这个类型的目标是，在一个具体类中提供高效的日期操作(加减)和存储(以最小空间来表示)。因此，date 在其内部用一个计数器来表示特定的日期。历法参数定义了内部表示与年-月-日间的转换策略。对于需要对同一个日期反复执行多次格式化的应用程序，采用年- 月-日表示法效果更好。

在内部，date 使用一个天数来表示日期。这是一个不变的时间表示法。该表示法允许快速的比较且易于编写相关的数值操作。本质上，内部的天数类似于调整后的儒略日。调整量 取决于纪元日，即表示历法的第1天的日期。第0天保留用于表示负无限，所以任何实际的日期都自动大于负无限。在从一个日期构造日期时，或进行格式化用于输 出时，要进行适当的转换，创建相应的年月日表示。
