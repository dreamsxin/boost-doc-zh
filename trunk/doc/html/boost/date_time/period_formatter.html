类模板 period_formatter

boost::date_time::period_formatter — 不是一个 facet, 只是一个用于指定和控制时间段格式的类。

概要

         typename OutItrT = std::ostreambuf_iterator<CharT, std::char_traits<CharT> > > 
class period_formatter {
public:

说明

提供以下设置：

  • period_separator -- 缺省为 '/'

  • period_open_start_delimeter -- 缺省为 '['

  • period_open_range_end_delimeter -- 缺省为 ')'

  • period_closed_range_end_delimeter -- 缺省为 ']'

  • display_as_open_range, display_as_closed_range -- 缺省为 closed_range

所以，时间段的缺省格式如下： 

  [period.start()/period.last()]
   *

对于典型的 date_period，为 

  [2004-Jan-04/2004-Feb-01]
   *

其中日期的格式由 date facet 控制。

    这个泛型代码使用 facet 输出时间段的‘元素’。例如，在 date_period 的情况下，元素为根据传入的 facet 参数进行格式后的日期实例。

    对时间段进行格式化的步骤是一致的：

    • 输出 start 界定符

    • 输出 start 元素

    • 输出分隔符

    • 依据区间设置输出 last 或 end 元素

    • 依据区间设置输出 end 界定符

    所以，对于典型的日期段，结果如下： 

        [March 01, 2004/June 07, 2004]   <-- 闭区间
        [March 01, 2004/June 08, 2004)   <-- 开区间
