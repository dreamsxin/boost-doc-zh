Class template linear_slist_algorithms

boost::intrusive::linear_slist_algorithms

Synopsis

template<typename NodeTraits> 
class linear_slist_algorithms {
public:
  // types
  typedef NodeTraits::node           node;          
  typedef NodeTraits::node_ptr       node_ptr;      
  typedef NodeTraits::const_node_ptr const_node_ptr;
  typedef NodeTraits                 node_traits;   

  // public static functions
  static void init(node_ptr) ;
  static bool unique(const_node_ptr) ;
  static bool inited(const_node_ptr) ;
  static void unlink_after(node_ptr) ;
  static void unlink_after(node_ptr, node_ptr) ;
  static void link_after(node_ptr, node_ptr) ;
  static void transfer_after(node_ptr, node_ptr, node_ptr) ;
  static void init_header(node_ptr) ;
  static node_ptr get_previous_node(node_ptr, node_ptr) ;
  static std::size_t count(const_node_ptr) ;
  static void swap_trailing_nodes(node_ptr, node_ptr) ;
  static node_ptr reverse(node_ptr) ;
  static std::pair< node_ptr, node_ptr > 
  move_first_n_backwards(node_ptr, std::size_t) ;
  static std::pair< node_ptr, node_ptr > 
  move_first_n_forward(node_ptr, std::size_t) ;
};

Description

linear_slist_algorithms provides basic algorithms to manipulate nodes forming a linear singly linked list.
linear_slist_algorithms 提供了操纵形成一个线性单链表的节点的基本算法。

linear_slist_algorithms is configured with a NodeTraits class, which encapsulates the information about the node to be manipulated. NodeTraits must support the following interface:
linear_slist_algorithms 用一个 NodeTraits 类来配置，它封装了被操纵的节点的信息。NodeTraits 必须支持以下接口：

Typedefs:

node: The type of the node that forms the linear list 形成线性链表的节点的类型

node_ptr: A pointer to a node 节点指针

const_node_ptr: A pointer to a const node 常量节点指针

Static functions:

static node_ptr get_next(const_node_ptr n);

static void set_next(node_ptr n, node_ptr next);

linear_slist_algorithms public static functions

  1. static void init(node_ptr this_node) ;

    Effects: Constructs an non-used list element, putting the next pointer to null: NodeTraits::get_next(this_node) == 0
    作用:     构造一个未用的链表元素，将 next 指针置为空：NodeTraits::get_next(this_node) == 0。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。

  2. static bool unique(const_node_ptr this_node) ;

    Requires: this_node must be in a circular list or be an empty circular list.
    作用:     this_node 必须在一个链表中或为空链表。

    Effects: Returns true is "this_node" is the only node of a circular list: or it's a not inserted node: return !NodeTraitsget_next(this_node) || NodeTraits::get_next(this_node) == this_node
    作用:     如果 "this_node" 是链表中的唯一节点，则返回 true：return !NodeTraitsget_next(this_node) || NodeTraits::get_next(this_node) == this_node。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。     

  3. static bool inited(const_node_ptr this_node) ;

    Effects: Returns true is "this_node" has the same state as if it was inited using "init(node_ptr)"
    作用:     如果 "this_node" 为由 "init(node_ptr)" 函数初始化的未用状态，则返回 true。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。     

  4. static void unlink_after(node_ptr prev_node) ;

    Requires: prev_node must be in a circular list or be an empty circular list.
    要求:     prev_node 必须在一个链表中或为空链表。

    Effects: Unlinks the next node of prev_node from the circular list.
    作用:     在链表中断开 prev_node 的下一个节点的链接。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。     

  5. static void unlink_after(node_ptr prev_node, node_ptr last_node) ;

    Requires: prev_node and last_node must be in a circular list or be an empty circular list.
    要求:     prev_node 和 last_node 必须是一个链表中的节点或者是一个空链表。

    Effects: Unlinks the range (prev_node, last_node) from the linear list.
    作用:     在线性链表中断开区间 (prev_node, last_node) 的链接。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。     

  6. static void link_after(node_ptr prev_node, node_ptr this_node) ;

    Requires: prev_node must be a node of a linear list.
    要求:     prev_node 必须是一个线性链表中的节点。

    Effects: Links this_node after prev_node in the linear list.
    作用:     在线性链表中将 this_node 链接到 prev_node 之后。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。     

  7. static void transfer_after(node_ptr p, node_ptr b, node_ptr e) ;

    Requires: b and e must be nodes of the same linear list or an empty range. and p must be a node of a different linear list.
    要求:     b 和 e 必须是同线性链表的节点或为空区间，且 p 必须是另一个线性链表的节点。

    Effects: Removes the nodes from (b, e] range from their linear list and inserts them after p in p's linear list.
    作用: 从线性链表断开    节点 [b, e) 的链接并将它们插入到 p 的线性链表中，位于 p 之后。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。     

  8. static void init_header(node_ptr this_node) ;

    Effects: Constructs an empty list, making this_node the only node of the circular list: NodeTraits::get_next(this_node) == this_node.
    作用: 构造一个空链表，    this_node 为链表中的唯一节点：NodeTraits::get_next(this_node) == this_node。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。     

  9. static node_ptr 
    get_previous_node(node_ptr prev_init_node, node_ptr this_node) ;

    Requires: this_node and prev_init_node must be in the same linear list.
    要求:     this_node 和 prev_init_node 必须是同一个线性链表的节点。

    Effects: Returns the previous node of this_node in the linear list starting. the search from prev_init_node. The first node checked for equality is NodeTraits::get_next(prev_init_node).
    作用: 返回     this_node 在线性链表中的前一个节点，从 prev_init_node 开始搜索。对第一个节点检查等式 NodeTraits::get_next(prev_init_node).

    Complexity: Linear to the number of elements between prev_init_node and this_node.
    复杂度: 与     prev_init_node 和 this_node 之间的元素数量成线性。

    Throws: Nothing.
    抛出: 无。     

  10. static std::size_t count(const_node_ptr this_node) ;

    Requires: this_node must be in a linear list or be an empty linear list..
    要求:     this_node 必须是一个线性链表的节点或是一个空线性链表。

    Effects: Returns the number of nodes in a linear list. If the linear list is empty, returns 1.
    作用:     返回线性链表中的节点数量。如果线性链表为空，返回 1。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。     

  11. static void swap_trailing_nodes(node_ptr this_node, node_ptr other_node) ;

    Requires: this_node and other_node must be nodes inserted in linear lists or be empty linear lists.
    要求:     this_node 和 other_node 必须是一个线性链表的节点或是一个空线性链表。

    Effects: Moves all the nodes previously chained after this_node after other_node and vice-versa.
    作用:     将 this_node 之后的节点移到 other_node 之后，反之亦然。

    Complexity: Constant
    复杂度: 常量。

    Throws: Nothing.
    抛出: 无。     

  12. static node_ptr reverse(node_ptr p) ;

    Effects: Reverses the order of elements in the list.
    作用: 倒转将链表中元素的顺序。

    Returns: The new first node of the list.
    返回: 链表新的首节点。

    Throws: Nothing.
    抛出: 无。

    Complexity: This function is linear to the contained elements.
    复杂度: 该函数与所含    元素数量成线性。 

  13. static std::pair< node_ptr, node_ptr > 
    move_first_n_backwards(node_ptr p, std::size_t n) ;

    Effects: Moves the first n nodes starting at p to the end of the list.
    作用: 将节点     p 向链表尾部移动 n 个位置。

    Returns: A pair containing the new first and last node of the list or if there has been any movement, a null pair if n leads to no movement.
    返回:     如果有移动，则返回一个含有链表新的首尾节点的 pair，如果 n 引致没有移动则返回 null pair。

    Throws: Nothing.
    抛出: 无。

    Complexity: Linear to the number of elements plus the number moved positions.
    复杂度: 与元素数量加上移动的位置数成线性。     

  14. static std::pair< node_ptr, node_ptr > 
    move_first_n_forward(node_ptr p, std::size_t n) ;

    Effects: Moves the first n nodes starting at p to the beginning of the list.
    作用: 将节点     p 向链表头部移动 n 个位置。

    Returns: A pair containing the new first and last node of the list or if there has been any movement, a null pair if n leads to no movement.
    返回:     如果有移动，则返回一个含有链表新的首尾节点的 pair，如果 n 引致没有移动则返回 null pair。

    Throws: Nothing.
    抛出: 无。

    Complexity: Linear to the number of elements plus the number moved positions.
    复杂度: 与元素数量加上移动的位置数成线性。
