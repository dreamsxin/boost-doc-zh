Function template function_equal（函数模板 function_equal）

boost::function_equal —

比较两个函数对象是否相等。

Synopsis（概要）

template<typename F, typename G> bool function_equal(const F& f, const G& g);

Description（说明）

返回： f == g。
抛出： 只有当 f == g 时才会抛出。
