类模板 make_variant_over

boost::make_variant_over —

提供一个 variant，其有界类型为给定类型序列的元素。

摘要

template<typename Sequence> 
class make_variant_over {
public:
  // types
  typedef variant< unspecified > type;
};

说明

type 的行为在各方面均等同于 variant< Sequence[0], Sequence[1], ... > (其中 Sequence[i] 表示 Sequence 的第i个元素)，除了一点，即类型的数量没有上限。

注：

  • Sequence 必须符合 MPL 的 序列 概念的要求。
  • 由于在某些编译器上的标准符合问题，make_variant_over 可能在你的编译器上不被支持。更多信息请见 BOOST_VARIANT_NO_TYPE_SEQUENCE_SUPPORT .
