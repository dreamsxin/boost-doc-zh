Function template from_ftime

boost::posix_time::from_ftime — 从一个已初始化的 FILETIME 结构创建一个 time 对象的函数。

Synopsis

// In header: <boost/date_time/posix_time/conversion.hpp>


template<typename TimeT, typename FileTimeT> 
  TimeT from_ftime(const FileTimeT & ft);

Description

这是一个从一个已初始化的 FILETIME 结构创建一个 time 对象的函数。FILETIME 结构以100纳秒为单位(0.0000001)保存时间。以微秒解析度进行构建时，将截断 FILETIME 的多余小数位。对于纳秒解析度则没有截断。

Notes:

FILETIME 是 Win32 API 的一部分，所以它不可移植到非windows平台。

该函数按 FILETIME 类型模板化，因此可用于原生的 FILETIME 以及特定的 boost::date_time::winapi::file_time 类型。
