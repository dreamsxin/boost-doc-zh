函数 operator!

boost::logic::operator! — 对一个tribool进行逻辑取否运算。 

概要

tribool operator!(tribool x);

说明

Returns:

tribool的逻辑取否, 依照以下表格：

!  
false true
true false
indeterminate indeterminate
Throws:

不会抛出异常。
