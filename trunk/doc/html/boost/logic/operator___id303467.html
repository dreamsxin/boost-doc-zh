函数模板 operator<<

boost::logic::operator<< — 将一个tribool的值写出到流。

Synopsis摘要

template<typename CharT, typename Traits> 
  std::basic_ostream< CharT, Traits > & 
  operator<<(std::basic_ostream< CharT, Traits > & out, tribool x);

Description描述

当 x 的值为 true 或 false 时，该函数等效于： 

 out << static_cast<bool>(x);

当 x 的值为 indeterminate 时，它输出整型值 2 (如果 (out.flags() & std::ios_base::boolalpha) == 0 的话) 或者 indeterminate 值的名字。indeterminate 值的名字来自于 indeterminate_name facet (如果在输出流的locale中有定义的话)，或者来自于 get_default_indeterminate_name 函数(如果在locale中没有定义或C++标准库的实现不支持locales)。

Returns:

out
