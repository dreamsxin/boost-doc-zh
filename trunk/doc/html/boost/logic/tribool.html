类 tribool

boost::logic::tribool — 一个三态布尔类型。

Synopsis摘要

class tribool {
public:
  // construct/copy/destruct
  tribool();
  tribool(bool);
  tribool(indeterminate_keyword_t);

  // public member functions
  operator safe_bool() const;

  enum boost::logic::tribool::value_t value;
};

Description描述

三态布尔值的取值范围为：true, false, 或 indeterminate.

tribool 构造/复制/析构

  1. tribool();

    构造一个新的三态布尔值，取值为 'false'.

    Throws: 不会抛出异常。 
  2. tribool(bool value);

    按给定的布尔值构造一个新的三态布尔值，取值可能是 true 或 false .

    Throws: 不会抛出异常。 
  3. tribool(indeterminate_keyword_t);

    构造一个新的三态布尔值，取值为"不确定"态。

    Throws: 不会抛出异常。

tribool 公有成员函数

  1. operator safe_bool() const;

    将三态布尔值用于一个布尔上下文中。只有当该三态布尔值确实为true时，该函数取值为true。

    Returns:

    返回true如果三态布尔值为 true, 否则返回false

    Throws: 不会抛出异常。
