函数 indeterminate

boost::logic::indeterminate — 既是"不确定"态的关键字，也是用于测试"不确定"态的函数。 

概要

bool indeterminate(tribool x, unspecified dummy = unspecified);

说明

indeterminate 函数有两个角色。第一个角色是作为一个单参函数，判断一个tribool值是否处于"不确定"态。第二个角色是作为一个关键字，用于表示"不确定"态(正如"true" 和 "false"用于表示 true 和 false 态一样)。如果你不喜欢"indeterminate"这个名字，宁可使用别的名字，请见宏 BOOST_TRIBOOL_THIRD_STATE .

Returns:

x.value == tribool::indeterminate_value

Throws: 不会抛出异常。
