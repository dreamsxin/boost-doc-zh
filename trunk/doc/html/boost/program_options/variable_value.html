class variable_value {

说明

持有选项值的类。包含了关于如何设置该值的细节，可以方便地获得该值。

  1. variable_value();
  2. variable_value(const boost::any & v, bool defaulted);

variable_value 公有成员函数

  1. template<typename T> const T & as() const;

    如果保存的值为类型 T，则返回该值。否则抛出 boost::bad_any_cast exception。 

  2. template<typename T> T & as() ;
  3. bool empty() const;
  4. bool defaulted() const;

    如果该值不是显式给出的而是有缺省值的，则返回 true。

    返回所包含的值。
