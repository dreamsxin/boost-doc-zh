boost::program_options::untyped_value

摘要

class untyped_value : public boost::program_options::boost::program_options::value_semantic_codecvt_helper< char >
{
public:
  // 构造/复制/析构
  untyped_value(bool = false);

  // 公有成员函数
  std::string name() const;
  unsigned min_tokens() const;
  unsigned max_tokens() const;
  bool is_composing() const;

  bool apply_default(boost::any &) const;

说明

用于指定一个值的简单处理的类：该值具有字符串类型且只允许一个 token。

untyped_value 公有构造/复制/析构

  1. untyped_value(bool zero_tokens = false);

untyped_value 公有成员函数public member functions

  1. std::string name() const;

    返回选项的名字。名字仅对于自动帮助信息有意义。 

  2. unsigned min_tokens() const;

    该选项在命令行中出现的最少 tokens 数。

  3. unsigned max_tokens() const;

    该选项在命令行中出现的最多 tokens 数。

  4. bool is_composing() const;

    如果来自于不同源的选项值可以组合，则返回 true。否则使用来自第一个源的选项值而忽略其它来源。

  5. void xparse(boost::any & value_store, 
                const std::vector< std::string > & new_tokens) const;

    如果 'value_store' 已初始化，或者 new_tokens 超过一个元素，则抛出。否则，将 'new_tokens' 的第一个字符串赋给 'value_store'，不作任何修改。 

  6. bool apply_default(boost::any &) const;

    无动作。 

  7. void notify(const boost::any &) const;

    无动作。
