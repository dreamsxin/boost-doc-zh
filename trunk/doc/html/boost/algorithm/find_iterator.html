说明

查找迭代器封装一个 Finder 并允许递增地在一个 string 中搜索。每一次递增将迭代器移动到下一个匹配项。

查找迭代器是一个可读的前向遍历迭代器。

解引用这个迭代器产生一个限定当前匹配项的 iterator_range。

find_iterator 公有 构造/拷贝/析构

  1. find_iterator();

    构造空迭代器。所有的空迭代器都相等。

    后置条件：

    eof()==true 

  2. find_iterator(const find_iterator & Other);

    构造 find_iterator 的一个拷贝

  3. template<typename FinderT> 
  find_iterator(IteratorT Begin, IteratorT End, FinderT Finder);

    为一个给定的 finder 和一个给定的 range 构造新的 find_iterator。

  4. template<typename FinderT, typename RangeT> 
  find_iterator(RangeT & Col, FinderT Finder);

    为一个给定的 finder 和一个给定的 range 构造新的 find_iterator。

find_iterator 公有成员函数

  1. bool eof() const;

    检查 eof 条件。eof 条件就是没有更多的东西被搜索到，也就是说，find_iterator 在最后一个匹配项之后。
