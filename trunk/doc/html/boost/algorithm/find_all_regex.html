函数模板 find_all_regex

boost::algorithm::find_all_regex — Find all regex 算法

概述

template<typename SequenceSequenceT, typename RangeT, typename CharT, 
         typename RegexTraitsT> 
  SequenceSequenceT & 
  find_all_regex(SequenceSequenceT & Result, const RangeT & Input, 
                 const basic_regex< CharT, RegexTraitsT > & Rx, 
                 match_flag_type Flags = match_default);

说明

这个算法在输入中查找与给定 regex 匹配的所有 substring。

每一个部分被拷贝并作为一个新元素加入到输出容器中。因此结果容器必须能够持有匹配项（类似于 std::string 这样的兼容结构）的拷贝或引用（例如，使用迭代器范围类）。这样的容器的例子包括 std::vector<std::string> 或 std::list<boost::iterator_range<std::string::iterator>>。

参数：

Flags

regex 选项

Input

一个将要被搜索的容器。

Result

一个能够持有 substrings 的引用的拷贝的容器。

Rx

一个正则表达式

返回：

一个结果的引用

注意：

结果中的先前内容将被覆盖。
