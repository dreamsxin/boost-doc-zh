函数模板 find

boost::algorithm::find — Generic find 算法

概述

template<typename RangeT, typename FinderT> 
  iterator_range< typename range_iterator< RangeT >::type > 
  find(RangeT & Input, const FinderT & Finder);

说明

使用给定的 finder 搜索输入。

参数：

Finder

用于搜索的 Finder 对象。

Input

将要被搜索的一个 string。

返回：

一个限定匹配项的 iterator_range。返回的迭代器是 RangeT::iterator 或 RangeT::const_iterator，这取决于输入参数的常量性。
