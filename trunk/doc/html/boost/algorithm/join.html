函数模板 join

boost::algorithm::join — Join 算法

概述

template<typename SequenceSequenceT, typename Range1T> 
  range_value< SequenceSequenceT >::type 
  join(const SequenceSequenceT & Input, const Range1T & Separator);

说明

这个算法将一个 'list' 中的全部 strings 结合成一个长 string。片段之间用给定的分隔符连接。

参数：

Input

一个持有输入 strings 的容器。它必须是一个容器的容器。

Separator

一个用于分割被连接片段的 string。

返回：

已连接好的 string。

注意：

这个函数提供强异常安全保证
