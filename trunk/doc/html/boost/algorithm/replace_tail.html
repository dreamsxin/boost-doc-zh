函数模板 replace_tail

boost::algorithm::replace_tail — Replace tail 算法

概述

template<typename SequenceT, typename RangeT> 
  void replace_tail(SequenceT & Input, int N, const RangeT & Format);

说明

用给定的格式化序列替换输入的末尾。末尾是指 string 的一个给定大小的后缀。如果这个序列比需要的短，则整个 string 都被当做末尾。输入序列在原地被改变。

参数：

Format

一个用于替换的 string

Input

一个输入 string

N

末尾的长度。如果 N>=0，最多 N 个 characters 被选取。如果 N<0，size(Input)-|N| 个 characters 被选取。
