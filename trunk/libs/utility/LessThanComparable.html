LessThanComparable 可小于比较

描述

一个类型是 LessThanComparable 的，如果它是有序的：必须可以用 operator< 对该类型的两个对象进行比较，并且 operator< 必须是一个严格弱序关系。

精化自

相关类型

符号

X 符合 LessThanComparable 的类型
x, y, z 类型 X 的对象

定义

考虑关系 !(x < y) && !(y < x). 如果该关系是可传递的(即，如果 !(x < y) && !(y < x) && !(y < z) && !(z < y) 则 !(x < z) && !(z < x))，那么它就满足等价关系的数学定义。这种情况下，operator< 就是一个严格弱序。

如果 operator< 是一个严格弱序，并且如果每个等价类只有一个单个元素，则 operator< 是一个全序。

复杂度保证

不变式

自反性 x < x 必须为 false
反对称性 x < y 则 !(y < x) [2]
传递性 x < y 且 y < z 则 x < z [3]

模型

  • int

注

[1] 只有 operator< 是基本的；其它不等操作符都是语法糖。

[2] 反对称性是一个定理，而不是公理：它来自于自反性和传递性。

[3] 由于自反性和传递性，operator< 总是满足 偏序 的定义。严格弱序 的定义稍为严格，而 全序 的定义则更为严格。
