概要： 迭代器概念检查类为模板提供了一个机制，当用户以一个不满足模板要求的类型去对模板进行实例化时，可以得到更清晰的错误信息。

有关使用概念检查类的介绍，请看 boost::concept_check 库的文档。

参考

迭代器访问概念

  • Readable Iterator可读迭代器
  • Writable Iterator可写迭代器
  • Swappable Iterator可交换迭代器
  • Lvalue Iterator左值迭代器

迭代器遍历概念

  • Incrementable Iterator可递增迭代器
  • Single Pass Iterator单遍迭代器
  • Forward Traversal前向遍历
  • Bidirectional Traversal双向遍历
  • Random Access Traversal随机访问遍历

iterator_concepts.hpp 概要

namespace boost_concepts {

    // 迭代器访问概念

    template <typename Iterator>
    class ReadableIteratorConcept;

    template <
        typename Iterator
      , typename ValueType = std::iterator_traits<Iterator>::value_type
    >
    class WritableIteratorConcept;

    template <typename Iterator>
    class SwappableIteratorConcept;

    template <typename Iterator>
    class LvalueIteratorConcept;

    // 迭代器遍历概念

    template <typename Iterator>
    class IncrementableIteratorConcept;

    template <typename Iterator>
    class SinglePassIteratorConcept;

    template <typename Iterator>
    class ForwardTraversalConcept;

    template <typename Iterator>
    class BidirectionalTraversalConcept;

    template <typename Iterator>
    class RandomAccessTraversalConcept;

    // 可交互性

    template <typename Iterator, typename ConstIterator>
    class InteroperableIteratorConcept;

}
