简介

Boost.Range 的 MFC/ATL 扩展提供了对 MFC/ATL 集合和字符串类型的 Boost.Range 支持。

CTypedPtrArray<CPtrArray, CList<CString> *> myArray;
...
BOOST_FOREACH (CList<CString> *theList, myArray)
{
    BOOST_FOREACH (CString& str, *theList)
    {
        boost::to_upper(str);
        std::sort(boost::begin(str), boost::end(str));
        ...
    }
}

要求

  • Boost C++ Libraries Version 1.34.0 或以后版本(无需编译)
  • Visual C++ 7.1 或 Visual C++ 8.0

MFC 区间

如果 <boost/range/mfc.hpp> 在 Boost.Range 头文件之前或之后被包含，则 MFC 的集合和字符串类型就成为了 Range 的 Model. 下表列出了 MFC 区间的遍历类别和 range_reference .

区间 遍历类别 range_reference<Range>::type
CArray<T,A> 随机访问 T&
CList<T,A> 双向 T&
CMap<K,AK,M,AM> 前向 Range::CPair&
CTypedPtrArray<B,T*> 随机访问 T* const
CTypedPtrList<B,T*> 双向 T* const
CTypedPtrMap<B,T*,V*> 前向 std::pair<T*,V*> const
CByteArray 随机访问 BYTE&
CDWordArray 随机访问 DWORD&
CObArray 随机访问 CObject* &
CPtrArray 随机访问 void* &
CStringArray 随机访问 CString&
CUIntArray 随机访问 UINT&
CWordArray 随机访问 WORD&
CObList 双向 CObject* &
CPtrList 双向 void* &
CStringList 双向 CString&
CMapPtrToWord 前向 std::pair<void*,WORD> const
CMapPtrToPtr 前向 std::pair<void*,void*> const
CMapStringToOb 前向 std::pair<String,CObject*> const
CMapStringToString 前向 Range::CPair&
CMapWordToOb 前向 std::pair<WORD,CObject*> const
CMapWordToPtr 前向 std::pair<WORD,void*> const

其它 Boost.Range 元函数定义如下。其中 Range 为以上所列的任一类型，ReF 与 range_reference<Range>::type 相同。 range_value<Range>::type 与 remove_reference<remove_const<Ref>::type>::type 相同， range_difference<Range>::type 与 std::ptrdiff_t 相同，而 range_pointer<Range>::type 与 add_pointer<remove_reference<Ref>::type>::type 相同。对于 const Range，请见 const 区间。

其它 Boost.Range 元函数定义如下。其中 Range 为以上所列的任一类型，ReF 与 range_reference<Range>::type 相同。 range_value<Range>::type 与 remove_reference<remove_const<Ref>::type>::type 相同， range_difference<Range>::type 与 std::ptrdiff_t 相同，而 range_pointer<Range>::type 与 add_pointer<remove_reference<Ref>::type>::type 相同。对于 const Range，请见 const 区间。

const 区间

range_reference<const Range>::type 由以下算法定义。其中 Range 为以上所列的任一类型，ReF 与 range_reference<Range>::type 相同。

if (Range is CObArray || Range is CObList)
    return CObject const * &
else if (Range is CPtrArray || Range is CPtrList)
    return void const * &
else if (there is a type X such that X& is the same as ReF)
    return X const &
else if (there is a type X such that X* const is the same as ReF)
    return X const * const
else
    return ReF

其它 Boost.Range 元函数定义如下。range_value<Range>::type 与 range_value<Range>::type 相同， range_difference<Range>::type 与 std::ptrdiff_t 相同，而 range_pointer<Range>::type 与 add_pointer<remove_reference<range_reference<const Range>::type>::type>::type 相同。
