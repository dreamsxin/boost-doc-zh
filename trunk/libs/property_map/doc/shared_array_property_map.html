shared_array_property_map<ValueType, OffsetMap>

这个属性映射是一个适配器，它包含一个 boost::shared_array 并使用该数组来保存属性映射的数据。所得的属性映射满足 左值属性映射。类型 OffsetMap 负责将键对象转换为可以在数组中用作偏移量的整数。

定义于

boost/property_map/shared_array_property_map.hpp

Model Of

左值属性映射

模板参数

参数描述缺省值
ValueType 属性映射的值类型。  
OffsetMap 必须满足 可读属性映射 且其值类型必须可转换为 std::size_t.
