Boost Flyweight Library


Tutorial

Flyweights are small-sized handle classes granting constant access to shared common data, thus allowing for the management of large amounts of entities within reasonable memory limits. Boost.Flyweight makes it easy to use this common programming idiom by providing the class template flyweight<T>, which acts as a drop-in replacement for const T.

Flyweights是一些小型的句柄类授权对公共共享数据的常量访问，于是可以在有限的内存里管理大量的实体。 Boost.Flyweight提供的类模板flyweight<T>让这种常见的编程范式易于使用，这个类模板起着const T非正式替代品的作用。 

Learning to use Boost.Flyweight can be accomplished in a matter of minutes. When special needs arise, however, an extensive customization interface is provided which allows the user to control and extend the following aspects:

学习使用Boost.Flyweight只需要花几分钟的时间。然而，当出现特殊的需求时，有一个可以广泛定制的接口允许用户对以下几个方面进行控制和扩展：

  • Type tagging for separation of domains using the same basic flyweight types.  采用类标注来区分使用了相同flyweight类型的域。
  • Specification and parameterization of the so-called flyweight factory.  所谓的flyweight工厂的专门化和参数化
  • Control of the factory instantiation procedure.  控制工厂实例化的过程
  • Specification of the internal synchronization mechanisms.  内部同步机制的专门化
  • Flyweight tracking, allowing for the disposal of stored values when no longer referenced by any flyweight object.  Flyweight跟踪，当存储的值不再被任何flyweight对象引用的时候释放它们。
