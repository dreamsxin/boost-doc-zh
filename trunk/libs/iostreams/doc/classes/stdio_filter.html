描述

类模板 basic_stdio_filter 是一个 DualUseFilter，可用作从标准输入读入未过滤数据并将过滤后数据写出到标准输出的过滤器的基类。

basic_stdio_filter 通过重定向 std::cin 和 std::cout (或者是 std::wcin 和 std::wcout)来实现。在调用 do_filter 之前，整个未过滤的数据流被读入并保存在内存中。所以，basic_stdio_filter 不适用于小内存的环境，也不适于处理没有自然结束的字符序列，如 stock tickers. 由于这些限制，basic_stdio_filter 最好用于学习使用 Iostreams 库。

头文件

<boost/iostreams/filter/stdio.hpp>

参考

摘要

namespace boost { namespace iostreams {

template<typename Ch>
class basic_stdio_filter {
public:
    typedef Ch                        char_type;
    typedef [implementation-defined]  category;
protected:
    basic_stdio_filter();
public:
    virtual ~basic_stdio_filter();
private:
    virtual void do_filter() = 0;
};

typedef basic_stdio_filter<char>     stdio_filter;
typedef basic_stdio_filter<wchar_t>  wstdio_filter;

} } // End namespace boost::io

模板参数

Ch -  字符类型

从标准输入读入并写出到标准输出，直至标准输入被耗尽。如果 Ch 是 char 则使用 std::cin 和 std::cout; 如果 Ch 是 wchar_t 则使用 std::wcin 和 std::wcout.
