描述

类模板 symmetric_filter 一个工具，可以从 SymmetricFilter 的 model 生成 DualUseFilter 的 model. 它可用于定义基于C语言API的过滤器。

Iostreams 库使用 symmetric_filter 来实现 Zlib 和 Bzip2 过滤器。

头文件

<boost/iostreams/filter/symmetric.hpp>

参考

摘要

namespace boost { namespace iostreams {

template< typename SymmetricFilter,
          typename Alloc = std::allocator<...> >
class symmetric_filter {
public:
    typedef typename char_type_of<SymmetricFilter>::type  char_type;
    typedef implementation-defined                        category;

    explicit symmetric_filter(int buffer_size);

    template<typename T0>
    symmetric_filter(int buffer_size, const T0& t0);

    template<typename T0, typename T1>
    symmetric_filter(int buffer_size, const T0& t0, const T1& t1);

    template<typename T0, ..., typename TN>
    symmetric_filter( int buffer_size, const T0& t0,
                              const T1& t1, ..., const TN& tN );

    ...
protected:
    SymmetricFilter& filter();
    std::basic_string<char_type, Alloc> unconsumed_input() const;
};

} } // End namespace boost::io

模板参数

SymmetricFilter - SymmetricFilter 的 model
Alloc - 一个标准库的分配器类型 ([ISO], 20.1.5), 用于分配字符缓冲区

symmetric_filter::symmetric_filter

    explicit symmetric_filter(int buffer_size);

基于模板参数 SymmetricFilter 的一个缺省构造的实例构造一个 symmetric_filter 实例，其内部缓冲区大小为 buffer_size.

symmetric_filter::symmetric_filter

    template<typename T0>
    symmetric_filter(int buffer_size, const T0& t0);

    template<typename T0, typename T1>
    symmetric_filter(int buffer_size, const T0& t0, const T1& t1);

    template<typename T0, ..., typename TN>
    symmetric_filter( int buffer_size, const T0& t0,
                              const T1& t1, ..., const TN& tN );

以上每个成员基于模板参数 SymmetricFilter 的一个按给定参数序列构造的实例，构造一个 symmetric_filter 实例，其内部缓冲区大小为 buffer_size.

symmetric_filter::filter

    SymmetricFilter& filter();

返回底层 SymmetricFilter 实例的一个引用。

symmetric_filter::unconsumed_input

    std::basic_string<char_type, Alloc> unconsumed_input() const;

以标准字符串的方式返回内部缓冲区中剩余的未消耗数据。该函数在以下时间内有效：从到达流结束时开始 — 即从 SymmetricFilter::filter 返回 false 时开始 — 直至 filter 被调用开始处理新的数据序列。
