简介

函数模板 putback 为将字符退回一个 Peekable 的 model 提供了统一接口，在 Filters 的定义中使用。

头文件

<boost/iostreams/operations.hpp>
<boost/iostreams/putback.hpp>

参考

描述

尝试退回一个字符到模板参数 Peekable 的一个给定实例中，成功则返回 true .

摘要

namespace boost { namespace iostreams {
              
    template<typename Peekable>     
    bool putback(Peekable& peekable, typename char_type_of<Peekable>::type c);

} } // End namespace boost::io

模板参数

Peekable - Source 的 model

语义

putback 的语义取决于 Peekable 的 category 如下：

category_of<Peekable>::type 语义
可转换为 istream_tag 调用 peekable.rdbuf()->putback(c) 并返回 true 如果操作成功
可转换为 streambuf_tag 但不可转换为 istream_tag 调用 peekable.sputbackc(c) 并返回 true 如果操作成功
可转换为 peekable_tag 但不可转换为 streambuf_tag 或 istream_tag 返回 peekable.putback(c)
否则 编译期错误
