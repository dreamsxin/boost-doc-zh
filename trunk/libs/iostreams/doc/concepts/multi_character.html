Mutli-Character Filter多字符过滤器

描述

指可以对其控制的序列一次访问多个字符的 Filter. 多字符过滤器与非多字符过滤器的区别反映在各个过滤器强化物的规格说明中。请见 InputFilter 和 OutputFilter.

强化自

Filter.

关联类型

与 Filter 一致，有以下额外要求：

Category 一个类型，可转换为 filter_tag 和 multichar_tag

有效表达式 / 语义

与 Filter 一致。

模型

  • 压缩与解压过滤器
  • basic_regex_filter基本正则表达式过滤器
  • aggregate_filter聚集过滤器
  • symmetric_filter对称过滤器
