Boost Exception

get_error_info

namespace
boost
    {
    template <class ErrorInfo,class E>
    typename ErrorInfo::value_type const * get_error_info( E const & x );
    }

要求:

  • ErrorInfo 必须是模板 error_info 的实例.
  • E 必须是多态类型的.

返回值:

  • 如果 dynamic_cast<boost::exception const *>(&x) 等于 0, 或者 x 没有存储任何 ErrorInfo 对象数据, 返回值是 null.
  • 否则, 返回的指针指向通过操作符 operator<< 存储在异常对象中的数值.当 x 析构的时候, 通过 get_error_info 得到的指针变为无效.

异常抛出:

没有异常.

注意:

函数 get_error_info 的接口会受宏 configuration macros设置的影响.
