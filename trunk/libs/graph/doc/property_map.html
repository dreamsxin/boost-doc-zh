property_map<PropertyGraph, PropertyTag>

这是一个 traits 类，对某个特定图类型生成指定 属性映射property map 对象的类型。属性由 PropertyTag 模板参数指定。Graph 图必须指定这一 traits 类，以提供它们自己的属性映射实现。 
  template <class Graph, class PropertyTag>
  struct property_map {
    typedef ... type;
    typedef ... const_type;
  };

Where Defined 定义于

boost/graph/properties.hpp

Associated Types 关联类型

Map Type type 用于将图的顶点描述符或边描述符映射至它们相关的属性值的(可变)属性映射类型，属性由 PropertyTag 指定。该类型是一个 属性映射。
Const Map Type const_type 用于将图的顶点描述符或边描述符映射至它们相关的属性值的(不可变)属性映射类型，属性由 PropertyTag 指定。该类型是一个 属性映射。
