Boost Graph Library: Python 绑定 (实验中)

BGL提供了用于C++的大量图算法和数据类型。这些算法非常灵活也很高效，但是达到这一目标所采用的机制可能会导致较长的编译时间，从而令开发周期过慢。

Python绑定是用 Boost.Python 库构建的。该绑定的目的是在可用性、灵活性和效率间取得平衡，使我们能够在Python里用BGL快速开发有用的系统。

BGL的Python绑定目前是一个独立项目的组成部分。它们不再出现在Boost开发树中。
