Graph 图

图(Graph)的概念包含了一些对于所有图概念都适用的要求。其中包括一些关联类型 vertex_descriptor, edge_descriptor, 等等。应该注意，以 Graph 为模并不要求满足 可赋值Assignable，因此算法应该通过引用来传递图对象。

Notation 记号

G 以 Graph 为模的一个类型。
g 一个类型为 G 的对象。

Associated Types 关联类型

boost::graph_traits<G>::vertex_descriptor
顶点描述符，对应于一个抽象图实例中的单个顶点。顶点描述符必须符合 可缺省构造, 可赋值, 以及 可比较相等性。 
boost::graph_traits<G>::edge_descriptor
边描述符，对应于一个图中的一条边 (u,v)。边描述符必须符合 可缺省构造, 可赋值, 以及 可比较相等性。 
boost::graph_traits<G>::directed_category
这个类型应该可以转换为 directed_tag 或 undirected_tag。 
boost::graph_traits<G>::edge_parallel_category
用于指定该图类是否允许插入平行边(即具有相同源和目标的边)。两个标签分别为 allow_parallel_edge_tag 和 disallow_parallel_edge_tag。 
boost::graph_traits<G>::traversal_category
用于指定以何种方式可以访问图中的顶点和边。选择包括有 incidence_graph_tag, adjacency_graph_tag, bidirectional_graph_tag, vertex_list_graph_tag, edge_list_graph_tag, 和 adjacency_matrix_tag。

Valid Expressions 有效表达式

boost::graph_traits<G>::null_vertex()
 返回一个特殊的 boost::graph_traits<G>::vertex_descriptor 对象，该对象不引向任一个类型为 G 的图对象的顶点。

Concept Checking Class 概念检查类

  template <class G>
  struct GraphConcept
  {
    typedef typename boost::graph_traits<G>::vertex_descriptor vertex_descriptor;
    typedef typename boost::graph_traits<G>::edge_descriptor edge_descriptor;
    typedef typename boost::graph_traits<G>::directed_category directed_category;
    typedef typename boost::graph_traits<G>::edge_parallel_category edge_parallel_category;
    typedef typename boost::graph_traits<G>::traversal_category traversal_category;

    void constraints() {
      function_requires< DefaultConstructibleConcept<vertex_descriptor> >();
      function_requires< EqualityComparableConcept<vertex_descriptor> >();
      function_requires< AssignableConcept<vertex_descriptor> >();
      function_requires< DefaultConstructibleConcept<edge_descriptor> >();
      function_requires< EqualityComparableConcept<edge_descriptor> >();
      function_requires< AssignableConcept<edge_descriptor> >();
    }
    G g;
  };
