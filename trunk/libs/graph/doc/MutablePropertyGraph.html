MutablePropertyGraph 可变属性图

可变属性图(MutablePropertyGraph)是一个带有关联至顶点或边的属性的 可变图MutableGraph。在增加顶点或边时，可以指定属性值。

Refinement of 精化自

可变图MutableGraph 和 属性图PropertyGraph

Notation 记号

G 一个以 Graph 为模的类型。
g 一个类型为 G 的对象。
e 一个类型为 boost::graph_traits<G>::edge_descriptor 的对象。
u,v 均为类型 boost::graph_traits<G>::vertex_descriptor 的对象。
ep一个类型为 G::edge_property_type 的对象
vp一个类型为 G::vertex_property_type 的对象

Associated Types 关联类型

边属性类型 graph_traits<G>::edge_property_type
顶点属性类型 graph_traits<G>::vertex_property_type

Valid Expressions 有效表达式

add_edge(u, v, ep, g) 将边 (u,v) 插入到图中，并将对象 ep 复制为该边的属性。
返回类型：std::pair<edge_descriptor, bool>
add_vertex(vp, g) 增加一个新的顶点到图中，并复制 vp 至该新顶点的属性。返回新顶点的 vertex_descriptor。
返回类型：vertex_descriptor

Models 模型

Concept Checking Class 概念检查类

  template <class G>
  struct MutablePropertyGraphConcept
  {
    typedef typename boost::graph_traits<G>::edge_descriptor edge_descriptor;
    void constraints() {
      function_requires< MutableGraphConcept<G> >();
      v = add_vertex(vp, g);
      p = add_edge(u, v, ep, g);
    }
    G g;
    std::pair<edge_descriptor, bool> p;
    typename boost::graph_traits<G>::vertex_descriptor u, v;
    typename boost::graph_traits<G>::vertex_property_type vp;
    typename boost::graph_traits<G>::edge_property_type ep;
  };
