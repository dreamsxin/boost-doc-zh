Boost Basic Linear Algebra - 发布备注

回到 uBLAS 主页

版本 1.43.0

缺陷修复

  • [3968] 修复在MSVC10中的 coordinate_matrix 排序问题
  • [3539] 将复数类型的 norm_inf 计算修改至与数学定义相匹配。
    说明：这可能会引起性能下降，因为现在要为每个 vector 元素调用 std::abs(z)。旧的实现使用 std::max(std::abs(real(z)),std::abs(imag(z)). 另外，norm_inf 和 norm_1 现在对于复数向量将返回相同的值。
  • [3501] 将 concepts.hpp 中的自由函数移至匿名名字空间。

版本 1.41.1

新特性

  • 实现了 vector/matrix 容器赋值操作的转移语义。可通过设置 BOOST_UBLAS_MOVE_SEMANTICS 来激活。更多细节在 预处理选项一页。
  • 引入新的自由函数。见 [3449], 新的测试程序在 libs/numeric/ublas/test 中，这些文件的在线文档在 boost/numeric/ublas/operation/ 中。

缺陷修复

  • [3293] 修复 identity_matrix 中的重整大小问题
  • [3499] 在概念检查中增加 DefaultConstructible

版本 1.40.0 及之前

  • 在此格式中无发布备注。
