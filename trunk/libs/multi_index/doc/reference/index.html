头文件依赖性

Boost.MultiIndex的头文件有以下依赖性：

  • "boost/multi_index_container.hpp" 包含
    • "boost/multi_index/indexed_by.hpp".
  • "boost/multi_index/ordered_index.hpp" 包含
    • "boost/multi_index/tag.hpp".
  • "boost/multi_index/hashed_index.hpp" 包含
    • "boost/multi_index/tag.hpp".
  • "boost/multi_index/sequenced_index.hpp" 包含
    • "boost/multi_index/tag.hpp".
  • "boost/multi_index/random_access_index.hpp" 包含
    • "boost/multi_index/tag.hpp".
  • "boost/multi_index/key_extractors.hpp" 包含
    • "boost/multi_index/identity.hpp",
    • "boost/multi_index/member.hpp",
    • "boost/multi_index/mem_fun.hpp" 
    • "boost/multi_index/global_fun.hpp" 和
    • "boost/multi_index/composite_key.hpp".
所以，使用Boost.MultiIndex的程序必须包含 "boost/multi_index_container.hpp"、以及定义相关索引类型和所使用的一个或多个键提取器的头文件。注意，包含 "boost/multi_index/key_extractors.hpp" 将自动获得 Boost.MultiIndex 所提供的所有键提取器。

为了使用 Boost.MultiIndex 的序列化特性，必须链接适当的 Boost.Serialization 库模块。除此之外，Boost.MultiIndex 仅是一个头文件库，不需要任何其它的对象模块。
