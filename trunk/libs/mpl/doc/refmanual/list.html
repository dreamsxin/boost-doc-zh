list

描述

list 是一个 variadic, 前向, 可扩展 的类型序列，它支持在序列头部的常量时间的元素插入和删除操作，以及在序列中部及尾部的线性时间的元素插入和删除操作。

头文件

序列形式 头文件
Variadic #include <boost/mpl/list.hpp>
Numbered #include <boost/mpl/list/listn.hpp>

Model of

  • Variadic 序列
  • 前向序列
  • 可扩展序列
  • 前端可扩展序列

表达式语义

在下表中，l 是一个 list, pos 和 last 是 l 中的迭代器，r 是一个 前向序列，而 t1,t2,... tn 和 x 是任意类型。

表达式 语义 
list<t1,t2,... tn>
listn<t1,t2,... tn>
 list 元素为 t1,t2,... tn; 请见 Variadic 序列 
list<t1,t2,... tn>::type
listn<t1,t2,... tn>::type
 等同于 listn<t1,t2,... tn>; 请见 Variadic 序列
begin<l>::type 指向 l 头部的迭代器；请见 前向序列
end<l>::type 指向 l 尾部的迭代器；请见 前向序列
size<l>::type l 的大小；请见 前向序列
empty<l>::type 布尔 整型常量 c 满足 c::value == true 当且仅当 l 为空；请见 前向序列
front<l>::type l 中的第一个元素；请见 前向序列
insert<l,pos,x>::type 一个新的 list，具有以下元素：[begin<l>::type, pos), x, [pos, end<l>::type)；请见 可扩展序列
insert_range<l,pos,r>::type 一个新的 list，具有以下元素：[begin<l>::type, pos), [begin<r>::type, end<r>::type) [pos, end<l>::type)；请见 可扩展序列
erase<l,pos>::type 一个新的 list，具有以下元素：[begin<l>::type, pos), [next<pos>::type, end<l>::type)；请见 可扩展序列
erase<l,pos,last>::type 一个新的 list，具有以下元素：[begin<l>::type, pos), [last, end<l>::type)；请见 可扩展序列
clear<l>::type 一个空的 list；请见 可扩展序列
push_front<l,x>::type 一个新的 list，包含 x 为第一个元素；请见 前向可扩展序列
pop_front<l>::type 一个新的 list，包含 l 中除第一个元素以外的所有元素且顺序相同；请见 前向可扩展序列

例子

typedef list<float,double,long double> floats;
typedef push_front<floating_types,int>::type types;

BOOST_MPL_ASSERT(( is_same< front<types>::type, int > ));
