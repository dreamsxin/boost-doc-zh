BOOST_MPL_HAS_XXX_TEMPLATE_NAMED_DEF

摘要

#define BOOST_MPL_HAS_XXX_TEMPLATE_NAMED_DEF(trait, name, default_) \
    unspecified token sequence \
/**/

描述

扩展为一个布尔 Metafunction trait 的定义，使得对于任意类型 x trait<x>::value == true 当且仅当 x 是一个类类型且具有一个内嵌模板成员 x::template name，它带有不超过 BOOST_MPL_LIMIT_METAFUNCTION_ARITY 个参数。

在那些有问题的，不能执行这项检测的编译器上，trait<x>::value 总是返回缺省值 default_. 系统提供了一个布尔配置宏 BOOST_MPL_CFG_NO_HAS_XXX_TEMPLATE, 用于表明某个特定的编译器是否存在这一缺陷。[注: 缺省值也可以在调用元函数时给出；有关细节请见 表达式语义 一节 — 注完]

头文件

#include <boost/mpl/has_xxx.hpp>

参数

参数 要求 描述
trait 一个合法的标识符 被生成的元函数的名字
name 一个合法的标识符 要检测的成员名字
default_ 布尔常量 在有问题的编译器上使用的返回值

表达式语义

对于任意的合法 C++ 标识符 trait 和 name, 布尔常量表达式 c1, 布尔 Integral Constant c2, 以及任意类型 x:

BOOST_MPL_HAS_XXX_TEMPLATE_NAMED_DEF(trait, name, c1)
前置条件:

位于名字空间域
返回类型:

无
语义:

扩展为如下类模板定义的等价物

template<
    typename X
  , typename fallback = boost::mpl::bool_<c1>
>
struct trait
{
    // unspecified
    // ...
};

其中 trait 是一个布尔 Metafunction，具有以下语义：

typedef trait<x>::type r;
返回类型:Integral Constant.
语义:If BOOST_MPL_CFG_NO_HAS_XXX_TEMPLATE 被定义，r::value == c1; 否则 r::value == true 当且仅当 x 是一个类类型且具有一个内嵌模板成员 x::template name，其参数不超过 BOOST_MPL_LIMIT_METAFUNCTION_ARITY. 
typedef trait< x, c2 >::type r;
返回类型:

Integral Constant.
语义:

如果 BOOST_MPL_CFG_NO_HAS_XXX_TEMPLATE 被定义，r::value == c2::value; 否则等价于

typedef trait<x>::type r;

例子

BOOST_MPL_HAS_XXX_TEMPLATE_NAMED_DEF(
    has_xxx, xxx, false
)

struct test1  {};
struct test2  { void xxx(); };
struct test3  { int xxx; };
struct test4  { static int xxx(); };
struct test5  { typedef int xxx; };
struct test6  { struct xxx; };
struct test7  { typedef void (*xxx)(); };
struct test8  { typedef void (xxx)(); };
struct test9  { template< class T > struct xxx {}; };

BOOST_MPL_ASSERT_NOT(( has_xxx<test1> ));
BOOST_MPL_ASSERT_NOT(( has_xxx<test2> ));
BOOST_MPL_ASSERT_NOT(( has_xxx<test3> ));
BOOST_MPL_ASSERT_NOT(( has_xxx<test4> ));
BOOST_MPL_ASSERT_NOT(( has_xxx<test5> ));
BOOST_MPL_ASSERT_NOT(( has_xxx<test6> ));
BOOST_MPL_ASSERT_NOT(( has_xxx<test7> ));
BOOST_MPL_ASSERT_NOT(( has_xxx<test8> ));

#if !defined(BOOST_MPL_CFG_NO_HAS_XXX_TEMPLATE)
BOOST_MPL_ASSERT(( has_xxx<test9> ));
#endif

BOOST_MPL_ASSERT(( has_xxx<test9, true_> ));
