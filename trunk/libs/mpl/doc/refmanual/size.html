size

摘要

template<
      typename Sequence
    >
struct size
{
    typedef unspecified type;
};

描述

size 返回序列中的元素数量，即区间 [begin<Sequence>::type, end<Sequence>::type) 中的元素数量。

头文件

#include <boost/mpl/size.hpp>

Model of

Tag 分派元函数

参数

参数 要求 描述
Sequence 前向序列 被查询的序列

表达式语义

对于任意 前向序列 s:

typedef size<s>::type n;
返回类型: 整型常量
语义:

等价于

typedef distance< begin<s>::type,end<s>::type >::type n;
后置条件: n::value >= 0.

复杂度

size 元函数的复杂度取决于所应用的特定序列的实现。最坏情况下，size 保证线性复杂度。

如果 s 是一个 随机访问序列, size<s>::type 是一个 O(1) 操作。反之则不一定为真 — 例如，一个符合 前向序列 的序列类也可以给出一个 O(1) 的 size 实现。

例子

typedef list0<> empty_list;
typedef vector_c<int,0,1,2,3,4,5> numbers;
typedef range_c<int,0,100> more_numbers;

BOOST_MPL_ASSERT_RELATION( size<list>::value, ==, 0 );
BOOST_MPL_ASSERT_RELATION( size<numbers>::value, ==, 5 );
BOOST_MPL_ASSERT_RELATION( size<more_numbers>::value, ==, 100 );

参见

前向序列, 随机访问序列, empty, begin, end, distance
