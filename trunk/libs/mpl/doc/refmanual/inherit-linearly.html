inherit_linearly

摘要

template<
      typename Types
    , typename Node
    , typename Root = empty_base
    >
struct inherit_linearly
    : fold<Types,Root,Node>
{
};

描述

fold 的一个便于使用的包装器，在基于序列的类合成的上下文中使用。持续调用二元的 Node 函数，以上一次 Node 调用的结果(第一次调用时使用 Root) 以及 前向序列 Types 中的类型为参数。

头文件

#include <boost/mpl/inherit_linearly.hpp>

Model of

元函数

参数

参数 要求 描述
Types 前向序列 被继承的类型
Node 二元 Lambda 表达式 派生元函数
Root 类类型 位于类继承层次根部的类型

表达式语义

对于任意 前向序列 types, 二元 Lambda 表达式 node, 以及任意类类型 root:

typedef inherit_linearly<types,node,root>::type r;
返回类型: 一个类类型
语义:

等价于

typedef fold<types,root,node>::type r;

复杂度

线性。刚好 size<types>::value 次对 node 的调用。

例子

template< typename T > struct tuple_field
{
    T field;
};

template< typename T >
inline
T& field(tuple_field<T>& t)
{
    return t.field;
}

typedef inherit_linearly<
      vector<int,char const*,bool>
    , inherit< _1, tuple_field<_2> >
    >::type tuple;


int main()
{
    tuple t;
    
    field<int>(t) = -1;
    field<char const*>(t) = "text";
    field<bool>(t) = false;

    std::cout
        << field<int>(t) << 'n'
        << field<char const*>(t) << 'n'
        << field<bool>(t) << 'n'
        ;
}
