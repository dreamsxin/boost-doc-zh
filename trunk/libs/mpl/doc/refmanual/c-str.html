c_str

摘要

template<
      typename Sequence
    >
struct c_str
{
    typedef unspecified type;
    static char const value[];
};

描述

c_str 将 整型常量 的 前向序列 Sequence 转换为一个包含相同序列的以null-结尾的字节串。

头文件

#include <boost/mpl/string.hpp>

Model of

元函数

参数

参数 要求 描述
Sequence 整型常量 的 前向序列 要转换为以null-结尾的字节串的序列。

表达式语义

对于任意的 整型常量 的 前向序列 s,

c_str<s>::value;
返回类型：

一个以null-结尾的字节串。
先验条件：

size<s>::value <= BOOST_MPL_STRING_MAX_LENGTH.
语义：

等同于

char const value[] = {
    at<s, 0>::type::value
  , ...
  , at<s, size<s>::value-1>::type::value
  , '\0'
};

复杂度

序列原型 复杂度
前向序列 线性。

示例

typedef vector_c<char,'h','e','l','l','o'> hello;
assert( 0 == std::strcmp( c_str<hello>::value, "hello" ) );

参见

前向序列, 整型常量, 字符串
