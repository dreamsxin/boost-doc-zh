transform_view

摘要

template<
      typename Sequence
    , typename F
    >
struct transform_view
{
    // unspecified
    // ...
};

描述

一个视图，由 Sequence 的所有元素转化而为。

头文件

#include <boost/mpl/transform_view.hpp>

Model of

  • 前向序列

参数

参数 要求 描述
Sequence 前向序列 包装的序列
F 单参 Lambda 表达式 转化函数

表达式语义

以下只列出 前向序列 中没有定义或定义不同的表达式语义。

在下表中，v 是一个 transform_view 实例，s 是任意一个 前向序列，f 为一个单参 Lambda 表达式。

表达式 语义 
transform_view<s,f>
transform_view<s,f>::type
 一个惰性的 前向序列，对于区间 [begin<v>::type, end<v>::type) 中的每个 i 和区间 [begin<s>::type, end<s>::type) 中的每个 j，deref<i>::type 等同于 apply< f, deref<j>::type >::type.
size<v>::type v 的大小; size<v>::value == size<s>::value; 线性复杂度；请见 前向序列。

例子

在一个序列中查找最大的类型。

typedef vector<int,long,char,char[50],double> types;
typedef max_element<
      transform_view< types, size_of<_> >
    >::type iter;

BOOST_MPL_ASSERT_RELATION( deref<iter>::type::value, ==, 50 );
