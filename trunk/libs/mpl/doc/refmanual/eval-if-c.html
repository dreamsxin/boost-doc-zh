eval_if_c

摘要

template< 
      bool c
    , typename F1
    , typename F2
    >
struct eval_if_c
{
    typedef unspecified type;
};

描述

根据整型常量值 c 的值对两个 无参元函数 参数 F1 或 F2 中的某一个进行求值。eval_if_c<c,f1,f2>::type 为 eval_if< bool_<c>,f1,f2 >::type 的缩写。

头文件

#include <boost/mpl/eval_if.hpp>

参数

参数 要求 描述
c 整型常量 求值的条件
F1, F2 无参 元函数 选择进行求值的元函数

表达式语义

对于任意整型常量 c 和无参 元函数 f1, f2:

typedef eval_if_c<c,f1,f2>::type t;
返回类型: 任意类型
语义: 等价于 typedef eval_if< bool_<c>,f1,f2 >::type t;

例子

typedef eval_if_c< true, identity<char>, identity<long> >::type t1;
typedef eval_if_c< false, identity<char>, identity<long> >::type t2;

BOOST_MPL_ASSERT(( is_same<t1,char> ));
BOOST_MPL_ASSERT(( is_same<t2,long> ));

参见

元函数, 整型常量, eval_if, if_, bool_
