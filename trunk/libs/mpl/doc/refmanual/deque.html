deque

描述

deque 是一个 variadic, 随机访问, 可扩展 的类型序列，它支持在两端的常量时间的元素插入和删除操作，以及在序列中部的线性时间的插入和删除操作。在本库的实现中，deque 是 vector 的同义词。

头文件

#include <boost/mpl/deque.hpp>

Model of

  • Variadic 序列
  • 随机访问序列
  • 可扩展序列
  • 后端可扩展序列
  • 前端可扩展序列

表达式语义

请见 vector 的相关说明。

例子

typedef deque<float,double,long double> floats;
typedef push_back<floats,int>::type types;

BOOST_MPL_ASSERT(( is_same< at_c<types,3>::type, int > ));

参见

序列，vector, list, set
