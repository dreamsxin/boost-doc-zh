empty_sequence

摘要

struct empty_sequence
{
    // unspecified
    // ...
};

描述

表示不包含元素的序列。

头文件

#include <boost/mpl/empty_sequence.hpp>

表达式语义

以下只列出 随机访问序列 中没有定义或定义不同的表达式语义。

在下表中，s 是一个 empty_sequence 实例。

表达式 语义
empty_sequence 一个空的 随机访问序列
size<s>::type size<s>::value == 0；请见 随机访问序列

例子

typedef begin<empty_sequence>::type first;
typedef end<empty_sequence>::type last;

BOOST_MPL_ASSERT(( is_same<first,last> ));
BOOST_MPL_ASSERT_RELATION( size<empty_sequence>::value, ==, 0 );

typedef transform_view<
      empty_sequence
    , add_pointer<_>
    > empty_view;

BOOST_MPL_ASSERT_RELATION( size<empty_sequence>::value, ==, 0 );

序列，视图，vector, list, single_view
