protect

摘要

template< 
      typename F
    >
struct protect
{
    // unspecified
    // ...
};

描述

protect 是一个用于 元函数类 的身份包装器，防止其参数被识别为 绑定表达式。

头文件

#include <boost/mpl/protect.hpp>

参数

参数 要求 描述
F 元函数类 被包装的元函数

表达式语义

对于任意 元函数类 f:

typedef protect<f> g;
返回类型: 元函数类
语义:

如果 f 是一个 绑定表达式，等价于

struct g
{
    template< 
          typename U1 = unspecified,... typename Un = unspecified
        >
    struct apply
        : apply_wrapn<f,U1,...Un>
    {
    };
};

否则等价于 typedef f g;.
