erase

摘要

template<
      typename Sequence
    , typename First
    , typename Last = unspecified
    >
struct erase
{
    typedef unspecified type;
};

描述

erase 从序列中的任意位置开始删除一个或多个连续元素。

头文件

#include <boost/mpl/erase.hpp>

Model of

Tag 分派元函数

参数

参数 要求 描述
Sequence 可扩展序列 或 可扩展关联序列 要删除元素的序列
First 前向迭代器 指向要删除的区间开端的迭代器
Last 前向迭代器 指向要删除的区间尾端的迭代器

表达式语义

对于任意 可扩展序列 s, 以及 s 中的迭代器 pos, first 和 last:

typedef erase<s,first,last>::type r;
返回类型: 可扩展序列
前置条件: [first,last) 为 s 中的有效区间
语义: r 是一个新序列, 概念上等同于 s, 具有以下元素: [begin<s>::type, pos), [last, end<s>::type).
后置条件:

r 中的元素的相对位置与 s 中的一样;

size<r>::value == size<s>::value - distance<first,last>::value
 
typedef erase<s,pos>::type r;
返回类型: 可扩展序列
前置条件: pos 是 s 中的一个可提领迭代器
语义:

等价于

typedef erase< s,pos,next<pos>::type >::type r;

对于任意 可扩展关联序列 s, 以及 s 中的迭代器 pos:

typedef erase<s,pos>::type r;
返回类型: 可扩展序列
前置条件: pos 是 s 中的一个可提领迭代器
语义: 删除指定位置 pos 的元素; 等价于 erase_key<s, deref<pos>::type >::type.
后置条件: size<r>::value == size<s>::value - 1.

复杂度

序列原型 复杂度(区间形式)
可扩展关联序列 分期常量时间
可扩展序列 最坏情况下为二次多项式，最好情况下为线性

例子

typedef vector_c<int,1,0,5,1,7,5,0,5> values;
typedef find< values, integral_c<int,7> >::type pos;
typedef erase<values,pos>::type result;

BOOST_MPL_ASSERT_RELATION( size<result>::value, ==, 7 );

typedef find<result, integral_c<int,7> >::type iter;
BOOST_MPL_ASSERT(( is_same< iter, end<result>::type > ));

参见

可扩展序列, 可扩展关联序列, erase_key, pop_front, pop_back, insert
