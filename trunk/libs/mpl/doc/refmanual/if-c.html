if_c

摘要

template< 
      bool c
    , typename T1
    , typename T2
    >
struct if_c
{
    typedef unspecified type;
};

描述

根据整型常量值 c 的值返回两个参数 T1 或 T2 中的一个。if_c<c,t1,t2>::type 是 if_< bool_<c>,t1,t2 >::type 的缩写。

头文件

#include <boost/mpl/if.hpp>

参数

参数 要求 描述
c 整型常量 选择的条件
T1, T2 任意类型 从中进行选择的类型

表达式语义

对于任意整型常量 c 和任意类型 t1, t2:

typedef if_c<c,t1,t2>::type t;
返回类型: 任意类型
语义: 等价于 typedef if_< bool_<c>,t1,t2 >::type t;

例子 

typedef if_c<true,char,long>::type t1;
typedef if_c<false,char,long>::type t2;

BOOST_MPL_ASSERT(( is_same<t1, char> ));
BOOST_MPL_ASSERT(( is_same<t2, long> ));
// allocates space for an object of class T on heap or "inplace"
// depending on its size
template< typename T > struct lightweight
{
    // ...
    typedef typename if_c<
          sizeof(T) <= sizeof(T*)
        , inplace_storage<T>
        , heap_storage<T>
        >::type impl_t;

    impl_t impl;
};
