joint_view

摘要

template<
      typename Sequence1
    , typename Sequence2
    >
struct joint_view
{
    // unspecified
    // ...
};

描述

一个视图，由 Sequence1 和 Sequence2 的元素连接而成。

头文件

#include <boost/mpl/joint_view.hpp>

Model of

  • 前向序列

参数

参数 要求 描述
Sequence1, Sequence2 前向序列 用于创建视图的序列

表达式语义

以下只列出 前向序列 中没有定义或定义不同的表达式语义。

在下表中，v 是一个 joint_view 实例, s1 和 s2 是任意 前向序列。

表达式 语义 
joint_view<s1,s2>
joint_view<s1,s2>::type
 一个惰性 前向序列，所有元素位于区间 [begin<s1>::type, end<s1>::type), [begin<s2>::type, end<s2>::type).
size<v>::type v 的大小; size<v>::value == size<s1>::value + size<s2>::value; 线性复杂度；请见 前向序列。

例子

typedef joint_view<
      range_c<int,0,10>
    , range_c<int,10,15>
    > numbers;

BOOST_MPL_ASSERT(( equal< numbers, range_c<int,0,15> > ));

参见

序列, 视图, filter_view, transform_view, zip_view, iterator_range
