always

摘要

template< 
      typename X
    >
struct always
{
    // unspecified
    // ...
};

描述

always<X> 特化类是一个变长参数的 元函数类，它总是返回类型 X, 不管传入的参数数量与类型。

头文件

#include <boost/mpl/always.hpp>

Model of

元函数类

参数

参数 要求 描述
X 任意类型 返回的类型

表达式语义

对于任意类型 x:

typedef always<x> f;
返回类型: 元函数类
语义:

等价于

struct f : bind< identity<_1>, x > {};

例子

typedef always<true_> always_true;

BOOST_MPL_ASSERT(( apply< always_true,false_> ));
BOOST_MPL_ASSERT(( apply< always_true,false_,false_ > ));
BOOST_MPL_ASSERT(( apply< always_true,false_,false_,false_ > ));

参见

元函数, 元函数类, identity, bind, apply
