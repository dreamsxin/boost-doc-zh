字符串

描述

string 是一个 variadic, 双向的, 可扩展的 字符整型序列包装器，支持在两端的分摊常量时间的元素插入和删除，以及在中间的线性时间的元素插入和删除。给 string 的参数是多字符字面值，可以是任何可读语法的编译期字符串。string 也可以作为 c_str 元函数的参数，生成一个以null-结尾的字符数组，与运行时字符串操作函数进行互动。

头文件

序列形式 头文件
Variadic #include <boost/mpl/string.hpp>

Model of

  • 整型序列包装器
  • Variadic 序列
  • 双向序列
  • 可扩展序列
  • 后端可扩展序列
  • 前端可扩展序列

表达式语义

在下表中，s 是一个 string 实例，pos 和 last 是 s 中的迭代器，r 是一个字符前向序列，n 和 x 是 整型常量，而 c1,c2,... cn 是任意(多个)字符。

表达式 语义 
string<c1,c2,... cn>
 字符 c1,c2,... cn 的string；见 Variadic 序列。 
string<c1,c2,... cn>::type
 与 string<c1,c2,... cn> 相同；见 Variadic 序列。
begin<s>::type 指向 s 始端的迭代器；见 双向序列。
end<s>::type 指向 s 末端的迭代器；见 双向序列。
size<s>::type s 的大小；见 双向序列。
empty<s>::type 布尔 整型常量 c 满足 c::value == true 当且仅当序列为空；见 双向序列。
front<s>::type s 中第一个元素；见 双向序列。
back<s>::type s 中最后一个元素；见 双向序列。
insert<s,pos,x>::type 一个新 string，其元素如下：[begin<s>::type, pos), x, [pos, end<s>::type)；见 可扩展序列。
insert_range<s,pos,r>::type 一个新 string，其元素如下：[begin<s>::type, pos), [begin<r>::type, end<r>::type) [pos, end<s>::type)；见 可扩展序列。
erase<s,pos>::type 一个新 string，其元素如下：[begin<s>::type, pos), [next<pos>::type, end<s>::type)；见 可扩展序列。
erase<s,pos,last>::type 一个新 string，其元素如下：[begin<s>::type, pos), [last, end<s>::type)；见 可扩展序列。
clear<s>::type 一个空 string；见 可扩展序列。
push_back<s,x>::type 一个新 string，其元素如下：[begin<s>::type, end<s>::type), x；见 后端可扩展序列。
pop_back<s>::type 一个新 string，其元素如下：[begin<s>::type, prior< end<s>::type >::type)；见 后端可扩展序列。
push_front<s,x>::type 一个新 string，其元素如下：[begin<s>::type, end<s>::type), x；见 前端可扩展序列。
pop_front<s>::type 一个新 string，其元素如下：[next< begin<s>::type >::type, end<s>::type)；见 前端可扩展序列。
c_str<s>::value 一个以null-结尾的字节串，满足 c_str<s>::value[n] 等于 s 中第 n-个字符，且 c_str<s>::value[size<s>::type::value] 为 '\0'。

例子

typedef mpl::string<'hell','o wo','rld'> hello;
typedef mpl::push_back<hello, mpl::char_<'!'> >::type hello2;

BOOST_ASSERT(0 == std::strcmp(mpl::c_str<hello2>::value, "hello world!"));
