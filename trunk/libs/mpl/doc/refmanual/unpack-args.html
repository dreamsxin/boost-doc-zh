unpack_args

摘要

template< 
      typename F
    >
struct unpack_args
{
    // unspecified
    // ...
};

描述

一个高阶的简单转化，将一个 n-元 Lambda 表达式 F 转化为一个单参 元函数类 g，接受由原来的 n 个参数所组成的序列为单个参数。

头文件

#include <boost/mpl/unpack_args.hpp>

Model of

元函数类

参数

参数 要求 描述
F Lambda 表达式 被转化的 lambda 表达式

表达式语义

对于任意 Lambda 表达式 f, 以及任意类型 a1,... an:

typedef unpack_args<f> g;
返回类型: 元函数类
语义:

g 是一个单参 元函数类，满足

apply_wrapn< g, vector<a1,...an> >::type

等同于

apply<F,a1,...an>::type

例子

BOOST_MPL_ASSERT(( apply<
      unpack_args< is_same<_1,_2> >
    , vector<int,int>
    > ));

参见

元函数, Lambda 表达式, 元函数类, apply, apply_wrap, bind
