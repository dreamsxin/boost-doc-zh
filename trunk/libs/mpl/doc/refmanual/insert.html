insert

摘要

template< 
      typename Sequence
    , typename Pos
    , typename T 
    >
struct insert
{
    typedef unspecified type;
};


template< 
      typename Sequence
    , typename T 
    >
struct insert
{
    typedef unspecified type;
};

描述

insert 是一个 重载的名字:

  • insert<Sequence,Pos,T> 将类型 T 插入于 Sequence 中的任意位置 Pos. 如果Sequence 是 可扩展关联序列 的 model，则Pos 被忽略。
  • insert<Sequence,T> 是 insert<Sequence,Pos,T> 的缩写，当Sequence 是 可扩展关联序列 的 model 时。

头文件

#include <boost/mpl/insert.hpp>

Model of

Tag 分派元函数

参数

参数 要求 描述
Sequence 可扩展序列 或 可扩展关联序列 插入元素的序列
Pos 前向迭代器 Sequence 中的迭代器，指定插入的位置
T 任意类型 插入的元素

表达式语义

对于任意 可扩展序列 s, s 中的迭代器 pos, 以及任意类型 x:

typedef insert<s,pos,x>::type r;
返回类型: 可扩展序列
前置条件: pos 是 s 中的一个迭代器
语义: r 是一个序列, 概念上等同于 s, 具有以下元素: [begin<s>::type, pos), x, [pos, end<s>::type).
后置条件:

r 中的元素的相对位置与 s 中的相同。

at< r, distance< begin<s>::type,pos >::type >::type

等同于 x;

size<r>::value == size<s>::value + 1;

对于任意 可扩展关联序列 s, s 中的迭代器 pos, 以及任意类型 x:

typedef insert<s,x>::type r;
返回类型: 可扩展关联序列
语义: r 概念于等同于 且等价于 s, 除了 at< r, key_type<s,x>::type >::type 等同于 value_type<s,x>::type.
后置条件: size<r>::value == size<s>::value + 1. 
typedef insert<s,pos,x>::type r;
返回类型: 可扩展关联序列
前置条件: pos 为 s 中的迭代器
语义: 等价于 typedef insert<s,x>::type r; pos 被忽略

复杂度

序列原型 复杂度
可扩展关联序列 分期常量时间
可扩展序列 最坏情况为线性时间，或者为分期常量时间

例子

typedef vector_c<int,0,1,3,4,5,6,7,8,9> numbers;
typedef find< numbers,integral_c<int,3> >::type pos;
typedef insert< numbers,pos,integral_c<int,2> >::type range;

BOOST_MPL_ASSERT_RELATION( size<range>::value, ==, 10 );
BOOST_MPL_ASSERT(( equal< range,range_c<int,0,10> > ));
typedef map< mpl::pair<int,unsigned> > m;
typedef insert<m,mpl::pair<char,long> >::type m1;

BOOST_MPL_ASSERT_RELATION( size<m1>::value, ==, 2 );
BOOST_MPL_ASSERT(( is_same< at<m1,int>::type,unsigned > ));
BOOST_MPL_ASSERT(( is_same< at<m1,char>::type,long > ));
