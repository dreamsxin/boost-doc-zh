iterator_category

摘要

template<
      typename Iterator
    >
struct iterator_category
{
    typedef typename Iterator::category type;
};

描述

返回以下迭代器类别 tag 之一：

  • forward_iterator_tag
  • bidirectional_iterator_tag
  • random_access_iterator_tag

头文件

#include <boost/mpl/iterator_category.hpp>
#include <boost/mpl/iterator_tags.hpp>

参数

参数 要求 描述
Iterator 前向迭代器 获取类别标识的迭代器

表达式语义

对于任意 前向迭代器 iter:

typedef iterator_category<iter>::type tag;
返回类型: 整型常量
语义: tag 为 forward_iterator_tag 如果 iter 符合 前向迭代器, 为 bidirectional_iterator_tag 如果 iter 符合 双向迭代器, 或为 random_access_iterator_tag 如果 iter 符合 随机访问迭代器
后置条件:
forward_iterator_tag::value < bidirectional_iterator_tag::value,
bidirectional_iterator_tag::value < random_access_iterator_tag::value.

复杂度

分期常量时间。

例子

template< typename Tag, typename Iterator >
struct algorithm_impl
{
    // O(n) 实现
};

template< typename Iterator >
struct algorithm_impl<random_access_iterator_tag,Iterator>
{
    // O(1) 实现
};

template< typename Iterator >
struct algorithm
    : algorithm_impl<
          iterator_category<Iterator>::type
        , Iterator
        >
{
};

参见

迭代器, begin / end, advance, distance, next
