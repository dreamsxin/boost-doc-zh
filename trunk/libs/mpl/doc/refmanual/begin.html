begin

摘要

template<
      typename X
    >
struct begin
{
    typedef unspecified type;
};

描述

返回一个迭代器，指向序列的第一个元素。如果其参数不是一个 前向序列, 则返回 void_.

头文件

#include <boost/mpl/begin_end.hpp>

Model of

Tag 分派元函数

参数

参数 要求 描述
X 任意类型 要返回其 begin 迭代器的类型

表达式语义

对于任意类型 x:

typedef begin<x>::type first;
返回类型: 前向迭代器 或 void_.
语义: 如果 x 是一个 前向序列, first 是一个指向 s 的第一个元素的迭代器; 否则 first 为 void_.
后置条件: 如果 first 是一个迭代器，要么它是可提领的，要么它是 past-the-end; 它是 past-the-end 当且仅当 size<x>::value == 0.

复杂度

分期常量时间。

例子

typedef vector< unsigned char,unsigned short,
    unsigned int,unsigned long > unsigned_types;

typedef begin<unsigned_types>::type iter;
BOOST_MPL_ASSERT(( is_same< deref<iter>::type, unsigned char > ));

BOOST_MPL_ASSERT(( is_same< begin<int>::type, void_ > ));

参见

迭代器, 前向序列, end, size, empty
