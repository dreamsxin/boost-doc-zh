lower_bound

摘要

template<
      typename Sequence
    , typename T
    , typename Pred = less<_1,_2>
    >
struct lower_bound
{
    typedef unspecified type;
};

描述

返回已排序的 Sequence 中第一个可以插入 T 而不破坏顺序性的位置。

头文件

#include <boost/mpl/lower_bound.hpp>

参数

参数 要求 描述
Sequence 前向序列 查找的已排序序列
T 任意类型 查找位置的类型
Pred 二元 Lambda 表达式 查找条件

表达式语义

对于任意已排序 前向序列 s, 二元 Lambda 表达式 pred, 和任意类型 x:

typedef lower_bound< s,x,pred >::type i;
返回类型: 前向迭代器
语义:

i 是 [begin<s>::type, end<s>::type) 中满足以下条件的最靠后的迭代器，对于 [begin<s>::type, i) 中的每个迭代器 j，

apply< pred, deref<j>::type, x >::type::value == true

复杂度

比较的次数是对数的：最多为 log2( size<s>::value ) + 1 次。如果 s 是一个 随机访问序列 则在区间中移动的步数也是对数的；否则移动的步数与 size<s>::value 也正比。

例子

typedef vector_c<int,1,2,3,3,3,5,8> numbers;
typedef lower_bound< numbers, int_<3> >::type iter;

BOOST_MPL_ASSERT_RELATION( 
      (distance< begin<numbers>::type,iter >::value), ==, 2
    );

BOOST_MPL_ASSERT_RELATION( deref<iter>::type::value, ==, 3 );
