extent 

template <class T, std::size_t N = 0>
struct extent : public integral_constant<std::size_t, EXTENT(T,N)> {};

继承：当 EXTENT(T,N) 是类型为 T 的第 N 个数组维的元素的个数时，类模板 extent 从 integral_constant<std::size_t,EXTENT(T,N)> 继承。

如果 T 不是一个数组类型，或者 N > rank<T>::value，或者第 N 个数组的绑定是不完全的，那么 EXTENT(T,N) 为 0。

头文件：#include <boost/type_traits/extent.hpp> 或 #include <boost/type_traits.hpp>

示例：

extent<int[1]> 从 integral_constant<std::size_t, 1> 继承。

extent<double[2][3][4], 1>::type 是 integral_constant<std::size_t, 3> 类型。

extent<int[4]>::value 为一个求值为 4 的整型常量表达式。

extent<int[][2]>::value 为一个求值为 0 的整型常量表达式。

extent<int[][2], 1>::value 为一个求值为 2 的整型常量表达式。

extent<int*>::value 为一个求值为 0 的整型常量表达式。

extent<T>::value_type 是 std::size_t 类型。
