is_base_of 

template <class Base, class Derived>
struct is_base_of : public true_type-or-false_type {};

继承：如果 Base 是类型 Derived 的基类，或者两个类型相同，则从 true_type 继承，否则从 false_type 继承。

这个模板可用于检测非公有基类，和 ambiguous base classes（歧义基类）。

注意，is_base_of<X,X> 总是从 true_type 继承。甚至在 X 不是一个类类型时也是如此。这是 Boost-1.33 为了符合 C++ 库扩展技术报告而做的行为上的更改。

类型 Base 和 Derived 一定不能是不完整类型。

C++ 标准参考：10。

头文件：#include<boost/type_traits/is_base_of.hpp> 或 #include<boost/type_traits.hpp>

编译器兼容性：如果编译器不支持类模板的偏特化，则这个模板可能不能和 function types（函数类型）一起使用。如果 Base 是 Derived 的私有基类，或者 Base 是 Derived 的 ambiguous base classes（歧义基类），则在一些老旧的编译器上可能会产生编译错误。这些编译器包括 Borland C++，Sun Forte C++ 的旧版本，Digital Mars C++，和基于 EDG 的旧版本的编译器。

示例：

给定：class Base{}; class Derived : public Base{};

is_base_of<Base, Derived> 从 true_type 继承。

is_base_of<Base, Derived>::type 是 true_type 类型。

is_base_of<Base, Derived>::value 是一个求值为 true 的 integral constant expression（整常表达式）。

is_base_of<Base, Derived>::value 是一个求值为 true 的 integral constant expression（整常表达式）。

is_base_of<Base, Base>::value 是一个求值为 true 的 integral constant expression（整常表达式）：一个类被看做它自己的基类。

is_base_of<T, T>::value_type 是 bool 类型。
