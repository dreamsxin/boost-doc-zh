decay 

template <class T>
struct decay
{
   typedef see-below type;
};

类型：设 U 为 remove_reference<T>::type 的结果，则若 U 为数组类型，则结果为 remove_extent<U>*，否则若 U 为函数类型则结果为 U*，否则结果为 U。

C++ 标准参考：3.9.1.

头文件： #include <boost/type_traits/decay.hpp> 或 #include <boost/type_traits.hpp>

表 1.10. 示例

表达式

结果类型

decay<int[2][3]>::type

int[2]*

decay<int(&)[2]>::type

int*

decay<int(&)(double)>::type

int(*)(double)

int(*)(double

int(*)(double)

int(double)

int(*)(double)
