add_pointer 

template <class T>
struct add_pointer
{
   typedef see-below type;
};

类型：与 remove_reference<T>::type* 类型相同。

这个模板的基本原理是：当 t 是类型 T 的一个对象时，它生成和 TYPEOF(&t) 一样的类型。

C++ 标准参考：8.3.1。

编译器兼容性：如果编译器不支持类模板的偏特化，这个模板也可以编译，但是成员 type 总是和 T 类型相同，除非编译器 workarounds 可用。

头文件：#include <boost/type_traits/add_pointer.hpp> 或 #include <boost/type_traits.hpp>
