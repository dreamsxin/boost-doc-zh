is_virtual_base_of 

template <class Base, class Derived>
struct is_virtual_base_of : public true_type-or-false_type {};

继承：如果 Base 是类型 Derived 的虚拟基类，则继承自 true_type，否则继承自 false_type.

类型 Base 和 Derived 必须不能是不完整类型。

C++ 标准参考：10.

头文件：#include <boost/type_traits/is_virtual_base_of.hpp> 或 #include <boost/type_traits.hpp>

编译器兼容性：这个 trait 要求 is_base_of trait 可用。

示例：

给定：class Base{}; class Derived : public virtual Base{};

is_virtual_base_of<Base, Derived> 继承自 true_type.

is_virtual_base_of<Base, Derived>::type 为 true_type.

is_virtual_base_of<Base, Derived>::value 是一个求值为 true 的整型常量表达式。

is_virtual_base_of<Base, Derived>::value 是一个求值为 true 的整型常量表达式。

is_virtual_base_of<T, U>::value_type 为 bool.
