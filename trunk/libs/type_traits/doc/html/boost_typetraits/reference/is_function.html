is_function 

template <class T>
struct is_function : public true_type-or-false_type {};

继承：如果 T 是一个（可能被 cv 修饰的）函数类型，则从 true_type 继承，否则从 false_type 继承。注意，这个模板不会检测指向函数的指针，或引向函数的引用，那些可以分别通过 is_pointer 和 is_reference 进行检测：

typedef int f1();      // f1 是函数类型。
typedef int (f2*)();   // f2 是函数指针。
typedef int (f3&)();   // f3 是函数引用。

C++ 标准参考：3.9.2p1 and 8.3.5。

头文件：#include<boost/type_traits/is_function.hpp> 或 #include<boost/type_traits.hpp>

示例：

is_function<int(void)> 从 true_type 继承。

is_function<long(double,int)>::type 是 true_type 类型。

is_function<long(double,int)>::value 是一个求值为 true 的 integral constant expression（整常表达式）。

is_function<long(*)(double,int)>::value 是一个求值为 false 的 integral constant expression（整常表达式）：本情况下，那个参数是指针类型，不是函数类型。

is_function<long(&)(double,int)>::value 是一个求值为 false 的 integral constant expression（整常表达式）：本情况下，那个参数是一个引向函数的引用，不是函数类型。

is_function<long(MyClass::*)(double,int)>::value 是一个求值为 false 的 integral constant expression（整常表达式）：本情况下，那个参数是一指向成员函数的指针。

is_function<T>::value_type 是 bool 类型。

提示

不要混淆 function-types（函数类型）和 pointers to functions（指向函数的指针）：

typedef int f (double);

定义一个 function type（函数类型），

f foo;

声明一个类型为 f 的函数的原型，

f* pf = foo;
f& fr = foo;

声明一个指向函数 foo 的指针和一个引向函数 foo 的引用。

如果你要检测某个类型是不是一个指向函数的指针，那就使用：

is_function<remove_pointer<T>::type>::value && is_pointer<T>::value

或者对于指向成员函数的指针你可以直接使用 is_member_function_pointer。
