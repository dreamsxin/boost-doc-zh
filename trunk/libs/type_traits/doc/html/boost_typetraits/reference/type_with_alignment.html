type_with_alignment 

template <std::size_t Align>
struct type_with_alignment
{
   typedef see-below type;
};

类型：对齐到 Align 倍数的某种内建或 POD 类型。

头文件：#include<boost/type_traits/type_with_alignment.hpp> 或 #include<boost/type_traits.hpp>
