is_unsigned 

template <class T>
struct is_unsigned : public true_type-or-false_type {};

继承：如果 T 是一个无符号整数类型或一个底层为无符号整数类型的枚举类型，则从 true_type 继承，否则从 false_type 继承。

C++ 标准参考：3.9.1, 7.2.

头文件：#include <boost/type_traits/is_unsigned.hpp> 或 #include <boost/type_traits.hpp>

示例：

is_unsigned<unsigned int> 从 true_type 继承。

is_unsigned<unsigned int const volatile>::type 是 true_type 类型。

is_unsigned<int>::value 是一个求值为 false 的 integral constant expression（整常表达式）。

is_unsigned<myclass>::value 是一个求值为 false 的 integral constant expression（整常表达式）。

is_unsigned<char>::value 是一个值依赖于类型 char 的符号性的 integral constant expression（整常表达式）。

is_unsigned<unsigned long long>::value 是一个求值为 true 的 integral constant expression（整常表达式）。

is_unsigned<T>::value_type 是 bool 类型。
