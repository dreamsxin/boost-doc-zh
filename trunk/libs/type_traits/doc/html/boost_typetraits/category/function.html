解析函数类型

类模板 function_traits 从函数类型（参见 is_function）中提取信息。这个 traits 类允许你知道一个函数持有多少个参数，那些参数的类型是什么，以及返回类型是什么。

概要 

template <std::size_t Align>
struct function_traits;
