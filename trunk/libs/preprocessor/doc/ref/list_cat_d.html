BOOST_PP_LIST_CAT_D 宏将一个链表中的所有元素串接起来。它以最高的效率重入 BOOST_PP_WHILE.  

用法

BOOST_PP_LIST_CAT_D(d, list)

参数

d
下一个可用的 BOOST_PP_WHILE 迭代。
list
要将元素进行串接的链表。

备注

元素被从左到右串接起来，从索引 0 开始。

参见

  • BOOST_PP_LIST_CAT

要求

头文件:  <boost/preprocessor/list/cat.hpp>

代码示例

#include <boost/preprocessor/cat.hpp>
#include <boost/preprocessor/list/cat.hpp>
#include <boost/preprocessor/list/fold_left.hpp>

#define LISTS \
   ((a, (b, BOOST_PP_NIL)), \
      ((c, (d, BOOST_PP_NIL)), \
         ((e, (f, BOOST_PP_NIL)), \
            BOOST_PP_NIL))) \
   /**/

#define OP(d, state, x) BOOST_PP_CAT(state, BOOST_PP_LIST_CAT_D(d, x))

BOOST_PP_LIST_FOLD_LEFT(OP, _, LISTS) // 展开为 _abcdef
