BOOST_PP_ARRAY_POP_FRONT_Z 宏从一个数组的头部弹出一个元素。它以最高的效率重入 BOOST_PP_REPEAT.

用法

BOOST_PP_ARRAY_POP_FRONT_Z(z, array)

参数

z
下一个可用的 BOOST_PP_REPEAT 维度。
array
弹出元素的数组。

备注

该宏返回删除最后一个元素后的数组。如果数组中没有元素，则该宏的结果为未定义。
