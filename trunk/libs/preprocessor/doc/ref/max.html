BOOST_PP_MAX 宏展开为两个参数中的较大者。

用法

BOOST_PP_MAX(x, y)

参数

x
第一个操作数。有效值为 0 到 BOOST_PP_LIMIT_MAG.
y
第二个操作数。有效值为 0 到 BOOST_PP_LIMIT_MAG.

备注

该宏返回其两个参数中的较大者，或者如果两个参数相等，则返回该值。
以前，该宏不可用在 BOOST_PP_WHILE 中。现在已没有该限制。不过在此情况下，使用 BOOST_PP_MAX_D 更为高效。

参见

  • BOOST_PP_LIMIT_MAG
  • BOOST_PP_MAX_D

要求

头文件:  <boost/preprocessor/selection/max.hpp>

代码示例

#include <boost/preprocessor/selection/max.hpp>

BOOST_PP_MAX(5, 7) // 展开为 7
BOOST_PP_MAX(3, 3) // 展开为 3
