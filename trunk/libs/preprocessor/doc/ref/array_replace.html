BOOST_PP_ARRAY_REPLACE 宏替换数组中的某个元素。

用法

BOOST_PP_ARRAY_REPLACE(array, i, elem)

参数

array
要替换元素的数组。
i
被替换的元素在数组中的位置，从零起计。有效值为 0 到 BOOST_PP_ARRAY_SIZE(array) - 1.
elem
替换的新元素。

备注

该宏在内部使用了 BOOST_PP_WHILE. 因此，要使用从其它使用了 BOOST_PP_WHILE 的宏所传入的 d 参数，请见 BOOST_PP_ARRAY_REPLACE_D.

参见

  • BOOST_PP_ARRAY_REPLACE_D

要求

头文件:  <boost/preprocessor/array/replace.hpp>

代码示例

#include <boost/preprocessor/array/replace.hpp>

#define ARRAY (3, (a, x, c))

BOOST_PP_ARRAY_REPLACE(ARRAY, 1, b) // 展开为 (3, (a, b, c))
