BOOST_PP_ADD 宏展开为其参数的和。

用法

BOOST_PP_ADD(x, y)

参数

x
操作的第一个加数。有效值范围为 0 到 BOOST_PP_LIMIT_MAG.
y
操作的第二个加数。有效值范围为 0 到 BOOST_PP_LIMIT_MAG. 

备注

如果 x 和 y 的和大于 BOOST_PP_LIMIT_MAG, 结果将被调整为 BOOST_PP_LIMIT_MAG.
以前，这个宏不能在 BOOST_PP_WHILE 中使用。现在已没有这样的限制了。不过，在这种情况下，使用 BOOST_PP_ADD_D 更为高效。
当 x 小于等于 y 时，这个宏最为高效。不过，得到的效率提高并不值得在调用之前对两个参数进行比较。换言之，x 很可能是两个参数中较大的一个。

参见

  • BOOST_PP_ADD_D
  • BOOST_PP_LIMIT_MAG

要求

头文件:  <boost/preprocessor/arithmetic/add.hpp>

代码示例

#include <boost/preprocessor/arithmetic/add.hpp>

BOOST_PP_ADD(4, 3) // 展开为 7
