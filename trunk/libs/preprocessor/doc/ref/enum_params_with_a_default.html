BOOST_PP_ENUM_PARAMS_WITH_A_DEFAULT 宏生成一个以逗号分隔的带有相同缺省值的参数列表。

用法

BOOST_PP_ENUM_PARAMS_WITH_A_DEFAULT(count, param, def)

参数

count
要生成的参数数量。有效值为 0 到 BOOST_PP_LIMIT_REPEAT.
param
参数的文本。BOOST_PP_ENUM_PARAMS_WITH_A_DEFAULT 会将范围 0 到 count - 1 间的数字连接到它以生成参数。
def
每个参数的缺省值。

备注

该宏展开为一个以逗号分隔的序列:  
param ## 0 = def, param ## 1 = def, ... param ## count - 1 = def
以前，该宏不可以在 BOOST_PP_REPEAT 中递归使用。现在这个限制已不存在，因为本库可以自动检测下一个可用的迭代深度。
该宏已被反对使用。它只是为了后向兼容而保留。请改用带 BOOST_PP_INTERCEPT 的 BOOST_PP_ENUM_BINARY_PARAMS:
BOOST_PP_ENUM_BINARY_PARAMS(count, param, = def BOOST_PP_INTERCEPT)

参见

  • BOOST_PP_ENUM_BINARY_PARAMS
  • BOOST_PP_INTERCEPT
  • BOOST_PP_LIMIT_REPEAT

要求

头文件:  <boost/preprocessor/repetition/enum_params_with_a_default.hpp>

代码示例

#include <boost/preprocessor/repetition/enum_params_with_a_default.hpp>

BOOST_PP_ENUM_PARAMS_WITH_A_DEFAULT(3, class T, int)
   // 展开为 T0 = int, T1 = int, T2 = int

BOOST_PP_ENUM_BINARY_PARAMS(3, class T, = int BOOST_PP_INTERCEPT)
   // 展开为 T0 = int, T1 = int, T2 = int
