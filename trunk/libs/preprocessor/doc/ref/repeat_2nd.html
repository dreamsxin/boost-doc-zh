BOOST_PP_REPEAT_2ND 宏表示 BOOST_PP_REPEAT 的第二维度。

用法

BOOST_PP_REPEAT_2ND(count, macro, data)

参数

count
重复调用 macro 的次数。有效值为 0 到 BOOST_PP_LIMIT_REPEAT.
macro
形如 macro(z, n, data) 的三元操作。该宏被 BOOST_PP_REPEAT_2ND 展开，参数分别为下一个可用的迭代深度、当前迭代值，以及辅助数据 data 参数。 
data
传递给 macro 的辅助数据。

备注

该宏展开为序列:
macro(z, 0, data) macro(z, 1, data) ... macro(z, count - 1, data)
传给 macro 的 z 值代表下一个可用的迭代维度。其它带 _Z 后缀的宏在内部使用了 BOOST_PP_REPEAT -- 例如 BOOST_PP_ENUM_PARAMS 和 BOOST_PP_ENUM_PARAMS_Z.  并不强制使用这些 _Z 版本，不过将 z 值(传给 macro 的值)传给这些宏可以使它们以最高的效率重入 BOOST_PP_REPEAT.
要直接使用这个 z 值，而不是只将它传给其它宏，请见 BOOST_PP_REPEAT_z.
该宏已被反对使用。它只是为了后向兼容性而保留。请改用 BOOST_PP_REPEAT.

参见

  • BOOST_PP_LIMIT_REPEAT
  • BOOST_PP_REPEAT
  • BOOST_PP_REPEAT_z

要求

头文件:  <boost/preprocessor/repetition/repeat.hpp>

代码示例

#include <boost/preprocessor/repetition/repeat.hpp>

#define DECL(z, n, text) text ## n = n;

BOOST_PP_REPEAT_2ND(5, DECL, int x)
