BOOST_PP_EXPAND 宏对其参数执行一个双重展开。

用法

BOOST_PP_EXPAND(x)

参见

x
要展开两次的参数。

备注

当生成某个宏调用的正确语义需要延迟时，该宏就非常有用。例如，当一个宏被展开为另一个宏的参数列表时。这个宏第一遍展开参数列表，然后再进行扫描以展开更多的宏。

要求

头文件:  <boost/preprocessor/facilities/expand.hpp>

代码示例

#include <boost/preprocessor/control/if.hpp>
#include <boost/preprocessor/facilities/expand.hpp>

#define MACRO(a, b, c) (a)(b)(c)
#define ARGS() (1, 2, 3)

BOOST_PP_EXPAND(MACRO ARGS()) // 展开为 (1)(2)(3)

#define SAMPLE(n) \
   BOOST_PP_EXPAND( \
      MACRO \
      BOOST_PP_IF( \
         n, \
         (x, y, z), \
         (a, b, c) \
      ) \
   ) \
   /**/

SAMPLE(0) // 展开为 (a)(b)(c)
SAMPLE(1) // 展开为 (x)(y)(z)
