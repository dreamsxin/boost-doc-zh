BOOST_PP_RPAREN 宏展开为一个右括号。

用法

BOOST_PP_RPAREN()

备注

预处理器在宏调用中会将括号解释为分隔符。因此，需要对括号进行特殊的处理。

参见

  • BOOST_PP_LPAREN

要求

头文件:  <boost/preprocessor/punctuation/paren.hpp>

代码示例

#include <boost/preprocessor/facilities/empty.hpp>
#include <boost/preprocessor/punctuation/paren.hpp>

#define X(x) x
#define MACRO(x, p) X ( x p

MACRO(abc, BOOST_PP_RPAREN()) // 展开为 abc

#define Y(x)

MACRO(BOOST_PP_EMPTY BOOST_PP_RPAREN()(), 10) // 展开为 10
