BOOST_PP_BITOR 宏展开为其操作数的按位 OR.

用法

BOOST_PP_BITOR(x, y)

参数

x
左操作数。其值必须展开为 0 或 1.
y
右操作数。其值必须展开为 0 或 1.

备注

如果 x 或 y 为 1, 该宏展开为 1. 否则展开为 0.
该宏在执行按位或的操作之前不会对其操作数执行布尔类型转换。如果需要转换，请使用 BOOST_PP_OR.

参见

  • BOOST_PP_OR

要求

头文件:  <boost/preprocessor/logical/bitor.hpp>

代码示例

#include <boost/preprocessor/logical/bitor.hpp>

BOOST_PP_BITOR(0, 0) // 展开为 0
BOOST_PP_BITOR(0, 1) // 展开为 1
BOOST_PP_BITOR(1, 0) // 展开为 1
BOOST_PP_BITOR(1, 1) // 展开为 1
