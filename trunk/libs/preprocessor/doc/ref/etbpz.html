BOOST_PP_ENUM_TRAILING_BINARY_PARAMS_Z 宏生成一个以逗号分隔并以逗号开头的二元参数列表。它以最高的效率重入 BOOST_PP_REPEAT. 

用法

BOOST_PP_ENUM_TRAILING_BINARY_PARAMS_Z(z, count, p1, p2)

参数

z
下一个可用的 BOOST_PP_REPEAT 维度。
count
要生成的参数数量。有效值为 0 到 BOOST_PP_LIMIT_REPEAT.
p1
参数的第一部分的文本。BOOST_PP_ENUM_TRAILING_BINARY_PARAMS_Z 会将范围 0 到 count - 1 间的数字连接到它以生成参数。
p2
参数的第二部分的文本。BOOST_PP_ENUM_TRAILING_BINARY_PARAMS_Z 会将范围 0 到 count - 1 间的数字连接到它以生成参数。

备注

该宏展开为一个以逗号分隔的序列:
, p1 ## 0 p2 ## 0, p1 ## 1 p2 ## 1, ... p1 ## count - 1 p2 ## count - 1

参见

  • BOOST_PP_ENUM_TRAILING_BINARY_PARAMS
  • BOOST_PP_LIMIT_REPEAT
