BOOST_PP_TUPLE_REM 宏展开为一个宏，该宏去掉一个指定大小的 tuple 的括号。

用法

BOOST_PP_TUPLE_REM(size)

参数

size
要去掉括号的 tuple 的大小。有效的 tuple 大小范围为 0 到 BOOST_PP_LIMIT_TUPLE. 

备注

size 参数必须为 tuple 的真实大小。

参见

  • BOOST_PP_LIMIT_TUPLE

要求

头文件:  <boost/preprocessor/tuple/rem.hpp>

代码示例

#include <boost/preprocessor/tuple/rem.hpp>

BOOST_PP_TUPLE_REM(3)(x, y, z) // 展开为 x, y, z
