BOOST_PP_EXPR_IIF 宏展开为它的第二个参数，如果第一个参数为 1；如果第一个参数为 0 则展开为空。

用法

BOOST_PP_EXPR_IIF(bit, expr)

参数

bit
确定结果为 expr 或为空的条件。该值必须展开为 0 或 1.
expr
当 bit 为 1 时的表达式结果。

备注

该宏不会对其第一个参数执行布尔类型转换。如果需要转换，请改用 BOOST_PP_EXPR_IF.

参见

  • BOOST_PP_EXPR_IF
  • BOOST_PP_LIMIT_MAG

要求

头文件:  <boost/preprocessor/control/expr_iif.hpp>

代码示例

#include <boost/preprocessor/logical/and.hpp>
#include <boost/preprocessor/control/expr_iif.hpp>

#define INSERT_AND(p, q, text) \
   BOOST_PP_EXPR_IIF( \
      BOOST_PP_AND(p, q), \
      text \
   ) \
   /**/

INSERT_AND(2, 3, abc) // 展开为 abc
INSERT_AND(0, 7, xyz) // 展开为空
