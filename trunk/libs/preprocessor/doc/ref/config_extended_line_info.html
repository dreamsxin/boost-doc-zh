BOOST_PP_CONFIG_EXTENDED_LINE_INFO 是一个用户自定义宏，它决定了 BOOST_PP_LINE 是否输出扩展的 文件迭代 状态信息。

用法

#define BOOST_PP_CONFIG_EXTENDED_LINE_INFO n

参见

n
该值决定了 BOOST_PP_LINE 是否输出扩展的 文件迭代 状态信息。该值必须为 0 或 1.

备注

如果 n 为 1, BOOST_PP_LINE 将输出扩展信息。缺省情况下，该宏被设为 0.
