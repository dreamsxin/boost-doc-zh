BOOST_PP_DIV 宏展开为其参数的商。

用法

BOOST_PP_DIV(x, y)

参数

x
操作的被除数(分子)。有效值为 0 到 BOOST_PP_LIMIT_MAG.
y
操作的除数(分母)。有效值为 1 到 BOOST_PP_LIMIT_MAG.

备注

以前，该宏不可用在 BOOST_PP_WHILE 内。这种限制已经取消。不过，在这种情况下，使用 BOOST_PP_DIV_D 会更高效。
如果 y 为 0, 则结果未定义。

参见

  • BOOST_PP_DIV_D
  • BOOST_PP_LIMIT_MAG

要求

头文件:  <boost/preprocessor/arithmetic/div.hpp>

代码示例

#include <boost/preprocessor/arithmetic/div.hpp>

BOOST_PP_DIV(11, 5) // 展开为 2
