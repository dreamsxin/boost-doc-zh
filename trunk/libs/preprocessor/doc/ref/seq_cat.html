BOOST_PP_SEQ_CAT 宏串接一个 seq 中的所有元素。

用法

BOOST_PP_SEQ_CAT(list)

参数

seq
要对其中元素进行串接的 seq.

备注

从索引 0 起计的元素被从左到右串接起来。
要达到最高效率，请使用 BOOST_PP_SEQ_CAT_S.

参见

  • BOOST_PP_SEQ_CAT_S

要求

头文件:  <boost/preprocessor/seq/cat.hpp>

代码示例 

#include <boost/preprocessor/seq/cat.hpp>

#define SEQ (a)(b)(c)

BOOST_PP_SEQ_CAT(SEQ) // 展开为 abc
