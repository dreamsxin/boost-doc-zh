BOOST_PP_LIST_FOLD_RIGHT 宏从右至左对一个链表的元素进行折叠fold(或叠加)。

用法

BOOST_PP_LIST_FOLD_RIGHT(op, state, list)

参数

op
形如 op(d, state, elem) 的三元操作。该宏将被对于 list 中的每个元素进行调用 -- 每次返回一个新的 state. 该操作被 BOOST_PP_LIST_FOLD_RIGHT 展开，参数分别为下一个可用的 BOOST_PP_WHILE 迭代、当前的 state, 以及当前元素。
state
进行折叠的初始状态。
list
被折叠的链表。

备注

该宏与以前的符号签名不同。参数进行了交换以提供与 BOOST_PP_LIST_FOLD_LEFT 统一的接口。
对于链表 (0, (1, (2, BOOST_PP_NIL))), 该宏展开为:
op(d, op(d, op(d, state, 2), 1), 0)
以前，该宏不可用在 BOOST_PP_WHILE 中。现在已没有该限制。不过在此情况下使用 BOOST_PP_LIST_FOLD_RIGHT_d 会更高效。

参见

  • BOOST_PP_LIST_FOLD_RIGHT_d

要求

头文件:  <boost/preprocessor/list/fold_right.hpp>

代码示例

#include <boost/preprocessor/cat.hpp>
#include <boost/preprocessor/list/fold_right.hpp>

#define LIST (a, (b, (c, BOOST_PP_NIL)))

#define OP(d, state, x) BOOST_PP_CAT(state, x)

BOOST_PP_LIST_FOLD_RIGHT(OP, _, LIST) // 展开为 _cba
