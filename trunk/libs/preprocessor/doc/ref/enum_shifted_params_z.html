BOOST_PP_ENUM_SHIFTED_PARAMS_Z 宏生成一个以逗号分隔的、移位后的参数列表。它以最高的效率重入 BOOST_PP_REPEAT.

用法

BOOST_PP_ENUM_SHIFTED_PARAMS_Z(z, count, param)

参数

z
下一个可用的 BOOST_PP_REPEAT 维度。
count
要生成的参数数量。有效值为 0 到 BOOST_PP_LIMIT_REPEAT.
param
参数的文本。BOOST_PP_ENUM_SHIFTED_PARAMS_Z 会将范围 1 到 count - 1 间的数字连接到它以生成参数。

备注

该宏展开为一个以逗号分隔的序列:
param ## 1, ... param ## count - 1

参见

  • BOOST_PP_LIMIT_REPEAT
  • BOOST_PP_ENUM_SHIFTED_PARAMS

要求

头文件:  <boost/preprocessor/repetition/enum_shifted_params.hpp>

代码示例

#include <boost/preprocessor/repetition/enum_params.hpp>
#include <boost/preprocessor/repetition/enum_shifted_params.hpp>
#include <boost/preprocessor/repetition/repeat.hpp>

int add(void) {
   return 0;
}

#define ADD_F(z, n, _) \
   int add_f(BOOST_PP_ENUM_PARAMS_Z(z, BOOST_PP_INC(n), int p)) { \
      return p0 + add_f(BOOST_PP_ENUM_SHIFTED_PARAMS_Z(z, BOOST_PP_INC(n), p)); \
   } \
   /**/

BOOST_PP_REPEAT(5, ADD_F, nil)
