BOOST_PP_LIST_FIRST_N 宏展开为一个由某个链表的前 count 个元素组成的链表。

用法

BOOST_PP_LIST_FIRST_N(count, list)

参数

count
要取出的元素数量。
list
要取出元素的链表。

备注

该宏从 list 的开头取出 count 个元素并将它们作为链表返回。
以前，该宏不可用在 BOOST_PP_WHILE 中。现在已没有这个限制。不过在此情况下使用 BOOST_PP_LIST_FIRST_N_D 更为高效。

参见

  • BOOST_PP_LIST_FIRST_N_D

要求

头文件:  <boost/preprocessor/list/first_n.hpp>

代码示例

#include <boost/preprocessor/list/first_n.hpp>

#define LIST (a, (b, (c, BOOST_PP_NIL)))

BOOST_PP_LIST_FIRST_N(2, LIST) // 展开为 (a, (b, BOOST_PP_NIL))
