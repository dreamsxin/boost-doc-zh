BOOST_PP_LIST_TO_TUPLE 宏将一个链表转换为 tuple.

用法

BOOST_PP_LIST_TO_TUPLE(list)

参数

list
被转换的链表。

备注

例如，如果 list 为 (a, (b, (c, BOOST_PP_NIL))), 则该宏将生成:
(a, b, c)
以前，该宏不可用在 BOOST_PP_FOR 中。现在已没有该限制。不过在此情况下使用 BOOST_PP_LIST_TO_TUPLE_R 更为高效。

参见

  • BOOST_PP_LIST_TO_TUPLE_R

要求

头文件:  <boost/preprocessor/list/to_tuple.hpp>

代码示例

#include <boost/preprocessor/list/to_tuple.hpp>

#define LIST (w, (x, (y, (z, BOOST_PP_NIL))))

BOOST_PP_LIST_TO_TUPLE(LIST) // 展开为 (w, x, y, z)
