BOOST_PP_TUPLE_ELEM 宏从 tuple 中取出一个元素。

用法

BOOST_PP_TUPLE_ELEM(size, i, tuple)

参数

size
tuple 的大小。有效的 tuple 大小范围为 0 到 BOOST_PP_LIMIT_TUPLE.
i
要取出的元素在 tuple 中的索引，从零起计。有效值为 0 到 size - 1.
tuple
要取出元素的 tuple.

备注

size 参数必须为 tuple 的真实大小，且 i 必须小于 tuple 的大小。

参见

  • BOOST_PP_LIMIT_TUPLE

要求

头文件:  <boost/preprocessor/tuple/elem.hpp>

代码示例

#include <boost/preprocessor/tuple/elem.hpp>

#define TUPLE (a, b, c, d)

BOOST_PP_TUPLE_ELEM(4, 0, TUPLE) // 展开为 a
BOOST_PP_TUPLE_ELEM(4, 3, TUPLE) // 展开为 d
