BOOST_PP_SUB 宏展开为其参数的差。

用法

BOOST_PP_SUB(x, y)

参数

x
操作的被减数。有效值范围为 0 到 BOOST_PP_LIMIT_MAG.
y
操作的减数。有效值范围为 0 到 BOOST_PP_LIMIT_MAG. 

备注

如果 x 和 y 的差小于 0, 则结果被调整为 0.
以前，这个宏不能在 BOOST_PP_WHILE 中使用。现在已没有这样的限制了。不过，在这种情况下，使用 BOOST_PP_SUB_D 更为高效。

参见

  • BOOST_PP_LIMIT_MAG
  • BOOST_PP_SUB_D

要求

头文件:  <boost/preprocessor/arithmetic/sub.hpp>

代码示例

#include <boost/preprocessor/arithmetic/sub.hpp>

BOOST_PP_SUB(4, 3) // 展开为 1
