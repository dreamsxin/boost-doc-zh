BOOST_PP_ARRAY_DATA 宏从一个数组中提取 tuple 数据。

用法

BOOST_PP_ARRAY_DATA(array)

参数

array
要转换为 tuple 的数组。

备注

该宏展开为一个数组中的 tuple 部分。

要求

头文件:  <boost/preprocessor/array/data.hpp>

代码示例

#include <boost/preprocessor/array/data.hpp>

#define ARRAY (3, (x, y, z))

BOOST_PP_ARRAY_DATA(ARRAY) // 展开为 (x, y, z)
