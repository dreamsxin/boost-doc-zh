第三类椭圆积分 - 勒让德形式

概要 

#include <boost/math/special_functions/ellint_3.hpp>


namespace boost { namespace math {

template <class T1, class T2, class T3>
calculated-result-type ellint_3(T1 k, T2 n, T3 phi);

template <class T1, class T2, class T3, class Policy>
calculated-result-type ellint_3(T1 k, T2 n, T3 phi, const Policy&);

template <class T1, class T2>
calculated-result-type ellint_3(T1 k, T2 n);

template <class T1, class T2, class Policy>
calculated-result-type ellint_3(T1 k, T2 n, const Policy&);

}} // namespaces
说明

这两个函数计算第三类不完全椭圆积分Π(n, φ, k) 以及第三类完全椭圆积分 Π(n, k) = E(n, π/2, k).

当T1和T2是不同类型的时候，这些函数的返回值的类型使用返回值类型推导法则 来确定: 如果它们是同一种类型，那么返回值的类型和参数的类型是一样的. 

template <class T1, class T2, class T3>
calculated-result-type ellint_3(T1 k, T2 n, T3 phi);

template <class T1, class T2, class T3, class Policy>
calculated-result-type ellint_3(T1 k, T2 n, T3 phi, const Policy&);

返回第三类不完全椭圆积分Π(n, φ, k):

要求-1 <= k <= 1 且 n < 1/sin2(φ), 否则返回定义域错误 (在这个范围之外结果将会很复杂).

最后一个策略 参数是可选的并且可以用来控制函数的表现: 如何处理错误, 使用哪种层次的精度等等. 参见策略文档了解更多信息。

注意

此外, n > 1 且 φ不在 [0, π/2]范围之内的区域暂时没有得到支持且返回定义域错误. 因为这个原因，推荐你将φ的范围保持在 [0, π/2]这间。 

template <class T1, class T2>
calculated-result-type ellint_3(T1 k, T2 n);

template <class T1, class T2, class Policy>
calculated-result-type ellint_3(T1 k, T2 n, const Policy&);

返回第三类完全椭圆积分Π(n, k):

要求-1 <= k <= 1 且 n < 1, 否则返回定义域错误 (在这个范围之外的结果将会十分复杂).

[可选的策略]

精确性

这个函数使用基本的算术运算来计算, 因此运算结果在不同的平台上并没有太多的差异. 注意：在那些将宽浮点类型给定为窄浮点类型的系统上，函数的返回值具有有效的零误差. 所有结果的相对误差都是10的-5次方数量级。
