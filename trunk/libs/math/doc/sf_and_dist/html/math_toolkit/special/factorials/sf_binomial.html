二项式系数 

#include <boost/math/special_functions/binomial.hpp>


namespace boost{ namespace math{

template <class T>
T binomial_coefficient(unsigned n, unsigned k);

template <class T, class Policy>
T binomial_coefficient(unsigned n, unsigned k, const Policy&);

}} // namespaces

返回二项式系数: nCk.

要求 k <= n.

最后一个策略 参数是可选的并且可以用来控制函数的行为: 如何处理错误, 使用哪种层次的精度等等. 参见策略文档了解更多信息。

如果计算结果太大以至于T类型无法表示，那么就可能返回溢出错误 .

精确度

对于较小参数的阶乘以及对于较大参数的 β 函数，精度是一样的(也就是说. 不超过一到两个10的-5次方)。.

测试

二项式系数的抽样测试数据由 functions.wolfram.com生成。

实现

在可能的情况下，二项式系数的计算使用阶乘查找表来计算，并使用下面的关系式：

nCk = n! / (k!(n-k)!)

否则以使用下面关系式的β函数的方式来实现：

nCk = 1 / (k * beta(k, n-k+1))

且

nCk = 1 / ((n-k) * beta(k+1, n-k))
