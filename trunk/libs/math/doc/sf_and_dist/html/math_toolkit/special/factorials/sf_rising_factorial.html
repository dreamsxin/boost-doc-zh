递增阶乘(Rising Factorial) 

#include <boost/math/special_functions/factorials.hpp>


namespace boost{ namespace math{

template <class T>
calculated-result-type rising_factorial(T x, int i);

template <class T, class Policy>
calculated-result-type rising_factorial(T x, int i, const Policy&);

}} // namespaces

返回x 和 i的递增阶乘：

rising_factorial(x, i) = Γ(x + i) / Γ(x);

或者

rising_factorial(x, i) = x(x+1)(x+2)(x+3)...(x+i)

注意：x 和 i 可以是正数也可以是负数：.

最后一个策略 参数是可选的并且可以用来控制函数的行为: 如何处理错误, 使用哪种层次的精度等等. 参见策略文档了解更多信息。

如果计算结果太大以至于T类型无法表示，那么就可能返回溢出错误 .

函数返回值的类型使用返回值类型推导法则来确定：如果T是int型，那么返回值的类型是double, 否则返回值类型为T.

精确性

这个函数的精确性与tgamma_delta_ratio 函数的精确性一样.

测试

递减阶乘的的抽样测试数据由 functions.wolfram.com生成

实现

递增和递减阶乘函数以作为γ函数的比的形式使用tgamma_delta_ratio来实现。由tgamma_delta_ratio函数内部实现针对较小的参数的优化。
