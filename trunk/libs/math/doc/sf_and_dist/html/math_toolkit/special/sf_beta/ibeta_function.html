说明

有4个不完全β函数 : 两个标准版本( normalised versions) (也称作规则 β函数 ) 返回值的区间为 [0, 1],以及两个非标准版本( non-normalised )且返回值的区间为[0, beta(a, b)]. 对于统计应用感兴趣的用户应当使用标准版本 (或规则版本 ) (ibeta 和 ibetac).

所有的这些函数都要求： a > 0, b > 0 且 0 <= x <= 1.

当T1,T2和T3是不同类型的时候，函数的返回值的类型使用函数返回值推导法则 来确定。.

最后一个策略 参数是可选的并且可以用来控制函数的行为: 如何处理错误, 使用哪种层次的精度等等. 参见策略文档了解更多信息。

测试

有两种类别的测试: 抽样测试(spot tests)比较数据取自于Mathworld's 在线函数求值 和这个库: 它们对这个库的实现提供了一个基本的 "合理性测试(sanity check)" , 对第一种实现都有一个抽取测试(spot-test ) (参见下面的实现说明).

精度测试使用精度非常高的测试数据 (NTL RR 类 设置为 1000-bit 精度), 使用"教书(textbook)"连分数表示( continued fraction representation )(参见下面的实现讨论中的第一个连分数).注意：连分数并没有用在这个库的实现中, 因此我们的测试数据完全独立于这些代码。
