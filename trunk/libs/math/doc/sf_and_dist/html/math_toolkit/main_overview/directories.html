目录和文件结构(Directory and File Structure)

boost/math

/concepts/

定义类RealType的基本 性质 (参考 real_concept.hpp)。大多数应用程序将使用double作为RealType(当可能 的时候，简写的分布名字的typedef为double类型保留)，一些应用可能使用float 或 long double，但也可能使用更高精度的类型，类似于满足real_concept所指定要求的NTL::RR 类型。

/constants/

一些高精度的数学常量的模板化的定义。(在constants.hpp中)。

/distributions/

在数学中使用的分布以及，尤其是，统计： Gaussian, Students-t, Fisher, Binomial 等等。

/policies/

策略框架(Policy framework)，用于处理用户要求的特性修改。

/special_functions/

数学函数通常看作是“特殊的”，类似于 beta，cbrt， erf， gamma， lgamma， tgamma ... (这些函数中的一部分在C++，以及 C99/TR1，以及或许TR2中指定)。

/tools/

函数所使用的工具，类似于计算多项式，连分数，根查找，精度和限制，以及测试。有一些工具在这个包之外可以找到实际应用。

boost/libs

/doc/

文档源文件，用 Quickbook 格式处理为 html 和 pdf 格式。

/examples/

使用这些数学函数和分布的例子。

/performance/

性能测试和调整程序。

/test/

测试文件，在各种 .cpp 文件中，大多数使用 Boost.Test (一些的测试文件为 .ipp 文件，通常使用NTL RR类型，使用对于这个类型足够大的精度，通常精度为高达256bit的实数类型)。

/tools/

用于产生测试数据的程序。以及对于NTL 发行包的一些修改，用于提供一些额外的(关键的)功能。
