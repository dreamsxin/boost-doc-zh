名字空间(Namespaces)

所有的数学函数和分布类型都在名字空间namespace boost::math 中

因此，例如，学生t分布模板在名字空间 boost::math 中为： 

template <class RealType> class students_t_distribution

并可以使用一个保留的名字students_t(对于RealType double)来实例化

typedef students_t_distribution<double> students_t;

student_t mydist(10);

没有准备作为一般应用程序使用的函数在名字空间boost::math::detail中。

可能更通用一些的函数，类似于digits (significand)，max_value，min_value 和epsilon 在 boost::math::tools中。

策略 和配置信息位于空间boost::math::policies中。
