计算两个浮点数之间可以表示的浮点距离(ULP) 函数float_distance

函数float_distance 查找两个浮点数之间的间隔/比特/ULP. 如果浮点数有有效位数被看作整数, 那么这两个浮点数之间的差就是 ULP/间隔/比特数目的不同.

概要 

#include <boost/math/special_functions/next.hpp>


namespace boost{ namespace math{

template <class FPT>
FPT float_distance(FPT a, FPT b);

}} // namespaces
float_distance函数说明

返回a 和 b之间的距离：结果永远都是一个有符号的整数值 (存储在浮点类型 FPT中) 用于表示 a 和b之间可以表示的间隔的数目。

注意：

  • float_distance(a, a) 永远返回 0.
  • float_distance(float_next(a), a) 永远返回 1.
  • float_distance(float_prior(a), a) 永远返回 -1.

函数float_distance 等价于计算a和b之间的 ULP (Units in the Last Place) 除了它返回一个有符号数用于指示a是否大于b。.

如果距离大太，那么就有可能无法以一个FPT类型的整数来表示距离, 但在实际中，这似乎不太可能成为一个问题。.
