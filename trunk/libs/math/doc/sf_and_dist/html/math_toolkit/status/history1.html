变更历史以及最新消息(History and What's New)

Boost-1.40.0
  • Added support for MPFR as a bignum type.
  • Added some full specializations of the policy classes to reduce compile times.
  • Added logistic and hypergeometric distributions, from Gautam Sewani's Google Summer of Code project.
  • Added Laplace distribution submitted by Thijs van den Berg.
  • Updated performance test code to include new distributions, and improved the performance of the non-central distributions.
  • Added SSE2 optimised Lanczos approximation code, from Gautam Sewani's Google Summer of Code project.
  • Fixed bug in cyl_bessel_i that used an incorrect approximation for ν = 0.5, also effects the non-central Chi Square Distribution when ν = 3, see bug report #2877.
  • Fixed minor bugs #2873.
Boost-1.38.0
  • 添加 Johan Råde 的优化浮点分类子程序。
  • 修正了代码，可以在 GCC 的 -pedantic 模式下编译(缺陷报告 #1451)。

Boost-1.37.0
  • 改进反双曲函数的精度和测试。

Boost-1.36.0
  • 添加 非中心χ方形分布( Noncentral Chi Squared Distribution).
  • 添加 非中心β分布(Noncentral Beta Distribution).
  • 添加 非中心F分布(Noncentral F Distribution).
  • 添加 非中心T分布(Noncentral T Distribution).
  • 添加 指数积分函数(Exponential Integral Functions).
  • 添加 ζ函数(Zeta Function).
  • 添加 舍入和截断函数(Rounding and Truncation functions).
  • 添加 运行时基础的编译时幂函数(Compile time powers of runtime bases).
  • 添加 对于兰克泽斯计算进行SSE2优化(SSE2 optimizations for Lanczos evaluation).

Boost-1.35.0: Post Review First Official Release
  • 添加策略框架来更好地控制函数的行为。
  • Breaking change: 将出现定义域错误，pole错误和溢出错误是的缺省行为改为抛出异常(基于复审的反馈)，这种行为可以使用策略来定制。
  • Breaking change: 将当一个内部计算错误发生时，将抛出的异常必变为 boost::math::evaluation_error。
  • Breaking change: 将离散分位点函数改为返回一个整数：这比查找朴实根要快20倍，这种行为可以使用策略来定制。
  • 多项式/有理函数计算现在是可定制的并且比以前更快一些。
  • 添加性能测试程序。

里程碑 4:第二次复审候选 (1st March 2007)
  • 将 Xiaogang Zhang 的Bessel 函数代码移植到这个库中，并将其与这个库中的其它代码相整合起来。
  • 添加 C# "分布浏览器(Distribution Explorer)"例子程序。

里程碑 3: 第一次复审候选 (31st Dec 2006)
  • 实现主要的概率分布和密度函数。
  • 实现 digamma函数。
  • 添加更多的阶乘函数。
  • 实现TR1中的 Hermite， Legendre 和 Laguerre 多项式以及球面低函数(spherical harmonic functions)。
  • 将 Xiaogang Zhang 的椭圆积分代码移植到这个库中，并将其与这个库中的其它代码相整合起来。
  • 将Hubert Holin的Boost.Math 特殊函数移植到这个库中，并将其与这个库中的其它代码相整合起来。

里程碑 2: September 10th 2006发行
  • 实现统计分布的预览发行( preview release )。
  • 添加统计分布手册。
  • 实现根查找算法。
  • 实现不完全gamma 和 beta 函数的反函数。
  • 将 erf/erfc 重写为一个有理逼近(精确度为 128-bit )。.
  • 集成使用测试数据和Boost.Test生成统计结果：使用一个预期结果的数据库，由测试，浮点类型，平台和编译器索引。
  • 改进在接近1 和 2时的lgamma函数 (有理逼近)。
  • 改进 erf/erfc函数的反函数(有理逼近)。
  • 实现有理函数生成 (Remez 算法)。

里程碑 1: March 31st 2006发行
  • 实现 gamma/beta/erf 函数以及不完全gamma/beta/erf 函数。
  • 生成高精度测试数据，依据这些数据来裁定将来的改进。
  • 提供用于计算无限级数，连分数，有理函数的工具。
  • 提供用于测试已制成表的测试数据的工具，并收集关于误差率的统计信息。
  • 提供足够的文档来使得用户可以很好地使用这个库。
