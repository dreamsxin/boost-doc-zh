与NTL库一起使用－一个高精度的浮点库(Using With NTL - a High-Precision Floating-Point Library)

这个库中的特殊函数和工具可以与NTL::RR (一个任意精度的浮点类型)一起使用，通过使用boost/math/bindings/rr.hpp。参考 NTL： A Library for doing Number Theory by Victor Shoup

不幸地是NTL::RR 并没有很好地满足我们的概念要求( conceptual requirements)，所以在 boost/math/bindings/rr.hpp 中定义了一个boost::math::ntl::RR 包装类，用于取代NTL::RR。这个类打算作为“真正的”NTL_RR库的一个混入的(drop-in)替代，增加了一些语法糖(syntactic sugar)来满足这个库的需要，以及在NTL中没有实现的一些标准库函数。

For those functions that are based upon the Lanczos approximation, the bindings defines a series of approximations with up to 61 terms and accuracy up to approximately 3e-113. This therefore sets the upper limit for accuracy to the majority of functions defined this library when used with NTL::RR.

There is a concept checking test program for NTL support here.
