Quaternion 数值运算

real 和 unreal 
template<typename T>	T              real(quaternion<T> const & q);
template<typename T>	quaternion<T>  unreal(quaternion<T> const & q);

这些函数分别返回 q.real() 和 q.unreal() .

conj 
template<typename T>	quaternion<T>  conj(quaternion<T> const & q);

这个函数返回这个四元数的共轭四元数.

sup

template<typename T> T sup(quaternion<T> const & q);

这个函数返回这个四元数的 sup 范数 ( abs(q.R_component_1())...abs(q.R_component_4())中的最大值).

l1 
template<typename T>	T  l1(quaternion<T> const & q);

这个函数返回这个四元数的 l1 范数 (abs(q.R_component_1())+...+abs(q.R_component_4())).

abs 
template<typename T>	T  abs(quaternion<T> const & q);

这个函数返回这个四元数的大小 (欧几里德范数 ，Euclidian norm).

norm 
template<typename T>	T  norm(quaternion<T>const  & q);

这个函数返回四元数的(Cayley)范数. 术语"norm", 或许令人产生困惑.因为大多数人将它与欧几里德范数(以及 二次函数)联系起来. 对于这一版本的四元数(通常所说的数学对象), 欧几里德范数(也被称作大小) 是Cayley 范数的平方根.

译注：关于 欧几里德范数(Euclidian norm),以及四元数在 wikipedia中有相关的描述.
