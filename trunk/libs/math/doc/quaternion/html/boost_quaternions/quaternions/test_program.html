测试程序

quaternion_test.cpp 测试四元数模板针对于单精度浮点数，双精度浮点数和长双精度浮点数的特化版本。 (sample output, 允许输出消息).

如果你定义了 boost_quaternions_TEST_VERBOSE, 你将获得额外的输出 (详细输出); 当然，仅当你同时允许输出程序内部的运行信息(通过取消测试程序中的相关的注释 或者通过过向你的命令行输入中添加--log_level=messages ,...). 在那种情况下, 如果你想在交互的模式下运行测试程序, 你可能需要额外定义符号 BOOST_INTERACTIVE_TEST_INPUT_ITERATOR 达到从标准输入端输入 您的选择来交互测试输入操作符(取代将输入硬编码到测试程序中).
