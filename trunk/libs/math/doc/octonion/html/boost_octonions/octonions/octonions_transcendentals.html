Octonions 超越函数

在这个八元数的实现中并没有提供log 或 sqrt函数, 并且函数pow 局限于指数的整数次幂. 关于这一点有如下理由: 一方面, 与八元数对应的解析开拓("分支切割")仍然有需要彻底的研究 (至少对我来说...), 并且我们希望避免重述在标准中介绍的复数的复数次幂(这是有良好定义的，但不在标准中...). 说到无稽之谈, 象说 pow(0,0) 的结果是由 "具体实现规定" 的，就是坏了脑的...

然而，我们确实提供了一些超越函数, 最主要的是指数运算. 指数运算是满足 闭公式 (译注："closed formula"是指逻辑闭式，闭公式), 这是作者的研究结果 (另一方面，指数运算的存在和定义, 尤其是对于八元数, 有几百年的历史). 基本上, 任意收敛的实系数幂级数, 在 C（复数域）上满足闭公式的, 都能够被转化为对应的O （(译注： 在数学中 O 表示八元数域） 上的幂级数. 这个八元数类型中的更多的超越函数可能会依据需求在将来添加. 需要指出的是，这些函数增加了对 boost/math/special_functions/sinc.hpp 和boost/math/special_functions/sinhc.hpp 头文件 的依赖.

exp 
template<typename T> 
octonion<T> exp(octonion<T> const & o);

计算八元数的 exp（指数）.

cos 
template<typename T> 
octonion<T> cos(octonion<T> const & o);

计算八元数的 cosine（余弦）

sin 
template<typename T> 
octonion<T> sin(octonion<T> const & o);

计算八元数的 sine（正弦）.

tan 
template<typename T> 
octonion<T> tan(octonion<T> const & o);

计算八元数的 tangent（正切）.

cosh 
template<typename T> 
octonion<T> cosh(octonion<T> const & o);

计算八元数的 hyperbolic cosine（双曲余弦）.

sinh 
template<typename T> 
octonion<T> sinh(octonion<T> const & o);

计算八元数的 hyperbolic sine（双曲正弦）.

tanh 
template<typename T> 
octonion<T> tanh(octonion<T> const & o);

计算八元数的 hyperbolic tangent（双曲正切）.

pow 
template<typename T> 
octonion<T>  pow(octonion<T> const & o, int n);

计算八元数 q 的 n 次幂.

译注: 关于 闭公式(closed formula) 以及 数域 的概念在 wikipedia中有相关的描述.
