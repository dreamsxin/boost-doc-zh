概览

八元数, 像 四元数一样, 是与复数类似的.

八元数在理论物理中有一定的应用.

在实际的术语中, 一个八元数只不过是我们可以写作o = α + βi + γj + δk + εe' + ζi' + ηj' + θk', 这种形式的八维实数(α,β,γ,δ,ε,ζ,η,θ). 其中i, j 和 k 与它们在四元数中的含义是一样的, 而 e', i', j' 和 k' 是和 i (or j or k)本质上起相同作用不同对象.

加法以及定义在这种八元数之上的乘法运算, 推广了在四元数上的加法与乘法操作. 这一次主要的新奇之处在于 乘法不但是不可交换的, 甚至也是不可结合的 (也就是说. 对于八元数 x, y 和 z x(yz) ≠ (xy)z)(译注：也即是不满足交换律和结合律). 一种记忆这些规则的方法是使用下面的乘法表:

一些传统的概念，例如指数，不需要太大的改变就可以在八元数的领域中继续延用, 但是其它的一些概念，比如求平方根就不能继续延用（指数运算具有闭型(closed form) 的事实源于作者，但是，八元数存在指数运算的事实是很早以前就有了）.
